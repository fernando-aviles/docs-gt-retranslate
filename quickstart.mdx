## Setup

Follow these steps to deploy and customize your docs with Mintlify.

### Create Your Docs Repo

Your docs are built from MDX files using our [starter kit](https://github.com/mintlify/starter).

<AccordionGroup>
  <Accordion icon="github" title="Set up repository">
    1. Clone our [starter template](https://github.com/mintlify/starter) into a new public repo
    2. Contact [support@mintlify.com](mailto:support@mintlify.com) to deploy

    Already created a repo through our onboarding? Find it on your [dashboard](https://dashboard.mintlify.com).

<Frame>
    ![Starter Template](https://mintlify.s3-us-west-1.amazonaws.com/mintlify/images/starter-template.png)
</Frame>
  </Accordion>

  <Accordion icon="message-bot" title="Install GitHub app">
    Install our GitHub app from the [dashboard settings](https://dashboard.mintlify.com/settings) for auto-deployments.

<Frame>
    ![GitHub Bot Verification](https://mintlify.s3-us-west-1.amazonaws.com/mintlify/images/github-bot-verification.png)
</Frame>
  </Accordion>
</AccordionGroup>

### Edit Content

<AccordionGroup>
  <Accordion icon="rectangle-terminal" title="Local development">
    1. Install [git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)
    2. Clone your repo: `git clone <your-repo>`
    3. Install Mintlify CLI: `npm i -g mintlify`
    
    See [local development guide](/development) for more details.
  </Accordion>

  <Accordion icon="pen-to-square" title="Web editor">
    Check our [web editor guide](/web-editor).
  </Accordion>

  <Accordion icon="palette" title="Brand customization">
    Edit `mint.json` to customize your docs:

    ```json
    "name": "Your Company"
    "logo": {
      "light": "/logo/light.svg",
      "dark": "/logo/dark.svg",
      "href": "https://yourcompany.com"
    },
    "favicon": "/favicon.svg",
    "colors": {
      "primary": "#2AB673",
      "light": "#55D799",
      "dark": "#117866",
    },
    ```

    See [global settings](/settings/global) for all options.
  </Accordion>

  <Accordion icon="pencil" title="Add pages">
    Create MDX files with this header:

    ```md
    ---
    title: "Page Title"
    sidebarTitle: "Sidebar title (optional)"
    description: "Description (optional)"
    ---
    ```

    Learn about [MDX syntax](/text) and [components](/content/components).
  </Accordion>

  <Accordion icon="rocket" title="Deploy changes">
    Push changes to deploy automatically. Manual updates available on the [dashboard](https://dashboard.mintlify.com).

<Frame>
    ![Manual Update](https://mintlify.s3-us-west-1.amazonaws.com/mintlify/images/manual-update.png)
</Frame>
  </Accordion>

  <Accordion icon="code-branch" title="API reference setup">
    1. Add OpenAPI spec (`openapi.yaml`/`openapi.json`) or set URL in `mint.json`:
    ```json
    "openapi": "your-openapi-url"
    ```

    2. Generate endpoint files:
    ```bash
    npx @mintlify/scraping@latest openapi-file <path-or-url-to-openapi-file>
    ```

    3. Add generated files to `mint.json` navigation

    See guides for [OpenAPI setup](/api-playground/openapi/setup) and [auth methods](/api-playground/openapi/writing-openapi).
  </Accordion>

  <Accordion icon="chart-simple" title="Analytics">
    Access built-in analytics on your [dashboard](https://dashboard.mintlify.com/analytics) or add third-party [integrations](/integrations/analytics/overview).
  </Accordion>
</AccordionGroup>
