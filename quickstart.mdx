## Getting Started

Follow these steps to deploy and customize your documentation with Mintlify.

### Setting Up

Mintlify docs are rendered from MDX files using our [starter kit](https://github.com/mintlify/starter). Get started by either:
- Creating a new project through the [dashboard](https://dashboard.mintlify.com)
- Cloning our [starter kit](https://github.com/mintlify/starter)

<AccordionGroup>
  <Accordion icon="message-bot" title="Install the GitHub app">
    Install our GitHub app through the [dashboard](https://dashboard.mintlify.com/settings) for automatic deployments. A check mark will appear next to the commit hash when installed successfully.
  </Accordion>

  <Accordion icon="code-branch" title="Monorepo Setup">
    For monorepo setups:
    1. Move docs content to your monorepo
    2. Set the docs path in the [dashboard](https://dashboard.mintlify.com/settings/deployment/git-settings)
  </Accordion>
</AccordionGroup>

### Making Changes

<AccordionGroup>
  <Accordion icon="rectangle-terminal" title="Local Development">
    1. Install [git](https://git-scm.com)
    2. Clone your docs repository
    3. Install Mintlify CLI: `npm i -g mintlify`
    4. Run `mintlify dev` to preview changes

    See the [local development guide](/development) for details.
  </Accordion>

  <Accordion icon="pen-to-square" title="Web Editor">
    Use our web editor for a visual editing experience. Learn more in our [web editor guide](/web-editor).
  </Accordion>

  <Accordion icon="palette" title="Customize Branding">
    Update `docs.json` to match your brand:

    ```json
    {
      "name": "Your Company",
      "logo": {
        "light": "/logo/light.svg",
        "dark": "/logo/dark.svg"
      },
      "colors": {
        "primary": "#2AB673"
      }
    }
    ```

    See [global settings](/settings/global) for all options.
  </Accordion>

  <Accordion icon="pencil" title="Add Content">
    Create MDX files with this template:
    ```md
    ---
    title: "Page Title"
    description: "Description"
    ---
    ```
    Learn more about [MDX syntax](/text) and [components](/content/components).
  </Accordion>

  <Accordion icon="code-branch" title="API References">
    Add API documentation by:
    1. Adding your OpenAPI file or URL to `docs.json`
    2. Generate endpoint files: `npx @mintlify/scraping@latest openapi-file <path-or-url>`
    3. Add generated files to navigation

    See our [OpenAPI guide](/api-playground/openapi/setup) for details.
  </Accordion>

  <Accordion icon="chart-simple" title="Analytics">
    Access analytics on your [dashboard](https://dashboard.mintlify.com/analytics) or integrate with [supported providers](/integrations/analytics/overview).
  </Accordion>
</AccordionGroup>
