Welcome to Mintlify! This guide will help you set up your documentation in just a few minutes.

<Steps>
  <Step title="Create your documentation repository">
    Get started with Mintlify by either:
    - Creating a new project through our [dashboard](https://dashboard.mintlify.com)
    - Cloning our [starter template](https://github.com/mintlify/starter)

    <Note>
      Install our GitHub app from the [dashboard settings](https://dashboard.mintlify.com/settings) 
      to enable automatic deployments when you push changes.
    </Note>
  </Step>

  <Step title="Add your content">
    Create new MDX files in your docs folder using this basic template:

    ```md
    ---
    title: "Page Title"
    description: "A description of the page"
    ---

    Your content here
    ```

    Add the page to your navigation in `mint.json`:

    ```json
    {
      "navigation": [
        {
          "group": "Getting Started",
          "pages": ["quickstart"]
        }
      ]
    }
    ```

    Learn more about [writing MDX](/text) and using our [components](/content/components).
  </Step>

  <Step title="Configure your docs">
    Customize your documentation's appearance in `mint.json`:

    ```json
    {
      "name": "Your Company",
      "logo": {
        "light": "/logo/light.svg",
        "dark": "/logo/dark.svg"
      },
      "favicon": "/favicon.svg",
      "colors": {
        "primary": "#2AB673",
        "light": "#55D799",
        "dark": "#117866"
      }
    }
    ```

    Get the full list of options in our [global settings](/settings/global).
  </Step>

  <Step title="Preview changes">
    You have two options to preview your changes:

    **Option 1: Local Development**
    ```bash
    # Install the Mintlify CLI
    npm i -g mintlify

    # Start local development
    mintlify dev
    ```

    **Option 2: Web Editor**
    Use our browser-based editor at your [dashboard](https://dashboard.mintlify.com). Learn more about the [web editor](/web-editor).
  </Step>

  <Step title="Deploy">
    Once ready, commit and push your changes to deploy your documentation site:

    ```bash
    git add .
    git commit -m "docs: update documentation"
    git push
    ```

    Add a custom domain through the [dashboard settings](https://dashboard.mintlify.com/settings) to make your docs accessible at your own domain.
  </Step>
</Steps>
