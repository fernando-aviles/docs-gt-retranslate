## Setup

Get started with Mintlify in three simple steps:

1. Create a repository using our [starter kit](https://github.com/mintlify/starter)
2. Install the [GitHub app](https://dashboard.mintlify.com/settings) for automatic deployments
3. Start editing your docs

<AccordionGroup>
  <Accordion icon="code-branch" title="Using with a monorepo">
    To add docs to your existing monorepo:
    1. Move docs content to your repo
    2. Set the path to `mint.json` in the [dashboard](https://dashboard.mintlify.com/settings/deployment/git-settings)
  </Accordion>
</AccordionGroup>

## Edit Your Docs

### Local Development
1. Install [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)
2. Clone your docs: `git clone <your-repo>`
3. Install Mintlify CLI: `npm i -g mintlify`

Need more details? Check our [local development guide](/development).

### Quick Customization

Update your `mint.json` to match your brand:

```json
"name": "Your Company"
"logo": {
  "light": "/logo/light.svg",
  "dark": "/logo/dark.svg",
  "href": "https://yourcompany.com"
},
"favicon": "/favicon.svg",
"colors": {
  "primary": "#2AB673",
  "light": "#55D799",
  "dark": "#117866",
}
```

### Adding Content

Create new pages with this template:

```md
---
title: "Page Title"
description: "Optional subtitle"
---
```

Learn more about [MDX syntax](/text) and [components](/content/components).

## API Documentation

To add API references:

1. Add your OpenAPI file (`openapi.yaml` or `openapi.json`) to your repo
2. Generate endpoint files:
```bash
npx @mintlify/scraping@latest openapi-file <path-to-openapi-file>
```
3. Add generated files to `mint.json` navigation

See our complete [OpenAPI guide](/api-playground/openapi/setup) for more details.
