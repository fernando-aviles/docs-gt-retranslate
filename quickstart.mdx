## Setup

1. Clone our [starter repository](https://github.com/mintlify/starter)
2. Install our [GitHub app](https://dashboard.mintlify.com/settings) for automatic deployments
3. Install the CLI: `npm i -g mintlify`

## Development

### Local Development
```bash
# Clone your docs
git clone <your-repo>

# Preview changes
mintlify dev
```

### Customize Your Docs

1. Configure `mint.json`:
```json
{
  "name": "Your Company",
  "logo": {
    "light": "/logo/light.svg",
    "dark": "/logo/dark.svg"
  },
  "colors": {
    "primary": "#2AB673"
  }
}
```

2. Add content in MDX:
```md
---
title: "Page Title"
description: "Description"
---

Your content here
```

## API Documentation

Generate API references from OpenAPI:

1. Add your OpenAPI spec to the repo
2. Generate endpoints:
```bash
npx @mintlify/scraping@latest openapi-file <path-to-spec>
```

## Deploy

1. Push changes to GitHub
2. Add your custom domain in the [dashboard](https://dashboard.mintlify.com/settings)
