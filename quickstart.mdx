## Getting Started

Mintlify makes it easy to create beautiful documentation that your users will love. Follow these simple steps to get your documentation site up and running.

### Setting Up Your Repository

<Steps>
  <Step title="Get the starter template">
    Create a new repository using our [starter template](https://github.com/mintlify/starter) or onboard through the [dashboard](https://dashboard.mintlify.com).
  </Step>
  <Step title="Install GitHub app">
    Install our GitHub app from your [dashboard settings](https://dashboard.mintlify.com/settings) to enable automatic deployments.
  </Step>
  <Step title="Preview changes locally">
    ```bash
    # Install the Mintlify CLI
    npm i -g mintlify

    # Start local development
    mintlify dev
    ```
  </Step>
</Steps>

### Making Updates

There are two ways to update your documentation:

<AccordionGroup>
  <Accordion icon="pen-to-square" title="Using the web editor">
    The fastest way to update your docs is through our [web editor](/web-editor). It provides:
    * Live preview of your changes
    * Visual editing of content
    * Direct publishing to your docs site
  </Accordion>

  <Accordion icon="rectangle-terminal" title="Using your IDE">
    For developers, you can clone your repository and make changes locally:
    1. Clone your docs repository
    2. Make changes using your preferred editor
    3. Preview with `mintlify dev`
    4. Push changes to deploy
    
    Learn more in our [local development guide](/development).
  </Accordion>
</AccordionGroup>

### Customization

The `mint.json` file controls your documentation configuration. Here are the essential settings to get started:

```json
{
  "name": "Your Company",
  "logo": {
    "light": "/logo/light.svg",
    "dark": "/logo/dark.svg"
  },
  "favicon": "/favicon.svg",
  "colors": {
    "primary": "#2AB673",
    "light": "#55D799",
    "dark": "#117866"
  }
}
```

Visit [global settings](/settings/global) to see all configuration options.
