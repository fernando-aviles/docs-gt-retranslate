title: "PostHog"
---

Configure PostHog analytics in your `mint.json`:

<CodeGroup>

```json Analytics options in mint.json
"analytics": {
    "posthog": {
        "apiKey": "required",
        "apiHost": "optional" // Defaults to https://app.posthog.com
    }
}
```

```json Example
"analytics": {
    "posthog": {
        "apiKey": "phc_TXdpocbYTeZVm5VJmMzHTMrCofBQu3e0kN7HGMNGTVW"
    }
}
```

</CodeGroup>

<Warning>
    Enabling PostHog analytics will disable the analytics on the Mintlify dashboard.
</Warning>

## Session Recordings
Add your docs website URL to PostHog's "Authorized domains for recordings" in project settings to enable session recordings.
