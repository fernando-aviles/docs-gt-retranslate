---
title: "PostHog"
---

Add PostHog analytics configuration to your `mint.json`:

<CodeGroup>

```json mint.json
"analytics": {
    "posthog": {
        "apiKey": "required",
        "apiHost": "optional" // Defaults to https://app.posthog.com
    }
}
```

</CodeGroup>

<Warning>
    Enabling PostHog analytics will disable the Mintlify dashboard analytics.
</Warning>

For session recordings, add your docs website URL to PostHog's "Authorized domains for recordings" in project settings.
