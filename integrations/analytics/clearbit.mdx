---
title: "Clearbit"
---

To enable Clearbit analytics, add this configuration to your `mint.json`:

<CodeGroup>

```json mint.json
"analytics": {
    "clearbit": {
        "publicApiKey": "required"
    }
}
```

```json Example Configuration
"analytics": {
    "clearbit": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
