---
title: "Google Tag Manager"
---

To add Google Tag Manager to your documentation:

1. Add your GTM configuration to the `mint.json` file
2. The script will automatically be injected into all your pages

Note: You are responsible for implementing any required cookie consent banners.

<CodeGroup>

```json mint.json
"analytics": {
    "gtm": {
        "tagId": "required"
    }
}
```

```json Example
"analytics": {
    "gtm": {
        "tagId": "GTM-MGBL4PW"
    }
}
```

</CodeGroup>
