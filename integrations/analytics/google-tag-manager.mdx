---
title: "Google Tag Manager"
---

Add your GTM tag ID to `mint.json` to enable Google Tag Manager on all pages. Note: You are responsible for implementing cookie consent banners if needed.

<CodeGroup>

```json Analytics options in mint.json
"analytics": {
    "gtm": {
        "tagId": "required"
    }
}
```

```json Example
"analytics": {
    "gtm": {
        "tagId": "GTM-MGBL4PW"
    }
}
```

</CodeGroup>
