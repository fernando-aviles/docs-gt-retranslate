---
title: "Plausible"
---

To start using Plausible analytics, simply add your website's domain name to your `mint.json` file.

<Info>

    Do not include `https://` for the domain or server.

</Info>

<CodeGroup>

```json Analytics options in mint.json
"analytics": {
    "plausible": {
        "domain": "required",
        "server": "optional"
    }
}
```

```json Example
"analytics": {
    "plausible": {
        "domain": "docs.domain.com"
    }
}
```

</CodeGroup>
