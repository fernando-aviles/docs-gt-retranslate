---
title: "Pirsch"
---

To enable Pirsch analytics, add this configuration to your `mint.json`:

<CodeGroup>

```json Analytics options in mint.json
"analytics": {
    "pirsch": {
        "id": "required"
    }
}
```

```json Example
"analytics": {
    "pirsch": {
        "id": "8Kw7OKxBfswOjnKGZa7P9Day8JmVYwTp"
    }
}
```

</CodeGroup>

Find your site ID in Pirsch under Settings > Developer > Identification Code.
