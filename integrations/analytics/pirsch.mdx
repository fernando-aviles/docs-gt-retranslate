---
title: "Pirsch"
---

Add Pirsch analytics to your `mint.json`:

```json
"analytics": {
    "pirsch": {
        "id": "required" // Get from Settings > Developer > Identification Code
    }
}
```

Example:
```json
"analytics": {
    "pirsch": {
        "id": "8Kw7OKxBfswOjnKGZa7P9Day8JmVYwTp"
    }
}
```
