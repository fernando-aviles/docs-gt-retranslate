Connect your documentation to <Tooltip tip="Google Analytics 4">GA4</Tooltip> to track visitor analytics.

## Setup Steps

1. Create a GA4 property if you don't have one already
2. Create a Web Stream in GA4:
   - Go to Settings (cog icon) → Data Streams
   - Click "Add Stream" → "Web"
   - Enter your Mintlify docs URL

<Frame>![](/images/ga4-web-streams.png)</Frame>

3. Copy your Measurement ID (`G-XXXXXXX`) from Stream Details
4. Add it to your `mint.json`:

```json mint.json
"analytics": {
    "ga4": {
        "measurementId": "G-XXXXXXX"
    }
}
```

<Note>
- Data takes 2-3 days to appear in Google Analytics
- Use [Google Analytics Debugger](https://chrome.google.com/webstore/detail/google-analytics-debugger/jnkmfdileelhofjcijamephohjechhna?hl=en) to verify setup
- Analytics are disabled on preview links
</Note>
