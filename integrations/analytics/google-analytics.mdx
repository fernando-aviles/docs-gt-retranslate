---
title: "Google Analytics 4"
---

To integrate <Tooltip tip="Google Analytics 4">GA4</Tooltip> with Mintlify, follow these steps:

1. Generate a new GA4 property (compatible with existing Universal Analytics projects)
2. Create a web stream:
   - Navigate to Data Streams from the settings (cog icon)
   - Add a new Web Stream using your Mintlify docs URL
   - Copy your Measurement ID (format: `G-XXXXXXX`)

3. Add the Measurement ID to your `mint.json`:
```json mint.json
"analytics": {
    "ga4": {
        "measurementId": "G-XXXXXXX"
    }
}
```

<Note>
- Data takes 2-3 days to appear in Google Analytics
- Use [Google Analytics Debugger](https://chrome.google.com/webstore/detail/google-analytics-debugger/jnkmfdileelhofjcijamephohjechhna?hl=en) to verify setup
- Analytics are disabled on preview links
</Note>
