title: "Google Analytics 4"
---

Generate a new <Tooltip tip="Google Analytics 4">GA4</Tooltip> property for your Mintlify documentation. If you're using Universal Analytics (UA), you can still create a GA4 property in the same project.

## Setup Steps

### 1. Create a Web Stream

1. Go to Google Analytics settings (cog icon, bottom left)
2. Navigate to Data Streams
3. Click "Add Stream" > "Web"
4. Enter your Mintlify docs URL

<Frame>![](/images/ga4-web-streams.png)</Frame>

Your Measurement ID (format: `G-XXXXXXX`) will appear in Stream Details.

### 2. Configure mint.json

Add your Measurement ID:

```json mint.json
"analytics": {
    "ga4": {
        "measurementId": "G-XXXXXXX"
    }
}
```

### 3. Verification

- Data appears in GA4 within 2-3 days
- Use [Google Analytics Debugger](https://chrome.google.com/webstore/detail/google-analytics-debugger/jnkmfdileelhofjcijamephohjechhna?hl=en) to verify setup
- Check browser console for GA4 requests

Preview links don't track analytics.
