---
title: "Front"
---

To integrate [Front Chat](https://front.com), add your Front Chat ID to `mint.json`:

<CodeGroup>

```json Integration options in mint.json
"integrations": {
    "frontchat": "CHAT_ID"
}
```

```json Example
"integrations": {
    "frontchat": "1365d046d7c023e9b030ce90d02d093a"
}
```

</CodeGroup>
