Speakeasy auto-generates code snippets from your SDKs and displays them in your [API playground](https://mintlify.com/docs/api-playground/overview).

## Setup SDK Repos

Add this to your `.speakeasy/workflow.yaml` in your SDK repositories:

```yaml .speakeasy/workflow.yaml
targets:
  my-target:
    target: typescript
    source: my-source
    codeSamples:
      output: codeSamples.yaml
```

## Setup Docs Repo

You can set up your docs repo either through CLI or manual configuration.

### Option 1: CLI Setup (Recommended)

1. Run these commands:
```bash
speakeasy configure sources
speakeasy configure github
```

2. Follow the prompts to:
   - Set up your OpenAPI spec source
   - Add the code snippet overlay
   - Configure the output spec path

### Option 2: Manual Setup

1. Create `.speakeasy/workflow.yaml`:
```yaml
workflowVersion: 1.0.0
sources:
  docs-source:
    inputs:
      - location: {{your_api_spec}}
    overlays:
      - location: https://raw.githubusercontent.com/{{your_sdk_repo}}/codeSamples.yaml
    output: openapi.yaml
targets: {}
```

2. Create `.github/workflows/sdk_generation.yaml`:
```yaml
name: Generate
permissions:
  checks: write
  contents: write
  pull-requests: write
  statuses: write
"on":
  workflow_dispatch:
    inputs:
      force:
        description: Force generation of SDKs
        type: boolean
        default: false
  schedule:
    - cron: 0 0 * * *
jobs:
  generate:
    uses: speakeasy-api/sdk-generation-action/.github/workflows/workflow-executor.yaml@v15
    with:
      force: ${{ github.event.inputs.force }}
      mode: pr
      speakeasy_version: latest
    secrets:
      github_access_token: ${{ secrets.GITHUB_TOKEN }}
      speakeasy_api_key: ${{ secrets.SPEAKEASY_API_KEY }}
```

## Final Step

Add your `SPEAKEASY_API_KEY` to repository secrets under `Settings > Secrets & Variables > Actions`. You can find your key in the Speakeasy dashboard under **API Keys**.
