---
title: Speakeasy
description: Automate your SDK usage snippets in the API playground
---

Speakeasy allows you to automatically display SDK code snippets in your Mintlify API reference documentation's [interactive playground](https://mintlify.com/docs/api-playground/overview).

## Setup Steps

### 1. Configure Your SDK Repository

Add this to the `targets` section in your SDK repo's `.speakeasy/workflow.yaml`:

```yaml .speakeasy/workflow.yaml
targets:
  my-target:
    target: typescript
    source: my-source
    codeSamples:
      output: codeSamples.yaml
```

### 2. Set Up Your Documentation Repository

You can set up your documentation repository in two ways:

#### Option A: Using the CLI (Recommended)

1. Run these commands:
```bash
speakeasy configure sources
speakeasy configure github
```

2. Follow the prompts to:
   - Set up your source OpenAPI spec
   - Add the code snippet overlay
   - Specify the output spec path

#### Option B: Manual Setup

1. Create `.speakeasy/workflow.yaml`:
```yaml
workflowVersion: 1.0.0
sources:
  docs-source:
    inputs:
      - location: {{your_api_spec}}
    overlays:
      - location: https://raw.githubusercontent.com/{{your_sdk_repo_1}}/codeSamples.yaml
      - location: https://raw.githubusercontent.com/{{your_sdk_repo_2}}/codeSamples.yaml
      - location: https://raw.githubusercontent.com/{{your_sdk_repo_3}}/codeSamples.yaml
    output: openapi.yaml
targets: {}
```

2. Create `.speakeasy/workflows/sdk_generation.yaml`:
```yaml
name: Generate
permissions:
  checks: write
  contents: write
  pull-requests: write
  statuses: write
"on":
  workflow_dispatch:
    inputs:
      force:
        description: Force generation of SDKs
        type: boolean
        default: false
  schedule:
    - cron: 0 0 * * *
jobs:
  generate:
    uses: speakeasy-api/sdk-generation-action/.github/workflows/workflow-executor.yaml@v15
    with:
      force: ${{ github.event.inputs.force }}
      mode: pr
      speakeasy_version: latest
    secrets:
      github_access_token: ${{ secrets.GITHUB_TOKEN }}
      speakeasy_api_key: ${{ secrets.SPEAKEASY_API_KEY }}
```

### 3. Add Your API Key

1. Get your Speakeasy API key from the Speakeasy dashboard's **API Keys** tab
2. Add it as `SPEAKEASY_API_KEY` in your repository's secrets (Settings > Secrets & Variables > Actions)
