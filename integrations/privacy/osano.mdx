To integrate the [Osano](https://www.osano.com/) cookie consent manager, add this to your `mint.json`:

<CodeGroup>

```json Integration options in mint.json
"integrations": {
    "osano": "SOURCE"
}
```

```json Example
"integrations": {
    "osano": "https://cmp.osano.com/2sUB2dqwqdkks/8dqwd-dwd86Â£-4a9b/osano.js"
}
```

</CodeGroup>

Replace `SOURCE` with your Osano script URL (found in the `src` attribute of your Osano code snippet, starts with `https://cmp.osano.com/`).
