---
title: "Osano"
---

To add the [Osano](https://www.osano.com/) cookie consent manager, add your Osano source URL to your `mint.json` file:

<CodeGroup>

```json mint.json
"integrations": {
    "osano": "https://cmp.osano.com/YOUR_SOURCE_URL/osano.js"
}
```

</CodeGroup>

You can find your source URL in the code snippet provided by Osano in your dashboard.
