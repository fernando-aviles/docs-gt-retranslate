## How to Add Privacy Tools to Your Docs

Want to add privacy features to your documentation? It's simple! Just follow these steps:

1. Open your `mint.json` file
2. Add an `integrations` section
3. Include your privacy platform with its source code

Here's an example of how to add Osano:

```json
  "integrations": {
    "osano": "SOURCE"
  }
```

Need a different privacy tool? Join [our community](https://mintlify.com/community) and let us know what you're looking for!</fileContent>
