To host documentation at a custom subpath on Vercel, add this configuration to `vercel.json`:

```json
{
  "rewrites": [
    {
      "source": "/docs",
      "destination": "https://[subdomain].mintlify.dev/docs"
    },
    {
      "source": "/docs/:match*",
      "destination": "https://[subdomain].mintlify.dev/docs/:match*"
    }
  ]
}
```

For more details, see [Vercel's rewrites documentation](https://vercel.com/docs/projects/project-configuration#rewrites).
