Use our API to programmatically trigger documentation updates. Additional functionality will be added over time. Share feature requests in [our community](https://mintlify.com/community).

## Authentication

Generate your API key through [the dashboard](https://dashboard.mintlify.com/settings/integrations). The key works across all deployments within your organization.
