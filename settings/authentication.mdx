Mintlify supports SAML and OIDC for SSO integration. We have specific guides for Okta and Google Workspace below.

<Note>
  SSO requires Enterprise plan. [Contact us](https://mintlify.com/enterprise)
</Note>

## Okta

<Tabs>
    <Tab title="SAML">
        <Steps>
            <Step title="Create application">
                Go to `Applications` → Create new app integration → SAML 2.0
            </Step>
            <Step title="Configure">
                Configure these settings:
                * Single sign-on URL: (from Mintlify)
                * Audience URI: (from Mintlify)
                * Name ID Format: `EmailAddress`
                * Attributes:
                    | Name | Format | Value
                    | ---- | ------ | -----
                    | `firstName` | Basic | `user.firstName` |
                    | `lastName` | Basic | `user.lastName` |
            </Step>
            <Step title="Share IdP details">
                Send us your metadata URL from the sign-on tab
            </Step>
        </Steps>
    </Tab>
    <Tab title="OIDC">
        <Steps>
            <Step title="Create application">
                Go to `Applications` → Create new app integration → OIDC → Web Application
            </Step>
            <Step title="Configure">
                Enable authorization code grant type and add Mintlify's Redirect URI
            </Step>
            <Step title="Share IdP details">
                Send us securely:
                - Client ID
                - Client secret
                - Okta instance URL (`<tenant-name>.okta.com`)
            </Step>
        </Steps>
    </Tab>
</Tabs>

## Google Workspace

<Tabs>
    <Tab title="SAML">
        <Steps>
            <Step title="Create application">
                Go to `Web and mobile apps` → `Add app` → `Add custom SAML app`
                <Frame>
                    ![](/images/gsuite-add-custom-saml-app.png)
                </Frame>              
            </Step>
            <Step title="Share IdP details">
                Send us:
                - SSO URL
                - Entity ID
                - x509 certificate
                <Frame>
                    ![](/images/gsuite-saml-metadata.png)                    
                </Frame>
            </Step>
            <Step title="Configure">
                Service provider settings:
                * ACS URL: (from Mintlify)
                * Entity ID: (from Mintlify)
                * Name ID format: `EMAIL`
                * Name ID: `Basic Information > Primary email`

                <Frame>
                    ![](/images/gsuite-sp-details.png)
                </Frame>

                Attribute mapping:
                | Google Directory | App Attribute |
                | ---------------- | ------------- |
                | `First name` | `firstName`  |
                | `Last name` | `lastName` |

                After assigning users, notify us to enable SSO.
            </Step>
        </Steps>
    </Tab>
</Tabs>
