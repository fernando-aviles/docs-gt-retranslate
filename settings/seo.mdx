## Default Meta Tags

You can set default meta tags by adding the `metadata` field to your `mint.json` file.

```json
"metadata": {
    "og:image": "link to your default meta tag image"
}
```

## Changing Meta Tags on One Page

Add your meta tags to the page's metadata as follows:

```md
---
title: 'Your Example Page Title'
'og:image': 'link to your meta tag image'
---
```

<Warning>
Meta tags with colons must be enclosed in quotes when added to the page metadata.
</Warning>

## Sitemaps

Mintlify automatically generates two important SEO files for your website:
- A `sitemap.xml` file containing all your pages
- A `robots.txt` file for search engine crawlers

To view your sitemap, add `/sitemap.xml` to the end of your website URL. We recommend submitting this sitemap to search engines to improve your site's SEO visibility.

By default, only pages included in your `mint.json` appear in the sitemap. To include hidden pages, add the following configuration to your `mint.json`:

```json
"seo": {
    "indexHiddenPages": true
}
```

## Disabling Search Engine Indexing

### Individual Pages
To prevent a specific page from being indexed by search engines, add the following to your page's [frontmatter](/page):

```md
---
noindex: true
---
```

### All Pages
To prevent all pages from being indexed, add the `metadata.robots` field to your `mint.json`:

```json
"metadata": {
   "robots": "noindex"
}
```
