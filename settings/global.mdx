## Quick Start

Every Mintlify site needs a `mint.json` file in your docs directory. This file controls how your documentation looks and behaves.

Here's a basic example to get you started:

```json Basic mint.json
{
  "name": "Your Company",
  "logo": {
    "light": "/logo/light.svg",
    "dark": "/logo/dark.svg"
  },
  "favicon": "/favicon.png",
  "colors": {
    "primary": "#0D9373",
    "dark": "#0D9373"
  },
  "navigation": [
    {
      "group": "Getting Started",
      "pages": ["introduction", "quickstart"]
    }
  ]
}
```

## Essential Settings

These are the core settings you'll want to configure first.

### Branding

<ResponseField name="name" type="string" required>
  Your company or project name. Used in the browser title and top navigation.
</ResponseField>

<ResponseField name="logo" type="Logo | string">
  Your logo. Can be a simple path to an image or an object with different logos for light and dark mode.

<CodeGroup>

```json Simple Logo
{
  "logo": "/logo.svg"
}
```

```json Light & Dark Mode
{
  "logo": {
    "light": "/logo/light.svg",
    "dark": "/logo/dark.svg",
    "href": "https://yoursite.com"
  }
}
```

</CodeGroup>
</ResponseField>

<ResponseField name="favicon" type="string">
  Path to your favicon, shown in browser tabs. SVG or PNG recommended.
</ResponseField>

### Colors

<ResponseField name="colors" type="Colors">
Define your brand colors. These are used throughout your docs for consistency.

```json
{
  "colors": {
    "primary": "#0D9373",
    "light": "#07C983",
    "dark": "#0D9373",
    "background": {
      "light": "#ffffff",
      "dark": "#000000"
    }
  }
}
```

Use a color contrast checker to ensure your colors are accessible in both light and dark modes.
</ResponseField>

### Navigation

<ResponseField name="navigation" type="Navigation[]" required>
  Define the structure of your sidebar. Group pages together and set their order.

```json
"navigation": [
  {
    "group": "Getting Started",
    "pages": ["introduction", "quickstart"]
  },
  {
    "group": "Features",
    "pages": [
      "feature-one",
      {
        "group": "Advanced Features",
        "pages": ["advanced/feature-two"]
      }
    ]
  }
]
```
</ResponseField>

## Customization

### Themes

<ResponseField name="theme" type={'"venus" | "quill" | "prism"'}>
  Choose a pre-built theme to quickly style your docs. Examples:
  - [Venus](https://starter-venus.mintlify.app)
  - [Quill](https://starter-quill.mintlify.app)
  - [Prism](https://starter-prism.mintlify.app)
</ResponseField>

### Fonts

<ResponseField name="font" type="FontConfig">
  Customize your typography. Use Google Fonts or your own custom fonts.

```json
{
  "font": {
    "headings": {
      "family": "Roboto",
      "weight": 600
    },
    "body": {
      "family": "Open Sans"
    }
  }
}
```
</ResponseField>

### Layout

<ResponseField name="layout" type={'"topnav" | "sidenav" | "solidSidenav"'} default="topnav">
  Set your preferred navigation layout:
  - `topnav`: Most common, puts the main nav at the top
  - `sidenav`: Puts all navigation in the sidebar
  - `solidSidenav`: Like sidenav but with a solid background
</ResponseField>

## Navigation Elements

### Top Navigation

<ResponseField name="topbarLinks" type="TopbarLink[]">
Add links to the top navigation bar.

```json
"topbarLinks": [
  {
    "name": "Contact Us",
    "url": "mailto:support@company.com"
  },
  {
    "name": "Help",
    "url": "/help"
  }
]
```
</ResponseField>

<ResponseField name="topbarCtaButton" type="CTA">
Add a call-to-action button to the top navigation.

```json
"topbarCtaButton": {
  "name": "Get Started",
  "url": "https://app.company.com",
  "type": "link",  // or "github"
  "style": "pill"  // or "roundedRectangle"
}
```
</ResponseField>

### Tabs & Anchors

<ResponseField name="tabs" type="Tab[]">
Create tab-based navigation at the top of your docs.

```json
"tabs": [
  {
    "name": "API Reference",
    "url": "api"
  },
  {
    "name": "Guides",
    "url": "guides"
  }
]
```
</ResponseField>

<ResponseField name="anchors" type="Anchor[]">
Add anchored sections to your navigation.

```json
"anchors": [
  {
    "name": "Community",
    "icon": "comments",
    "url": "community",
    "color": "#2564eb"
  }
]
```
</ResponseField>

## Features

### Search

<ResponseField name="search" type="SearchConfig">
Configure the search functionality.

```json
"search": {
  "prompt": "Search docs...",
  "location": "top"  // or "side"
}
```
</ResponseField>

### Feedback

<ResponseField name="feedback" type="FeedbackConfig">
Add feedback buttons to your pages.

```json
"feedback": {
  "thumbsRating": true,
  "suggestEdit": true,
  "raiseIssue": true
}
```
</ResponseField>

### Integrations

<ResponseField name="integrations" type="Integrations">
Add third-party integrations.

```json
"integrations": {
  "intercom": "your-app-id",
  "frontchat": "your-chat-id"
}
```
</ResponseField>

## Advanced Features

<Accordion title="API Configuration">
```json
{
  "api": {

      "method": "bearer",

      "mode": "show"
    }
  }
```

</Accordion>

<Accordion title="SEO Settings">
```json
{
  "seo": {
    "indexHiddenPages": false
  },
  "metadata": {
    "og:image": "https://company.com/social.jpg"
  }
}
```
</Accordion>

<Accordion title="Redirects">
```json
{
  "redirects": [
    {
      "source": "/old-page",
      "destination": "/new-page"
    }
  ]
}
```
</Accordion>
