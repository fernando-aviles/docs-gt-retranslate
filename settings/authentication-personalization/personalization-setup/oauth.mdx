## Setup

<Steps>
  <Step title="Create Info API">
    Build an API endpoint that:
    - Accepts OAuth access token
    - Returns JSON in [User](../sending-data) format
    - Note the required scopes
  </Step>
  <Step title="Configure Personalization">
    In [Mintlify dashboard](https://dashboard.mintlify.com/products/authentication), select OAuth and configure:
    - **Authorization URL**: Base URL for auth requests
    - **Client ID**: OAuth 2.0 client identifier
    - **Scopes**: Required scope array
    - **Token URL**: Token exchange URL
    - **Info API URL**: User info endpoint
  </Step>
  <Step title="Set Redirect URL">
    Add the Redirect URL from [dashboard settings](https://dashboard.mintlify.com/products/authentication) to your OAuth server's allowed redirects.
  </Step>
</Steps>

## Example

For docs hosted at `foo.com/docs`:

1. Create endpoint `api.foo.com/docs/user-info`:
   - Requires: OAuth token with `docs-user-info` scope
   - Returns: User data per Mintlify spec

2. Configure OAuth in dashboard:
```
Authorization URL: https://auth.foo.com/authorization
Client ID: ydybo4SD8PR73vzWWd6S0ObH
Scopes: ['docs-user-info']
Token URL: https://auth.foo.com/exchange
Info API URL: https://api.foo.com/docs/user-info
```

3. Add dashboard's Redirect URL to OAuth client config
