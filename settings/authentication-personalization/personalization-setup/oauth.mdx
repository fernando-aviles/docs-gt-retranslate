</Info>

## Implementation

<Steps>
  <Step title="Create your Info API">
    Create an API endpoint accessible with an OAuth access token that responds with a JSON payload following the [UserInfo](../sending-data) format. Note the required scope(s) for this endpoint.
  </Step>
  <Step title="Configure Personalization settings">
    In your [Mintlify dashboard settings](https://dashboard.mintlify.com/products/authentication):
    - Select OAuth option
    - Configure required fields:
      - **Authorization URL**: Base authorization request URL
      - **Client ID**: OAuth 2.0 client identifier
      - **Scopes**: Required scope array
      - **Token URL**: Token exchange request URL
      - **Info API URL**: User info retrieval endpoint
  </Step>
  <Step title="Configure OAuth client">
    Add the Redirect URL from your [Mintlify dashboard settings](https://dashboard.mintlify.com/products/authentication) to your OAuth server's authorized redirect URLs.
  </Step>
</Steps>
