---
title: "Mintlify Dashboard Handshake"
description: "Use Mintlify to authenticate users"
---

<Info>
  The Mintlify Dashboard Handshake is available only for Authentication, *not*
  Personalization.
</Info>

Mintlify Dashboard Handshake allows you to authenticate and manage documentation access through your dashboard. Users with dashboard access will automatically have documentation access. This method also enables private preview deployments for authenticated users only.

<Note>
  Authentication is available by request. <a href="mailto:sales@mintlify.com">Contact sales</a> for access.
</Note>

## Implementation

<Steps>
  <Step title="Configure Authentication">
    Enable Mintlify Auth Handshake in your [dashboard authentication settings](https://dashboard.mintlify.com/products/authentication).
  </Step>
  <Step title="Add users">
    Add users who need documentation access in your [dashboard member settings](https://dashboard.mintlify.com/settings/organization/members).
  </Step>
</Steps>
