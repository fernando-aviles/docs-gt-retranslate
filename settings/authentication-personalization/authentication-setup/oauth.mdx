</Info>

## Implementation

<Steps>
  <Step title="Configure Authentication settings">
    In your [Mintlify authentication settings](https://dashboard.mintlify.com/products/authentication), select OAuth and configure:
    - **Authorization URL**: Base URL for authorization requests
    - **Client ID**: Your OAuth 2.0 client ID
    - **Scopes**: Required authorization scopes
    - **Token URL**: Base URL for token exchange
    - **Info API URL** (optional): Endpoint for user info retrieval
  </Step>
  <Step title="Set up OAuth client">
    Add the Redirect URL from your Mintlify authentication settings to your OAuth server's authorized redirect URLs.
  </Step>
  <Step title="Configure Info API (Optional)">
    To enable customization features, create an endpoint accessible with OAuth tokens that returns user info in the [UserInfo](../sending-data) format. Add this endpoint URL to your OAuth configuration.
  </Step>
</Steps>

## Example

For docs hosted at `foo.com/docs`, configure:
- **Authorization URL**: `https://auth.foo.com/authorization`
- **Client ID**: `ydybo4SD8PR73vzWWd6S0ObH`
- **Scopes**: `['docs-user-info']`
- **Token URL**: `https://auth.foo.com/exchange`
- **Info API URL**: `https://api.foo.com/docs/user-info`
