## Setup Steps

<Steps>
  <Step title="Set up OAuth in Mintlify">
    Navigate to [Authentication Settings](https://dashboard.mintlify.com/products/authentication) and select OAuth. Enter:

    - **Authorization URL**: Your OAuth authorization endpoint
    - **Client ID**: Your OAuth client ID
    - **Scopes**: Required OAuth scopes
    - **Token URL**: Your token exchange endpoint
    - **Info API URL** (optional): Endpoint to fetch user data
  </Step>
  <Step title="Add Redirect URL">
    Copy the Redirect URL from your Mintlify authentication settings and add it to your OAuth server's allowed redirects.
  </Step>
  <Step title="Configure User Info API (Optional)">
    To enable user customization:
    1. Create an API endpoint that accepts OAuth tokens
    2. Return user data in the [User](../sending-data) format
    3. Add this endpoint URL as your Info API URL in Mintlify settings
  </Step>
</Steps>

## Quick Example

Consider a documentation site at `foo.com/docs`:

```
Authorization URL: https://auth.foo.com/authorization
Client ID: ydybo4SD8PR73vzWWd6S0ObH
Scopes: ['docs-user-info']
Token URL: https://auth.foo.com/exchange
Info API URL: https://api.foo.com/docs/user-info
```

Add the Mintlify Redirect URL to your OAuth server, and you're ready to go.
