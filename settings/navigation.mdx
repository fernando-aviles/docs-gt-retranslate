## Navigation Components

The navigation system in Mintlify consists of several key components that help organize your documentation structure. Let's explore each component and how to configure them effectively.

### Tabs Overview
Tabs provide a high-level way to separate different sections of your documentation. They appear prominently above the main sidebar, making them perfect for distinguishing between major documentation categories.

<Frame>
  <img className="block dark:hidden rounded-md" src="/images/tabs-light.png" />
  <img className="hidden dark:block rounded-md" src="/images/tabs-dark.png" />
</Frame>

### Configuring Tabs
Add tabs using the `tabs` field in your `mint.json` file. Each tab can either link to a content folder or an external URL.

```json Basic Tab Configuration
"tabs": [
    {
        "name": "API References",
        "url": "api-references"
    },
    {
        "name": "Content",
        "url": "content"
    }
]
```

For external links, simply use the complete URL:

```json Tab with External Link
"tabs": [
    {
        "name": "Blog",
        "url": "https://your-website.com/blog"
    }
]
```

### Primary Tab Settings
The default "Documentation" tab can be customized by adding the `primaryTab` field to your `mint.json`:

```json Primary Tab Configuration
"primaryTab": {
    "name": "Home"
}
```

## Using Anchors

### Anchor Overview
Anchors offer another navigation option for your documentation. They can direct users to specific documentation sections or external resources.

<Frame>
  <img className="block dark:hidden rounded-md" src="/images/anchor-light.png" />
  <img className="hidden dark:block rounded-md" src="/images/anchor-dark.png" />
</Frame>

### Anchor Configuration
Configure anchors in your `mint.json` file using the `anchors` field:

```json Basic Anchor Configuration
"anchors": [
    {
        "name": "API References",
        "icon": "code",
        "url": "api-references"
    },
    {
        "name": "Content",
        "icon": "pencil",
        "url": "content"
    }
]
```

For external links in anchors:

```json Anchor with External Link
"anchors": [
    {
        "name": "Contact us",
        "icon": "envelope",
        "url": "https://mintlify.com/contact-us"
    }
]
```

### Default Anchor Customization
Customize the default "Documentation" anchor using the `topAnchor` field:

```json Top Anchor Configuration
"topAnchor": {
    "name": "Home",
    "icon": "building"
}
```

## Sidebar Navigation

### Basic Structure
The sidebar is configured through the `navigation` property in your `mint.json`. Files are organized into groups for better structure.

<Info>
  You must explicitly add pages to your `mint.json` to make them appear in the sidebar. Adding a file to your docs directory alone won't make it show up automatically.
</Info>

```json Basic Sidebar Structure
"navigation": [
    {
        "group": "Getting Started",
        "pages": ["quickstart"]
    }
]
```

### Nested Groups
Create deeper navigation structures by nesting groups within groups:

```json Nested Navigation Structure
"navigation": [
    {
        "group": "Getting Started",
        "pages": [
            "quickstart",
            {
                "group": "Nested Reference Pages",
                "pages": ["nested-reference-page"]
            }
        ]
    }
]
```

### Working with Folders

#### Basic Folder Organization
Organize your content by placing MDX files in folders and referencing their paths in `mint.json`.

<Warning>
  The folder name `api` is reserved at the root level. Use `api-reference` or nest it within another folder instead.
</Warning>

Example folder structure for the URL `https://yoursite.com/your-folder/your-page`:
```
docs/
  └── your-folder/
      └── your-page.mdx
```

Configure this in your navigation:
```json Folder Navigation
"navigation": [
    {
        "group": "Group Name",
        "pages": ["your-folder/your-page"]
    }
]
```

### Hidden Pages
Pages not listed in your `mint.json` navigation are hidden from the sidebar but remain accessible via direct links. By default, these pages aren't indexed for search.

To include hidden pages in search:
```json Search Configuration
{
    "seo": {
        "indexHiddenPages": true
    }
}
```

## Topbar Configuration

### Adding Links
Enhance your topbar with custom links using the `topbarLinks` field:

<Frame>
  <img className="block rounded-md" src="/images/topbar-links.png" />
</Frame>

```json Topbar Links
"topbarLinks": [
    {
        "name": "Community",
        "url": "https://mintlify.com/community"
    }
]
```

### Call-to-Action Button

#### Basic CTA
Add a call-to-action button to your topbar:

<Frame>
  <img className="block rounded-md" src="/images/topbar-cta-get-started.png" />
</Frame>

```json Basic CTA Button
"topbarCtaButton": {
    "name": "Get Started",
    "url": "https://mintlify.com/get-started"
}
```

#### GitHub Integration
Link directly to your GitHub repository:

<Frame>
  <img className="block rounded-md" src="/images/topbar-cta-github.png" />
</Frame>

```json GitHub CTA Configuration
"topbarCtaButton": {
    "type": "github",
    "url": "https://github.com/mintlify/docs"
}
```
