The `<ResponseField>` component defines API return values and can be used to list types in documentation.

<ResponseField name="response" type="string" required>
  A response field example
</ResponseField>

## Props

<ResponseField name="name" type="string" required>
  The name of the response value.
</ResponseField>

<ResponseField name="type" type="string" required>
  Expected type of the response value
</ResponseField>

<ResponseField name="default" type="string">
  The default value.
</ResponseField>

<ResponseField name="required" type="boolean">
  Show "required" beside the field name.
</ResponseField>
