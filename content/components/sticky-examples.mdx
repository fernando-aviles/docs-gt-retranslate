`<RequestExample>` and `<ResponseExample>` components stick code blocks to the top-right of desktop views, while displaying as regular code blocks on mobile devices.

## Request Example

`<RequestExample>` works like [CodeGroup](/content/components/code-groups) but displays in the right sidebar. Include a name for each code block inside.

<RequestExample>
````md RequestExample Example
<RequestExample>

```bash Request
  curl --request POST \
    --url https://dog-api.kinduff.com/api/facts
```

</RequestExample>
````
</RequestExample>

## Response Example

`<ResponseExample>` displays below `<RequestExample>` in the sidebar.

<ResponseExample>
````md ResponseExample Example
<ResponseExample>

```json Response
{ "status": "success" }
```

</ResponseExample>
````
