Callouts help highlight important information in your content. Here are the available types:

<Note>Use Notes for additional context</Note>
<Warning>Use Warnings for cautionary information</Warning>
<Info>Use Info for important details</Info>
<Tip>Use Tips for helpful suggestions</Tip>
<Check>Use Check for confirmation or success states</Check>

```jsx
// Example usage
<Note>Your message here</Note>
<Warning>Your message here</Warning>
<Info>Your message here</Info>
<Tip>Your message here</Tip>
<Check>Your message here</Check>
```
