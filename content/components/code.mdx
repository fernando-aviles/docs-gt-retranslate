## Code Blocks

Use [fenced code blocks](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) with three backticks to display code. Add a language identifier for syntax highlighting via [Prism](https://prismjs.com/#supported-languages).

```javascript
const greeting = "Hello, World!";
console.log(greeting);
```

## Additional Features

- **Names**: Add text after the language for naming your code example
- **Line Highlighting**: Use `{1,3-5}` syntax to highlight specific lines
- **Multiple Examples**: Use Code Groups for multiple examples

```javascript Example with Highlighting {1,3}
const greeting = "Hello, World!";
function sayHello() {
  console.log(greeting);
}
```
