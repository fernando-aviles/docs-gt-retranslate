The `Update` component is used to keep track of changes and updates.

  <Frame>
    <img
      className="block"

  #### Features
  - Responsive design
  - Sticky section for each changelog
</Update>

<Update label="2024-10-11" description="v0.1.0">
  ```md
  <Update label="2024-10-12" description="v0.1.1">
    Add changelog content here with optional label and description
  </Update>
  ```

  Use multiple `Update` components to create changelogs.
</Update>

<Tip>
  Each `label` creates an anchor and appears in the table of contents.
</Tip>

## Props

<ResponseField name="label" type="string" required>
  Label in the changelog, on the sticky left side.
</ResponseField>

<ResponseField name="description" type="string">
  Description below the label.
</ResponseField>
