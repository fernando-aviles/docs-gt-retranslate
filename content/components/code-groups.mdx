---
title: "Code Groups"
description: "The CodeGroup component lets you combine code blocks in a display separated by tabs"
icon: "rectangle-code"
---

Wrap multiple [Code Blocks](/content/components/code) with the `<CodeGroup>` component to display them in tabs. Each Code Block requires a filename for the tab labels.

<CodeGroup>

```javascript helloWorld.js
console.log("Hello World");
```

```python hello_world.py
print('Hello World!')
```

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

</CodeGroup>
