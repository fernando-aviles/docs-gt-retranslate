---
title: "Accordion Groups"
description: "Group multiple accordions into a single display."
icon: "table-rows"
---

Wrap multiple `<Accordion>` components with `<AccordionGroup>` to group them together.

<AccordionGroup>
  <Accordion title="Basic Example">
    You can put any content inside Accordions.
  </Accordion>

  <Accordion title="With Icon" icon="alien-8bit">
    Add icons using the `icon` prop.
  </Accordion>
  
  <Accordion title="Learn More">
    See the [Accordion](/content/components/accordions) docs for all available props.
  </Accordion>
</AccordionGroup>

<RequestExample>
```jsx
<AccordionGroup>
  <Accordion title="Basic Example">
    You can put any content inside Accordions.
  </Accordion>

  <Accordion title="With Icon" icon="alien-8bit">
    Add icons using the `icon` prop.
  </Accordion>

  <Accordion title="Learn More">
    See the Accordion docs for all available props.
  </Accordion>
</AccordionGroup>
```
</RequestExample>

Note: `AccordionGroup` does not accept any props.
