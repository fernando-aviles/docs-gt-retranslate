Mintlify menghasilkan format yang dioptimalkan dan menyediakan pintasan yang membantu pengguna mendapatkan respons yang lebih cepat dan akurat saat menggunakan dokumentasi Anda sebagai konteks untuk LLM dan alat AI.

## Menu kontekstual

Berikan akses cepat ke konten yang dioptimalkan untuk AI dan integrasi langsung dengan alat AI populer dari menu kontekstual di halaman Anda.

* **Salin halaman**: Menyalin halaman saat ini sebagai Markdown untuk ditempel sebagai konteks ke dalam alat AI.
* **Lihat sebagai Markdown**: Membuka halaman saat ini sebagai Markdown.
* **Buka di ChatGPT**: Membuat percakapan ChatGPT dengan halaman saat ini sebagai konteks.
* **Buka di Claude**: Membuat percakapan Claude dengan halaman saat ini sebagai konteks.

### Mengaktifkan menu kontekstual

Tambahkan bidang `contextual` ke `docs.json` Anda dan tentukan opsi mana yang ingin Anda sertakan dalam menu Anda.

```json
{
 "contextual": {
   "options": [
     "copy",
     "view",
     "chatgpt",
     "claude"
   ]
 }
}
```

## /llms.txt

File [/llms.txt](https://llmstxt.org) adalah standar industri yang membantu LLM tujuan umum mengindeks lebih efisien, mirip dengan bagaimana sitemap membantu mesin pencari.

Setiap situs dokumentasi secara otomatis meng-host file `/llms.txt` di root yang mencantumkan semua halaman yang tersedia dalam dokumentasi Anda. Alat AI dapat menggunakan file ini untuk memahami struktur dokumentasi Anda dan menemukan konten yang relevan dengan prompt pengguna.

<PreviewButton href="https://mintlify.com/docs/llms.txt">Buka llms.txt untuk situs ini</PreviewButton>

## /llms-full.txt

File `/llms-full.txt` menggabungkan seluruh situs dokumentasi Anda menjadi satu file sebagai konteks untuk alat AI.

Setiap situs dokumentasi secara otomatis meng-host file `/llms-full.txt` di root.

<PreviewButton href="https://mintlify.com/docs/llms-full.txt">Buka llms-full.txt untuk situs ini</PreviewButton>

## Menghasilkan versi Markdown dari halaman

Markdown menyediakan teks terstruktur yang dapat diproses alat AI lebih efisien daripada HTML, yang menghasilkan waktu respons yang lebih baik dan penggunaan token yang lebih rendah.

### .md extension

Tambahkan `.md` ke URL halaman untuk menampilkan versi Markdown dari halaman tersebut.

<PreviewButton href="https://mintlify.com/docs/quickstart.md">Buka quickstart.md</PreviewButton>

### Pintasan Command + C

Pilih Command + C (Ctrl + C di Windows) untuk menyalin halaman apa pun sebagai Markdown.
