The `<RequestExample>` dan `<ResponseExample>` menempelkan blok kode di kanan atas halaman bahkan saat Anda menggulir. Komponen ini berfungsi di semua halaman bahkan jika Anda tidak menggunakan API playground.

`<RequestExample>` dan `<ResponseExample>` tampil seperti blok kode biasa di perangkat mobile.

## Contoh Request

The `<RequestExample>` komponen bekerja mirip dengan [CodeGroup](/content/components/code-groups), tetapi menampilkan konten request di sidebar kanan. Dengan demikian, Anda dapat menempatkan beberapa blok kode di dalam `<RequestExample>`.

Harap tetapkan nama pada setiap blok kode yang Anda tempatkan di dalam RequestExample.

<RequestExample>
  ````md RequestExample Example
  <RequestExample>

  ```bash Request
    curl --request POST \
      --url https://dog-api.kinduff.com/api/facts
  ```

  </RequestExample>
  ````
</RequestExample>

## Contoh Response

The `<ResponseExample>` komponen sama dengan `<RequestExample>` tetapi akan muncul di bawahnya.

<ResponseExample>
  ````md ResponseExample Example
  <ResponseExample>

  ```json Response
  { "status": "success" }
  ```

  </ResponseExample>
  ````
</ResponseExample>
