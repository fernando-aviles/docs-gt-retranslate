Cukup tambahkan`<AccordionGroup>`di sekitar komponen`<Accordion>`yang ada.

<AccordionGroup>
  <Accordion title="FAQ without Icon">
    Anda dapat menempatkan komponen lain di dalam Akordeon.

    ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
  </Accordion>

  <Accordion title="FAQ with Icon" icon="alien-8bit">
    Lihat dokumentasi[Accordion](/content/components/accordions)untuk semua props yang didukung.
  </Accordion>

  <Accordion title="FAQ without Icon">
    Lihat dokumentasi[Accordion](/content/components/accordions)untuk semua props yang didukung.
  </Accordion>
</AccordionGroup>
