Komponen `<ResponseField>` dirancang untuk mendefinisikan nilai pengembalian dari API. Banyak dokumentasi juga menggunakan `<ResponseField>` pada halaman ketika Anda perlu mendaftar tipe sesuatu.

<ResponseField name="response" type="string" required>
  Contoh response field
</ResponseField>

<RequestExample>
  ```jsx ResponseField Example
  <ResponseField name="response" type="string" required>
    A response field example
  </ResponseField>
  ```
</RequestExample>

## Props

<ResponseField name="name" type="string" required>
  Nama dari nilai respons.
</ResponseField>

<ResponseField name="type" type="string" required>
  Tipe yang diharapkan dari nilai respons
</ResponseField>

<ResponseField name="default" type="string">
  Nilai default.
</ResponseField>

<ResponseField name="required" type="boolean">
  Tampilkan "required" di samping nama field.
</ResponseField>
