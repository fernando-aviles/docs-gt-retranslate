## Ikhtisar

Model Context Protocol (MCP) adalah protokol terbuka yang menghubungkan fungsi-fungsi Anda ke LLM dan aplikasi AI. Dengan integrasi Mintlify, Anda dapat secara otomatis menghasilkan server MCP dari dokumentasi yang ada atau spesifikasi OpenAPI, memungkinkan interaksi berbasis AI yang mulus dengan produk Anda.

## Panduan Penggunaan Cepat

Setiap dokumentasi publik yang dihosting di Mintlify dapat diekstrak sebagai server MCP menggunakan perintah sederhana:

```bash
npx mint-mcp add <your-subdomain-or-domain>
```

Examples:

```bash
# Using subdomain
npx mint-mcp add mintlify

# Using domain
npx mint-mcp add mintlify.com
```

### Autentikasi

Ketika Anda menjalankan perintah, Anda perlu menyediakan dua kunci API, `External Admin Key` dan `Assistant API Key`.

Ini dapat ditemukan di [Mintlify Dashboard](https://dashboard.mintlify.com/settings/organization/api-keys) Anda di bawah **Settings > API Keys**.

## Mengkonfigurasi Server MCP Anda

Sekarang mari kita lihat cara mengkonfigurasi server MCP Anda.

### Fungsionalitas Default

Semua server MCP menyertakan alat `search` secara default, memungkinkan pengguna untuk menanyakan informasi di seluruh dokumentasi Anda.

### Menambahkan Fungsi API

Jika Anda memiliki spesifikasi OpenAPI, Anda dapat mengekspos endpoint tertentu sebagai alat MCP dengan menggunakan ekstensi `x-mcp` baik di tingkat file atau endpoint.

#### Konfigurasi Tingkat File

Aktifkan MCP untuk semua endpoint dalam file spesifikasi:

```json
{
  "openapi": "3.1.0",
  "x-mcp": {
    "enabled": true
  },
  // Other OpenAPI content
}
```

#### Konfigurasi Tingkat Endpoint

Aktifkan MCP hanya untuk endpoint tertentu:

```json
{
  "paths": {
    "/api/v1/users": {
      "x-mcp": {
        "enabled": true
      },
      // Endpoint configuration
    }
  }
}
```

### Penanganan Autentikasi

Jika spesifikasi OpenAPI Anda mendefinisikan autentikasi menggunakan securitySchemes, metode autentikasi ini akan secara otomatis diterapkan ke server MCP Anda.

### Memantau Server MCP Anda

Setelah mempublikasikan perubahan Anda, Anda dapat melihat semua alat MCP yang tersedia di bagian **Available Tools** pada halaman server MCP di dashboard Anda.

## Distribusi

### Instalasi Pengguna

Pengguna Anda dapat menginstal dan menggunakan server MCP Anda dengan:

```bash
npx mint-mcp add <your-subdomain-or-domain>
```

Ini memberikan mereka server MCP yang siap digunakan yang menghubungkan fungsi-fungsi yang didokumentasikan ke aplikasi AI.
