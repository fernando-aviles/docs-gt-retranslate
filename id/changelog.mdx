<Update label="Februari 2025">
  ## Skema konfigurasi baru `docs.json`

  <Frame>
    ![docs.json screenshot](/images/changelog/docsjson.png)
  </Frame>

  Kami telah memperkenalkan `docs.json` skema baru sebagai pengganti `mint.json`, untuk mendukung versi multi-level yang lebih baik, pemahaman visual yang lebih mudah, dan terminologi yang lebih konsisten. Untuk informasi lebih lanjut tentang apa yang berubah, [lihat blog kami](https://mintlify.com/blog/refactoring-mint-json-into-docs-json).

  Tingkatkan dari `mint.json` ke `docs.json` dengan langkah-langkah berikut:

  1. Pastikan CLI Anda adalah versi terbaru

  ```
  npm i mint@latest -g
  ```

  1. Di repositori dokumen Anda, jalankan

  ```
  mint upgrade
  ```

  1. Hapus file mint.json lama Anda dan push perubahan Anda

  ## Pemeriksaan CI
