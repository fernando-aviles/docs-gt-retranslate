Tambahkan CSS & JS kustom ke dokumentasi Anda untuk menyesuaikan tampilan dan nuansa sepenuhnya.

## CSS Kustom

Tambahkan sejumlah file CSS ke repositori Anda dan nama kelas yang didefinisikan akan diterapkan dan tersedia di semua file MDX Anda.

### Menambahkan style.css

Misalnya, Anda dapat menambahkan file`style.css`berikut untuk menyesuaikan gaya navbar dan footer.

```css
#navbar {
  background: "#fffff2";
  padding: 1rem;
}

footer {
  margin-top: 2rem;
}
```

### Menggunakan Identifier

Mintlify memiliki serangkaian identifier umum untuk membantu Anda menandai elemen penting dari UI. Beberapa, tetapi tidak semua tercantum dalam berikut ini:

`#topbar-cta-button` `#navbar` `#sidebar` `#content-area` `#table-of-contents`

<Tip>
  Gunakan inspect element untuk menemukan referensi ke elemen yang ingin Anda sesuaikan.
</Tip>

<Warning>
  Referensi dan gaya elemen umum dapat berubah seiring berkembangnya platform. Harap gunakan gaya kustom dengan hati-hati.
</Warning>

## JS Kustom

JS Kustom memungkinkan Anda menambahkan kode yang dapat dieksekusi secara global. Ini setara dengan menambahkan tag`<script>`dengan kode JS ke setiap halaman.

### Menambahkan JavaScript Kustom

Setiap file`.js`di dalam direktori konten dokumentasi Anda akan disertakan di setiap halaman dokumentasi. Misalnya, Anda dapat menambahkan file`ga.js`berikut untuk mengaktifkan[Google Analytics](https://marketingplatform.google.com/about/analytics)di seluruh dokumentasi.

```js
window.dataLayer = window.dataLayer || [];
function gtag() {
  dataLayer.push(arguments);
}
gtag('js', new Date());

gtag('config', 'TAG_ID');
```

<Warning>
  Harap gunakan dengan hati-hati untuk tidak memperkenalkan kerentanan keamanan.
</Warning>
