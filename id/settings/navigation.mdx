Properti `navigation` mengontrol hierarki dokumentasi Anda.

## Halaman

Jika Anda tidak menginginkan hierarki, Anda dapat mendefinisikan halaman dalam bidang `navigation` Anda.

Setiap entri dari array `pages` harus berupa path ke file yang ada dalam repo Anda.
Perhatikan bahwa Anda tidak perlu menambahkan `.mdx` ke path file.

```json
{
  "navigation": {
    "pages": [
      "overview",
      "quickstart",
      "advanced/components",
      "advanced/integrations"
    ]
  }
}
```

## Grup

Grup memungkinkan Anda mengelompokkan halaman Anda. Grup juga dapat disarangkan satu sama lain.

```json
{
  "navigation": {
    "groups": [
      {
        "group": "Getting Started",
        "pages": [
          "quickstart",
          {
            "group": "Editing",
            "icon": "pen-paintbrush",
            "pages": ["development", "web-editor"]
          }
        ]
      },
      {
        "group": "Writing Content",
        "pages": ["writing-content/page", "writing-content/text"]
      }
    ]
  }
}
```

## Tab

Tab membantu membedakan antara topik atau bagian yang berbeda dari dokumentasi Anda.

  <img className="block dark:hidden rounded-md" src="/images/tabs-light.png" />

  <img className="hidden dark:block rounded-md" src="/images/tabs-dark.png" />

```json docs.json
"navigation": {
  "tabs": [
    {
      "tab": "API References",
      "pages": [
        "api-reference/get",
        "api-reference/post",
        "api-reference/delete"
      ]
    },
    {
      "tab": "SDKs",
      "pages": [
        "sdk/fetch",
        "sdk/create",
        "sdk/delete",
      ]
    },
    {
      "tab": "Blog",
      "href": "https://external-link.com/blog"
    }
  ]
}
```

## Jangkar

Jangkar adalah cara lain untuk membagi konten Anda.

<Frame>
  <img className="block dark:hidden rounded-md" src="/images/anchor-light.png" />

  <img className="hidden dark:block rounded-md" src="/images/anchor-dark.png" />
</Frame>

Konfigurasinya sangat mirip dengan konfigurasi tab.

```json docs.json
"navigation": {
  "anchors": [
    {
      "anchor": "API References",
      "pages": [
        "api-reference/get",
        "api-reference/post",
        "api-reference/delete"
      ]
    },
    {
      "anchor": "SDKs",
      "pages": [
        "sdk/fetch",
        "sdk/create",
        "sdk/delete",
      ]
    },
    {
      "anchor": "Blog",
      "href": "https://external-link.com/blog"
    }
  ]
}
```

## Tautan Global

Jika Anda ingin menggunakan tab atau jangkar hanya untuk tujuan mendaftar tautan eksternal, gunakan properti `navigation.global`.

```json
"navigation": {
  "global": {
    "anchors": [
      {
        "anchor": "Slack",
        "icon": "slack",
        "href": "https://slack.com
      },
      {
        "anchor": "API Reference",
        "icon": "code",
        "href": "https://api-reference.com"
      }
    ]
  }
}
```

### Halaman Tersembunyi

File MDX yang tidak termasuk dalam `docs.json` tidak akan muncul di sidebar tetapi dapat diakses dengan menautkan langsung ke mereka.

Halaman tersembunyi tidak diindeks untuk pencarian dalam dokumen Anda secara default. Jika Anda ingin mengganti perilaku ini, Anda dapat mengatur atribut

[`seo.indexing`](/settings/global#param-indexing) dalam `docs.json` ke `navigable`.

## Navbar

### Tautan

Tambahkan tautan ke topbar dengan bidang `links` dalam file `docs.json`.

<Frame>
  <img className="block rounded-md" src="/images/topbar-links.png" />
</Frame>

Bidang `links` mendukung bidang berikut: `label`, `href`.

```json
"navbar": {
  "links": [
    {
      "label": "Community",
      "href": "https://mintlify.com/community"
    }
  ]
}
```

### Tombol Utama

Sesuaikan tombol utama di topbar menggunakan bidang `primary`
.

<Frame>
  <img className="block rounded-md" src="/images/topbar-cta-get-started.png" />
</Frame>

Bidang `primary` mendukung bidang berikut: `label`, `href`, `type`. Untuk informasi lebih lanjut tentang opsi untuk bidang-bidang ini, lihat [skema docs.json](/settings/global#param-navbar).

```json
"navbar": {
  "primary": {
    "type": "button",
    "label": "Get Started",
    "href": "https://mintlify.com/get-started"
  }
}
```

#### GitHub

Anda juga dapat mengonfigurasi tombol CTA untuk menautkan langsung ke repositori GitHub Anda. Gunakan bidang `primary` dengan `type` diatur ke `github`.

<Frame>
  <img className="block rounded-md" src="/images/topbar-cta-github.png" />
</Frame>

```json
"navbar": {
  "primary": {
    "type": "github",
    "label": "GitHub",
    "href": "https://github.com/mintlify/docs"
  }
}
```
