Semua halaman dokumentasi secara otomatis di-host dalam berbagai format markdown untuk memudahkan:

* Pengguna akhir untuk memasukkan dokumentasi Anda ke dalam model bahasa besar (LLM) untuk pengambilan informasi spesifik
* LLM untuk mengindeks konten Anda secara efisien tanpa mengurai HTML atau JavaScript

## Format yang didukung

Kami mendukung standar industri seperti [/llms.txt](https://llmstxt.org) serta format markdown tambahan.

* **/llms.txt**: File markdown dari indeks terstruktur dokumentasi Anda secara otomatis di-host di /llms.txt. Membantu LLM tujuan umum mengindeks lebih efisien (misalnya sitemap untuk AI).
* **/llms-full.txt**: File markdown dari semua konten dokumentasi Anda secara otomatis di-host di /llms-full.txt. Membantu pengguna memuat konteks lengkap ke dalam alat AI dalam satu tautan.
* **dukungan .md**: Tambahkan .md ke halaman dokumentasi individual untuk melihat versi markdown dari halaman tersebut. Membantu memuat halaman individual ke dalam alat AI dengan efisiensi token & waktu inferensi yang lebih baik.
