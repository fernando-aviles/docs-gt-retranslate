Mintlify terintegrasi dengan API GitHub, memungkinkan sinkronisasi antara
dokumentasi Anda dan repositori GitHub Anda. Integrasi ini dimungkinkan melalui
penggunaan
[Aplikasi GitHub](https://docs.github.com/en/developers/apps/getting-started-with-apps/about-apps#about-github-apps).

## Instalasi

Untuk memulai, Anda perlu menginstal Aplikasi GitHub Mintlify ke akun GitHub
di mana repo dokumentasi Anda berada. Menginstal Aplikasi GitHub memerlukan
kepemilikan organisasi atau izin admin dalam repositori. Jika Anda tidak memiliki
izin yang diperlukan, pemilik repositori perlu menyetujui permintaan tersebut.
Anda dapat mengakses halaman instalasi dengan masuk ke
[dasbor Mintlify](https://dashboard.mintlify.com).

Untuk Aplikasi GitHub, Anda dapat memilih untuk hanya memberikan izin ke satu repositori.
Kami sangat menyarankan Anda melakukannya karena kami hanya membutuhkan akses ke repositori
di mana dokumentasi Anda dihosting.

## Izin

Selama instalasi aplikasi GitHub kami, Anda akan diminta untuk memberikan izin
tertentu:

* Akses baca dan tulis ke `checks`, `contents`, `deployments`, `pull requests`,
  dan `workflows`

Izin-izin ini dimanfaatkan untuk memberikan pengalaman yang mulus saat mengelola
dokumentasi Anda.

Ketika Anda membuat commit ke cabang yang Anda konfigurasi sebagai cabang
penerapan dokumentasi, kami mengambil konten dari file yang diubah untuk
memperbarui dokumentasi Anda. Untuk mencapai ini kami membutuhkan akses baca ke `contents`.

Ketika pull request dibuat, kami membuat pemeriksaan dan pratinjau deployment yang merupakan alasan mengapa kami membutuhkan akses tulis ke`checks` dan `deployments`.

Di dalam editor web Mintlify, Mintlify bekerja atas nama Anda untuk membuat cabang dan pull request yang merupakan alasan mengapa kami membutuhkan akses tulis ke`pull requests` dan
`workflows`.

<Info>
  Jika Anda khawatir tentang izin tulis, Aplikasi GitHub hanya akan memiliki akses ke repositori yang Anda berikan akses*dan* jika Anda memiliki perlindungan cabang*tidak dapat* langsung push ke cabang Anda - ini mematuhi aturan perlindungan cabang Anda.
</Info>

## Instalasi Repositori

Saat menginstal aplikasi GitHub kami, Anda akan diminta untuk memilih semua repositori atau sebagian dari repositori tersebut. Pilihan ini dapat diubah kapan saja dengan membuka
[pengaturan aplikasi GitHub](https://github.com/apps/mintlify/installations/new).
