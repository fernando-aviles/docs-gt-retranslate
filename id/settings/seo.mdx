## Meta tag default

Anda dapat mengatur meta tag default dengan menambahkan`metadata` bidang ke`docs.json`.

```json
"seo": {
    "metatags": {
        "og:image": "link to your default meta tag image"
    }
}
```

## Mengubah meta tag pada satu halaman

Tambahkan meta tag Anda ke metadata halaman seperti ini:

```md
---
title: 'Your Example Page Title'
'og:image': 'link to your meta tag image'
---
```

<Warning>
  Meta tag dengan tanda titik dua perlu diketik dengan tanda kutip di sekitarnya ketika Anda memasukkannya ke dalam metadata halaman.
</Warning>

## Sitemap

Mintlify secara otomatis menghasilkan`sitemap.xml` file dan`robots.txt` file. Cukup tambahkan`/sitemap.xml` di akhir URL situs web Anda untuk melihat sitemap Anda. Kami merekomendasikan untuk mengirimkan sitemap ke mesin pencari untuk meningkatkan SEO.

Hanya halaman yang termasuk dalam`docs.json` yang dimasukkan secara default. Untuk menyertakan tautan tersembunyi, tambahkan`seo.indexing` ke`docs.json`:

```json
"seo": {
    "indexing": all
}
```

## Menonaktifkan pengindeksan

Jika Anda ingin menghentikan halaman agar tidak diindeks oleh mesin pencari, Anda dapat menyertakan hal berikut dalam[frontmatter](/page) halaman Anda:

```
---
noindex: true
---
```

Anda juga dapat menentukan`noindex` untuk semua halaman dalam dokumentasi Anda dengan mengatur`metadata.robots` bidang ke`"noindex"` dalam`docs.json`:

```json
"seo": {
    "metatags": {
      "robots": "noindex"
    }
  }
```
