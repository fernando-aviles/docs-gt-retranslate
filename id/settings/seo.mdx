Platform secara otomatis menghasilkan sebagian besar meta tag.

Namun, Anda dapat sepenuhnya menyesuaikannya dengan menambahkan bidang [metadata](settings#param-metadata) ke `docs.json` Anda atau frontmatter halaman.

## Meta tag global

Anda dapat mengatur meta tag default dengan menambahkan bidang `metadata` ke `docs.json` Anda.

```json
"seo": {
    "metatags": {
        "og:image": "link to your default meta tag image"
    }
}
```

## Meta tag khusus halaman

Tambahkan meta tag Anda ke metadata halaman seperti ini:

```md
---
title: 'Your Example Page Title'
'og:image': 'link to your meta tag image'
---
```

<Warning>
  Meta tag dengan titik dua perlu diketik dengan tanda kutip di sekitarnya ketika Anda memasukkannya ke dalam metadata halaman.
</Warning>

## Sitemap

Mintlify secara otomatis menghasilkan file `sitemap.xml` dan file `robots.txt`. Cukup navigasi ke `/sitemap.xml` untuk melihat sitemap Anda.

Hanya halaman yang termasuk dalam `docs.json` Anda yang dimasukkan secara default. Untuk menyertakan tautan tersembunyi, tambahkan `seo.indexing` ke `docs.json` Anda:

```json
"seo": {
    "indexing": all
}
```

## Menonaktifkan pengindeksan

Jika Anda ingin menghentikan pengindeksan halaman oleh mesin pencari, Anda dapat menyertakan hal berikut di [frontmatter](pages) halaman Anda:

```
---
noindex: true
---
```

Anda juga dapat menentukan `noindex` untuk semua halaman di dokumen Anda dengan mengatur bidang `metadata.robots` ke `"noindex"` di `docs.json` Anda:

```json
"seo": {
    "metatags": {
      "robots": "noindex"
    }
  }
```
