Preview Deployments memungkinkan Anda untuk melihat pratinjau perubahan pada docs Anda dalam deployment langsung tanpa menggabungkan perubahan tersebut ke cabang produksi Anda.

Jika Anda telah membuat pull atau merge request saat menggunakan Git, URL pratinjau yang dihasilkan akan tersedia sebagai komentar dari bot Mintlify di PR. URL ini akan selalu mencerminkan perubahan deployment terbaru.

## Berbagi Preview Deployments

Setelah menghasilkan URL pratinjau, Anda dapat membagikan tautan tersebut kepada siapa pun untuk dilihat.

### Preview Deployments Terotentikasi

Jika Anda ingin membatasi akses ke Preview Deployments Anda, Anda dapat mengatur otentikasi deployment pratinjau. Saat menggunakan metode otentikasi ini, hanya pengguna yang terotentikasi Mintlify yang akan dapat mengakses Preview Deployments Anda.

Untuk mengatur otentikasi deployment pratinjau, silakan [contact sales](mailto:sales@mintlify.com).
