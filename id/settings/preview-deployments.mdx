Preview Deployments memungkinkan Anda untuk melihat pratinjau perubahan pada dokumentasi Anda dalam deployment langsung tanpa menggabungkan perubahan tersebut ke branch produksi.

Jika Anda telah membuat pull atau merge request saat menggunakan Git, URL pratinjau yang dihasilkan akan tersedia sebagai komentar dari bot Mintlify di PR. URL ini akan selalu mencerminkan perubahan deployment terbaru.

## Berbagi Preview Deployments

Setelah menghasilkan URL pratinjau, Anda dapat membagikan tautan tersebut kepada siapa pun untuk dilihat.

### Preview Deployments Terautentikasi

Jika Anda ingin membatasi akses ke Preview Deployments Anda, Anda dapat mengatur autentikasi preview deployment. Saat menggunakan metode autentikasi ini, hanya pengguna Mintlify yang terautentikasi yang akan dapat mengakses Preview Deployments Anda.

Untuk mengatur autentikasi preview deployment, silakan [hubungi sales](mailto:sales@mintlify.com).
