<Note>
  Jika Anda saat ini menggunakan`mint.json` file konfigurasi lama, silakan perbarui CLI:

  ```sh
  npm i -g mintlify@latest
  ```

  Dan jalankan`upgrade` perintah baru di repositori docs Anda:

  ```sh
  mintlify upgrade
  ```

  Ini akan menghasilkan`docs.json` berdasarkan`mint.json`. Kemudian, silakan hapus`mint.json` file dari repositori Anda.
</Note>
