Kami menawarkan tema yang sudah jadi sehingga Anda dapat membedakan tampilan dan nuansa dokumentasi Anda berdasarkan merek Anda.

## Konfigurasi

Anda dapat mengonfigurasi tema di file `docs.json`.

Misalnya, untuk menggunakan tema `maple`, Anda dapat menambahkan berikut ini ke file `docs.json` Anda:

```json
{
  "theme": "maple"
}
```
