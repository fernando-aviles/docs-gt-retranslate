## Mengaktifkan Autentikasi

Anda dapat menambahkan metode autentikasi ke docs.json Anda untuk mengaktifkannya di setiap halaman atau Anda dapat mengaturnya berdasarkan halaman.

Metode autentikasi halaman akan menggantikan docs.json jika keduanya diatur.

### Bearer Token

<CodeGroup>
  ```json docs.json
  "api": {
      "mdx": {
        "auth": {
          "method": "bearer"
        }
      }
  }
  ```

  ```md Page Metadata
  ---
  title: "Your page title"
  authMethod: "bearer"
  ---
  ```
</CodeGroup>

### Basic Authentication

<CodeGroup>
  ```json docs.json
  "api": {
      "mdx": {
        "auth": {
          "method": "basic"
        }
      }
  }
  ```

  ```md Page Metadata
  ---
  title: "Your page title"
  authMethod: "basic"
  ---
  ```
</CodeGroup>

### API Key

<CodeGroup>
  ```json docs.json
  "api": {
      "mdx": {
        "auth": {
          "method": "key",
          "name": "x-api-key"
        }
      }
  }
  ```

  ```md Page Metadata
  ---
  title: "Your page title"
  authMethod: "key"
  ---
  ```
</CodeGroup>

### None

Metode autentikasi "none" berguna untuk menonaktifkan autentikasi pada endpoint tertentu setelah mengatur default di docs.json.

<CodeGroup>
  ```md Page Metadata
  ---
  title: "Your page title"
  authMethod: "none"
  ---
  ```
</CodeGroup>
