## Konfigurasi vercel.json

Untuk meng-host dokumentasi Anda di subpath kustom menggunakan Vercel, Anda perlu menambahkan konfigurasi berikut ke`vercel.json` file.

```json
{
  "rewrites": [
    {
      "source": "/docs",
      "destination": "https://[subdomain].mintlify.dev/docs"
    },
    {
      "source": "/docs/:match*",
      "destination": "https://[subdomain].mintlify.dev/docs/:match*"
    }
  ]
}
```

<Note>
  Untuk informasi lebih lanjut, Anda juga dapat merujuk ke panduan resmi Vercel tentang rewrites:[Project Configuration:
  Rewrites](https://vercel.com/docs/projects/project-configuration#rewrites)
</Note>
