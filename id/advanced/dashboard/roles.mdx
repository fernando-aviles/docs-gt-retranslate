---
- title: Peran
- description: Kontrol akses ke dasbor Anda dengan peran.
---

Mintlify menyediakan dua tingkat akses dasbor: Editor dan Admin.

Berikut ini menjelaskan tindakan yang terbatas pada peran Admin:

|                             | Editor | Admin |
| --------------------------- | :----: | :---: |
| Perbarui peran pengguna     |    ❌   |   ✅   |
| Hapus pengguna              |    ❌   |   ✅   |
| Undang pengguna admin       |    ❌   |   ✅   |
| Kelola & perbarui penagihan |    ❌   |   ✅   |
| Perbarui domain kustom      |    ❌   |   ✅   |
| Perbarui sumber Git         |    ❌   |   ✅   |
| Hapus organisasi            |    ❌   |   ✅   |

Tindakan lain di dasbor tersedia untuk kedua peran.

Anda dapat mengundang sebanyak mungkin admin yang Anda inginkan, tetapi kami menyarankan untuk membatasi akses admin hanya kepada pengguna yang membutuhkannya.
