---
- title: Halaman tersembunyi
- description: Kecualikan halaman dari `docs.json` Anda
- icon: eye-closed
---

Anda dapat menggunakan halaman tersembunyi untuk konten yang tidak benar-benar rahasia, tetapi belum siap untuk ditampilkan. Halaman tersembunyi dikecualikan dari navigasi situs Anda, sehingga orang hanya dapat mengakses halaman tersembunyi jika mereka mengetahui URL yang tepat.

Sebuah halaman tersembunyi sampai Anda menambahkannya ke grup, tab, atau elemen navigasi lainnya di `docs.json`.

<Note>
  Beberapa elemen navigasi seperti sidebar, dropdown, dan tab mungkin bergeser atau muncul kosong pada halaman tersembunyi.
</Note>

Jika Anda ingin halaman tersembunyi diindeks untuk tujuan SEO dan agar kontennya muncul ketika orang mencari di dalam dokumentasi Anda, tambahkan pengaturan berikut ke `docs.json`:

```
"seo" {
    "indexing": all
}
```

Jika Anda ingin mengecualikan konten halaman tersembunyi tertentu dari hasil pencarian mesin pencari dan pencarian internal, tambahkan `noindex: true` ke frontmatter halaman tersebut.

Jika Anda memiliki halaman yang hanya ingin diakses oleh kelompok pengguna tertentu, Anda dapat menggunakan akses kelompok selektif yang tersedia melalui [Personalization](/id/settings/authentication-personalization/personalization#showing-hiding-pages).

Jika Anda memiliki halaman yang ingin Anda jaga kontrol aksesnya dengan ketat, Anda dapat menggunakan [Authentication](/id/settings/authentication-personalization/authentication).
