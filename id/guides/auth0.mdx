- title: Menggunakan Auth0 dengan OAuth Handshake
- description: Jika Auth0 adalah sumber kebenaran untuk data pengguna Anda, Anda
    dapat mengatur Mintlify sebagai aplikasi klien OAuth untuk mengautentikasi
    pengguna Anda.
---

<Note>
  **Peringatan Keamanan**: Meskipun kami menyediakan panduan ini untuk membantu Anda mengintegrasikan Auth0 dengan Mintlify, silakan berkonsultasi dengan tim keamanan Anda sebelum menerapkan solusi autentikasi apa pun. Mintlify tidak bertanggung jawab atas masalah keamanan yang mungkin timbul dari implementasi spesifik Anda.
</Note>

## Ikhtisar

Panduan ini memandu Anda melalui pengaturan Auth0 sebagai penyedia autentikasi untuk dokumentasi Mintlify Anda. Pada akhirnya, pengguna Anda akan dapat masuk ke dokumentasi Anda menggunakan kredensial Auth0 mereka.

<Steps>
  <Step title="Buat Aplikasi Web Reguler di Auth0">
    Masuk ke dashboard Auth0 Anda dan navigasikan ke **Applications** > **Applications**. Klik tombol **Create Application**, beri nama aplikasi Anda (misalnya, "Mintlify"), dan pilih **Regular Web Applications** sebagai jenis aplikasi. Kemudian klik **Create**.

    <Frame>![Creating a new application in Auth0](/images/guides/auth0/auth0-create-application.png)</Frame>
  </Step>

  <Step title="Dapatkan informasi klien">
    Setelah membuat aplikasi Anda, Anda akan dibawa ke halaman pengaturan aplikasi. Di sini, Anda akan menemukan kredensial penting yang diperlukan untuk integrasi OAuth:

    <Frame>![Auth0 client settings page](/images/guides/auth0/auth0clientsettings.png)</Frame>

    Catat informasi berikut:

    * **Domain**: Ini adalah domain tenant Auth0 Anda (misalnya, `your-tenant.auth0.com`)
    * **Client ID**: Pengidentifikasi publik untuk aplikasi Anda
    * **Client Secret**: Kunci rahasia untuk aplikasi Anda (jaga keamanannya)

    Anda akan memerlukan nilai-nilai ini untuk mengkonfigurasi Mintlify di langkah berikutnya.
  </Step>

  <Step title="Siapkan klien Mintlify">
    Navigasikan ke Dashboard Mintlify Anda dan pergi ke bagian **Settings** > **Authentication**. Pilih **OAuth** sebagai metode autentikasi Anda dan Anda akan melihat formulir konfigurasi OAuth:

    <Frame>![Mintlify OAuth client settings](/images/guides/auth0/mintlifyoauthclientsettings.png)</Frame>

    Isi formulir dengan nilai-nilai berikut:

    * **Authorization URL**: `https://YOUR_AUTH0_DOMAIN/authorize` (ganti `YOUR_AUTH0_DOMAIN` dengan domain Auth0 Anda yang sebenarnya dari langkah 2)
    * **Client ID**: Masukkan Client ID dari aplikasi Auth0 Anda
    * **Client Secret**: Masukkan Client Secret dari aplikasi Auth0 Anda
    * **Scopes**: Biarkan kosong kecuali Anda memiliki scope kustom yang diatur di Auth0
    * **Token URL**: `https://YOUR_AUTH0_DOMAIN/oauth/token` (ganti `YOUR_AUTH0_DOMAIN` dengan domain Auth0 Anda yang sebenarnya)

    Setelah mengisi detail-detail ini, klik **Save changes** untuk menyimpan konfigurasi OAuth Anda.
  </Step>

  <Step title="Konfigurasi Callback URL">
    Mintlify akan menghasilkan Redirect URL unik yang perlu dikenali oleh Auth0 agar alur OAuth berfungsi dengan baik.

    Salin Redirect URL dari pengaturan Authentication di Dashboard Mintlify Anda:

    <Frame>![Mintlify redirect URL](/images/guides/auth0/mintlifyredirecturl.png)</Frame>

    Kembali ke halaman pengaturan aplikasi Auth0 Anda, gulir ke bawah ke bagian **Application URIs**, dan tempel Redirect URL ke dalam **Allowed Callback URLs** field:
    <Frame>![Auth0 redirect URL configuration](/images/guides/auth0/auth0redirecturl.png)</Frame>

    Klik **Save Changes** di bagian bawah halaman Auth0 untuk menerapkan konfigurasi ini.
  </Step>
</Steps>
