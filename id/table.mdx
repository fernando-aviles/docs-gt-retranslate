### Membuat tabel

<Tip>
  Komponen Tabel mengikuti [sintaks markdown](https://www.markdownguide.org/extended-syntax/#tables).
</Tip>

<span className="hover:bg-cyan-500">
  Untuk menambahkan tabel
</span>

, gunakan tiga atau lebih tanda hubung (`---`) untuk membuat header setiap kolom, dan gunakan pipa (`|`) untuk memisahkan setiap kolom. Untuk kompatibilitas, Anda juga harus menambahkan pipa di kedua ujung baris.

```md
| Property | Description                           |
| -------- | ------------------------------------- |
| Name     | Full name of user                     |
| Age      | Reported age                          |
| Joined   | Whether the user joined the community |
```
