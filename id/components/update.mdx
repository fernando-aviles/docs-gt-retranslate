Komponen`Update`digunakan untuk melacak perubahan dan pembaruan.

<Update label="2024-10-12" description="v0.1.1">
  <Frame>
    <img className="block" src="https://mintlify-assets.b-cdn.net/hero-light.png" />
  </Frame>

  ## Changelog

  Anda dapat menambahkan apa saja di sini, seperti tangkapan layar, cuplikan kode, atau daftar perubahan.

  #### Fitur

  * Desain responsif
  * Bagian sticky untuk setiap changelog
</Update>

<Update label="2024-10-11" description="v0.1.0">
  ### Cara penggunaan

  ```md
  <Update label="2024-10-12" description="v0.1.1">
    This is how you use a changelog with a label 
    and a description.
  </Update>
  <Update label="2024-10-11" description="v0.1.0">
    This is how you use a changelog with a label 
    and a description.
  </Update>
  ```

  Anda dapat menggunakan beberapa komponen`Update`untuk membuat changelog.
</Update>

<Tip>
  Setiap`label`membuat jangkar dan juga muncul di daftar isi di sebelah kanan.
</Tip>

## Props

<ResponseField name="label" type="string" required>
  Label yang diberikan pada pembaruan, muncul sebagai teks sticky di sebelah kiri changelog.
</ResponseField>

<ResponseField name="tags" type="string[]">
  Tag untuk changelog, akan ditampilkan sebagai filter di panel sisi kanan.
</ResponseField>

<ResponseField name="description" type="string">
  Deskripsi pembaruan, muncul di bawah label dan tag.
</ResponseField>

  ```jsx Update Example
  </Update>
  ```
</RequestExample>
