## Kartu horizontal

Tambahkan properti `horizontal` untuk menampilkan kartu secara horizontal.

<Card title="Kartu Horizontal" icon="text" horizontal>
  Ini adalah contoh kartu horizontal
</Card>

## Kartu gambar

Tambahkan properti `img` untuk menampilkan gambar di bagian atas kartu.

<Card title="Kartu Gambar" img="https://mintlify-assets.b-cdn.net/yosemite.jpg">
  Ini adalah contoh kartu dengan gambar
</Card>

## Kartu tautan

Anda dapat menyesuaikan CTA dan apakah akan menampilkan panah pada kartu atau tidak. Secara default, panah hanya akan ditampilkan untuk tautan eksternal.

<Card title="Kartu tautan" icon="link" href="/id/components/columns" arrow="true" cta="Klik di sini">
  Ini adalah cara menggunakan kartu dengan ikon dan tautan. Mengklik kartu ini
  membawa Anda ke halaman Kolom.
</Card>

<RequestExample>
  ```jsx Card Example
  <Card
    title="Link card"
    icon="link"
    href="/components/columns"
    arrow="true"
    cta="Click here"
  >
    This is how you use a card with an icon and a link. Clicking on this card
    brings you to the Columns page.
  </Card>
  ```
</RequestExample>

## Mengelompokkan kartu

Anda dapat mengelompokkan kartu dalam [columns](/id/components/columns).

<Columns cols={2}>
  <Card title="Kartu Pertama" icon="panel-left-close">
    Ini adalah kartu pertama.
  </Card>

  <Card title="Kartu Kedua" icon="panel-right-close">
    Ini adalah kartu kedua.
  </Card>
</Columns>

## Props

<ResponseField name="title" type="string" required>
  Judul kartu
</ResponseField>

<ResponseField name="icon" type="string or svg">
  Ikon [Font Awesome icon](https://fontawesome.com/icons), [Lucide icon](https://lucide.dev/icons), atau kode SVG dalam `icon={}`
</ResponseField>

<ResponseField name="iconType" type="string">
  Salah satu dari `regular`, `solid`, `light`, `thin`, `sharp-solid`, `duotone`, `brands`
</ResponseField>

<ResponseField name="color" type="string">
  Warna ikon sebagai kode hex
</ResponseField>

<ResponseField name="href" type="string">
  URL yang akan dinavigasi pengguna saat mengklik kartu
</ResponseField>

<ResponseField name="horizontal" type="boolean">
  Membuat kartu lebih ringkas dan horizontal
</ResponseField>

<ResponseField name="img" type="string">
  URL atau path lokal ke gambar yang akan ditampilkan di bagian atas kartu
</ResponseField>

<ResponseField name="cta" type="string">
  Label untuk tombol aksi
</ResponseField>

<ResponseField name="arrow" type="boolean">
  Mengaktifkan atau menonaktifkan ikon panah tautan
</ResponseField>
