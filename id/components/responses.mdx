---
- title: Bidang Respons
- description: Tampilkan nilai respons API
---

Komponen`<ResponseField>` dirancang untuk mendefinisikan nilai pengembalian dari sebuah API. Banyak dokumentasi juga menggunakan`<ResponseField>` pada halaman ketika Anda perlu mendaftar jenis-jenis sesuatu.

<ResponseField name="response" type="string" required>
  Contoh bidang respons
</ResponseField>

```jsx
<ResponseField name="response" type="string" required>
  A response field example
</ResponseField>
```

## Props

<ResponseField name="name" type="string" required>
  Nama nilai respons.
</ResponseField>

<ResponseField name="type" type="string" required>
  Tipe yang diharapkan dari nilai respons - ini bisa berupa string arbitrer apa pun.
</ResponseField>

<ResponseField name="default" type="string">
  Nilai default.
</ResponseField>

<ResponseField name="required" type="boolean">
  Tampilkan "required" di samping nama bidang.
</ResponseField>

<ResponseField name="deprecated" type="boolean">
  Apakah sebuah bidang sudah usang atau tidak.
</ResponseField>

<ResponseField name="pre" type="string[]">
  Label yang ditampilkan sebelum nama bidang
</ResponseField>

<ResponseField name="post" type="string[]">
  Label yang ditampilkan setelah nama bidang
</ResponseField>
