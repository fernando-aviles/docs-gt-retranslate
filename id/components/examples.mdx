Komponen `<RequestExample>` dan `<ResponseExample>` menempelkan blok kode ke kanan atas halaman bahkan saat Anda menggulir. Komponen ini berfungsi di semua halaman bahkan jika Anda tidak menggunakan playground API.

`<RequestExample>` dan `<ResponseExample>` muncul seperti blok kode biasa pada perangkat mobile.

## Contoh Permintaan

Komponen `<RequestExample>` bekerja mirip dengan [CodeGroup](/id/components/code-groups), tetapi menampilkan konten permintaan di sidebar kanan. Dengan demikian, Anda dapat menempatkan beberapa blok kode di dalam `<RequestExample>`.

Harap atur nama pada setiap blok kode yang Anda tempatkan di dalam RequestExample.

<RequestExample>
  ````md RequestExample Example
  <RequestExample>

  ```bash Request
    curl --request POST \
      --url https://dog-api.kinduff.com/api/facts
  ```

  </RequestExample>
  ````
</RequestExample>

## Contoh Respons

Komponen `<ResponseExample>` sama dengan `<RequestExample>` tetapi akan muncul di bawahnya.

<ResponseExample>
  ````md ResponseExample Example
  <ResponseExample>

  ```json Response
  { "status": "success" }
  ```

  </ResponseExample>
  ````
</ResponseExample>
