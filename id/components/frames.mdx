Frame sangat membantu jika Anda ingin memusatkan gambar.

<Frame>

## Caption

Anda dapat menambahkan konteks tambahan ke gambar menggunakan prop opsional `caption`.

<Frame caption="Taman Nasional Yosemite dikunjungi oleh lebih dari 3,5 juta orang setiap tahun">
  <img src="https://mintlify-assets.b-cdn.net/yosemite.jpg" />
</Frame>

## Props

<ResponseField name="caption" type="string">
  Teks caption opsional untuk ditampilkan di tengah di bawah komponen Anda.
</ResponseField>

<RequestExample>
  ```jsx Frame
  <Frame>
    <img src="/path/image.jpg" />
  </Frame>
  ```

  ```jsx Frame with Captions
  <Frame caption="Caption Text">
    <img src="/path/image.jpg" />
  </Frame>
  ```
</RequestExample>
