---
- title: Callouts
- description: Gunakan callouts untuk menambahkan konteks yang menarik perhatian
    ke konten Anda
- icon: info
---

Callouts dapat diberi gaya sebagai Note, Warning, Info, Tip, atau Check:

<Note>Ini menambahkan catatan dalam konten</Note>

```jsx
<Note>This adds a note in the content</Note>
```

<Warning>Ini memunculkan peringatan untuk diperhatikan</Warning>

```jsx
<Warning>This raises a warning to watch out for</Warning>
```

<Info>Ini menarik perhatian pada informasi penting</Info>

```jsx
<Info>This draws attention to important information</Info>
```

<Tip>Ini menyarankan tip yang berguna</Tip>

```jsx
<Tip>This suggests a helpful tip</Tip>
```

<Check>Ini membawa kita status yang dicentang</Check>

```jsx
<Check>This brings us a checked status</Check>
```

  ```jsx Callout Example
  <Note>This adds a note in the content</Note>
  ```
</RequestExample>
