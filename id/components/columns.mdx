- title: Kolom
- description: Tampilkan kartu berdampingan dalam format grid
- icon: columns-2
---

Komponen `Columns` memungkinkan Anda mengelompokkan beberapa komponen `Card` bersama-sama. Ini paling sering digunakan untuk menempatkan beberapa kartu dalam grid, dengan menentukan jumlah kolom grid.

<Columns cols={2}>
  <Card title="Kartu Pertama" icon="panel-left-close">
    Neque porro quisquam est qui dolorem ipsum quia dolor sit amet
  </Card>

  <Card title="Kartu Kedua" icon="panel-right-close">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit
  </Card>
</Columns>

<RequestExample>
  ```jsx Card Group Example
  <Columns cols={2}>
    <Card title="First Card">
      Neque porro quisquam est qui dolorem ipsum quia dolor sit amet
    </Card>
    <Card title="Second Card">
      Lorem ipsum dolor sit amet, consectetur adipiscing elit
    </Card>
  </Columns>
  ```
</RequestExample>

### Props

<ResponseField name="cols" default={2}>
  Jumlah kolom per baris
</ResponseField>
