---
- title: Grup Kode
- description: Komponen CodeGroup memungkinkan Anda menggabungkan blok kode dalam
    tampilan yang dipisahkan oleh tab
- icon: group
---

Anda perlu membuat [Blok Kode](/id/code) kemudian tambahkan komponen `<CodeGroup>` di sekitarnya. Setiap Blok Kode harus memiliki nama file karena kami menggunakan nama untuk tombol tab.

Lihat di bawah untuk contoh hasil akhir.

<CodeGroup>
  ```javascript helloWorld.js
  console.log("Hello World");
  ```

  ```python hello_world.py
  print('Hello World!')
  ```

  ```java HelloWorld.java
  class HelloWorld {
      public static void main(String[] args) {
          System.out.println("Hello, World!");
      }
  }
  ```
</CodeGroup>
