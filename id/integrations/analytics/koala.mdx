---
- title: Koala
---

Tambahkan berikut ini ke file `docs.json` Anda untuk mengirim analitik ke Koala.

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "koala": {
          "publicApiKey": "required"
      }
  }
  ```

  ```json Example
  "integrations": {
      "koala": {
          "publicApiKey": "pk_1a1882"
      }
  }
  ```
</CodeGroup>
