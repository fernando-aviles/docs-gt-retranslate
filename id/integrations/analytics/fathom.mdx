---
- title: Fathom
---

Tambahkan yang berikut ke file `docs.json` Anda untuk mengirim analitik ke Fathom.

Anda dapat mendapatkan `siteId` dari pengaturan skrip Anda.

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "fathom": {
          "siteId": "required"
      }
  }
  ```

  ```json Example
  "integrations": {
      "fathom": {
          "siteId": "YSVMSDAY"
      }
  }
  ```
</CodeGroup>
