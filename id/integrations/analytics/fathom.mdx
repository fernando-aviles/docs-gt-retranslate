---
title: Fathom
---

Tambahkan berikut ini ke`docs.json` file untuk mengirim analitik ke Fathom.

Anda bisa mendapatkan`siteId` dari pengaturan skrip Anda.

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "fathom": {
          "siteId": "required"
      }
  }
  ```

  ```json Example
  "integrations": {
      "fathom": {
          "siteId": "YSVMSDAY"
      }
  }
  ```
</CodeGroup>
