---
- title: Plausible
---

Tambahkan domain situs Anda ke `docs.json` untuk mengirim analitik ke Plausible.

<Info>
  Jangan sertakan `https://` untuk domain atau server.
</Info>

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "plausible": {
          "domain": "required",
          "server": "optional"
      }
  }
  ```

  ```json Example
  "integrations": {
      "plausible": {
          "domain": "docs.domain.com"
      }
  }
  ```
</CodeGroup>
