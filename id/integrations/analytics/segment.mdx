---
- title: Segment
---

Tambahkan kunci tulis Segment Anda ke file `docs.json` Anda untuk mengirim analitik ke Segment.

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "segment": {
          "key": "required",
      }
  }
  ```

  ```json Example
  "integrations": {
      "segment": {
          "key": "nqJxiRG15Y6M594P8Sb8ESEciU3VC2"
      }
  }
  ```
</CodeGroup>
