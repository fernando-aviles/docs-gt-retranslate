---
title: Heap
---

Tambahkan berikut ini ke`docs.json` file untuk mengirim analitik ke Heap.

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "heap": {
          "appId": "required"
      }
  }
  ```

  ```json Example
  "integrations": {
      "heap": {
          "appId": "1234567890"
      }
  }
  ```
</CodeGroup>
