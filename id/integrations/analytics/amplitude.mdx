---
title: Amplitude
---

Tambahkan berikut ini ke`docs.json` file untuk mengirim analitik ke Amplitude.

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "amplitude": {
          "apiKey": "required"
      }
  }
  ```

  ```json Example
  "integrations": {
      "amplitude": {
          "apiKey": "76bb138bf3fbf58186XXX00000"
      }
  }
  ```
</CodeGroup>
