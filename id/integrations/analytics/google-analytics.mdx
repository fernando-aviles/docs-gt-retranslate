Anda perlu menghasilkan<Tooltip tip="Google Analytics 4">GA4</Tooltip> properti untuk digunakan dengan Mintlify. Data yang dikumpulkan akan masuk ke dalam proyek yang sama dengan data Google Analytics Anda lainnya.

Jika Anda menggunakan versi lama Google Analytics, Universal Analytics, Anda masih akan dapat menghasilkan <Tooltip tip="Google Analytics 4">GA4</Tooltip> properti.<Tooltip tip="Google Analytics 4">GA4</Tooltip> data sedikit berbeda dari data UA tetapi masih dikumpulkan dalam proyek yang sama.

## Cara Menghubungkan GA4 ke Mintlify

### Buat Web Stream

Anda perlu membuat web stream untuk mendapatkan Measurement ID yang akan dimasukkan ke Mintlify.

Klik ikon roda gigi di pojok kiri bawah layar Google Analytics. Kemudian klik pada Data Streams.

<Frame>![](/images/ga4-web-streams.png)</Frame>

Buat Web Stream dan masukkan URL situs dokumentasi Mintlify Anda sebagai URL stream.

Measurement ID Anda terlihat seperti `G-XXXXXXX` dan akan muncul di bawah Stream Details segera setelah Anda membuat Web Stream.

### Masukkan Measurement ID ke docs.json

Tambahkan Measurement ID Anda ke file `docs.json` Anda seperti ini:

```json docs.json
"integrations": {
    "ga4": {
        "measurementId": "G-XXXXXXX"
    }
}
```

### Tunggu

Google Analytics membutuhkan waktu dua hingga tiga hari untuk menampilkan data Anda.

Anda dapat menggunakan [Google Analytics Debugger](https://chrome.google.com/webstore/detail/google-analytics-debugger/jnkmfdileelhofjcijamephohjechhna?hl=en) untuk memeriksa apakah analitik telah diaktifkan dengan benar. Ekstensi ini akan mencatat ke konsol browser Anda setiap kali GA4 membuat permintaan.

<Note>
  Tautan pratinjau memiliki analitik yang dinonaktifkan.
</Note>
