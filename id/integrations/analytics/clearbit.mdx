---
title: Clearbit
---

Tambahkan berikut ini ke`docs.json` file untuk mengirim analitik ke Clearbit.

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "clearbit": {
          "publicApiKey": "required"
      }
  }
  ```

  ```json Example
  "integrations": {
      "clearbit": {
          "publicApiKey": "pk_1a1882"
      }
  }
  ```
</CodeGroup>
