## Prasyarat

* Memiliki akun [Stainless](https://app.stainless.com).

## Integrasi dengan Stainless

<Steps>
  <Step title="Siapkan dekorasi OpenAPI di Stainless.">
    Dalam file konfigurasi `stainless.yml` Anda, tambahkan `openapi.code_samples: 'mintlify'`. Lihat [dokumentasi Stainless](https://app.stainless.com/docs/guides/integrate-docs) untuk informasi lebih lanjut.
  </Step>

  <Step title="Publikasikan URL ke spesifikasi OpenAPI Anda.">
    Dalam proyek Stainless Anda:

    1. Pilih tab **Release**.
    2. Pilih **Setup OpenAPI publishing**.
    3. Salin URL ke spesifikasi OpenAPI Anda yang dapat diakses publik.

    <img src="/images/stainless-public-OpenAPI-spec.png" alt="Stainless release page with the OpenAPI spec URL highlighted with a green box." />
  </Step>

  <Step title="Tambahkan URL spesifikasi OpenAPI Anda ke `docs.json` Anda.">
    Dalam file `docs.json` Anda, tambahkan URL dari Stainless ke bidang `openapi`. Lihat [OpenAPI Setup](/id/api-playground/openapi/setup) untuk informasi lebih lanjut.
  </Step>
</Steps>
