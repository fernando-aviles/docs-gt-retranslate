---
title: Front
---

Tambahkan berikut ini ke`docs.json` file untuk menambahkan[Front Chat](https://front.com) widget.

<CodeGroup>
  ```json Integration options in docs.json
  "integrations": {
      "frontchat": "CHAT_ID"
  }
  ```

  ```json Example
  "integrations": {
      "frontchat": "1365d046d7c023e9b030ce90d02d093a"
  }
  ```
</CodeGroup>
