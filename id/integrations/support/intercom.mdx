title: Intercom
---

Tambahkan berikut ini ke`docs.json` file untuk menambahkan[Intercom](https://www.intercom.com) widget.

<CodeGroup>
  ```json Integration options in docs.json
  "integrations": {
        "intercom": {
              "appId": "APP_ID"
        }
  }
  ```
