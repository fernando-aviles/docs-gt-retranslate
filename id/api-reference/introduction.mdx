## Pemicu Pembaruan

Anda dapat memanfaatkan REST API untuk memicu pembaruan secara programatik saat diinginkan.

## Autentikasi

Anda dapat menghasilkan kunci API melalui

[dashboard](https://dashboard.mintlify.com/settings/organization/api-keys). Kunci API tersebut
terkait dengan seluruh organisasi dan dapat digunakan di berbagai deployment.

<Frame>
  <img src="/images/external-api-key.png" />
</Frame>

## Kunci API Admin

Kunci API Admin digunakan untuk sebagian besar API. Ini digunakan untuk memicu pembaruan melalui [endpoint Pembaruan](/id/api-reference/update/trigger).

## Kunci API Asisten

API Asisten memungkinkan Anda untuk menyematkan pengalaman asisten AI yang didasarkan pada dokumentasi Anda dan terus diperbarui ke dalam aplikasi pilihan Anda.

Respons mencakup kutipan sehingga Anda dapat mengarahkan pengguna Anda ke tempat yang tepat yang mereka butuhkan untuk mendapatkan bantuan.

<Note>
  Token API Asisten adalah token publik yang dapat direferensikan dalam kode
  frontend Anda sedangkan kunci API adalah token sisi server yang harus dijaga
  kerahasiaannya.
</Note>

Sekarang Anda memiliki kunci API, lihat [contoh](https://github.com/mintlify/discovery-api-example) kami tentang cara menggunakan
API untuk asisten AI. Anda juga dapat melihat versi yang sudah di-deploy dari contoh ini di [chat.mintlify.com](https://chat.mintlify.com).
