## Code Usage

### Inline Code
Use backticks (`) to mark `inline code`.

### Code Block
For multi-line code, use triple backticks with an optional language identifier for syntax highlighting.

```java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

For more details, see the [Code Block component docs](/content/components/code).
