## Prerequisites

<Info>
Please install Node.js (version 19 or higher) before proceeding with the installation.
</Info>

## Installation

<Steps>
  <Step title="Install Mintlify CLI">
    Choose your preferred package manager to install Mintlify globally:

    <CodeGroup>
      ```bash npm
      npm i -g mintlify
      ```

      ```bash yarn
      yarn global add mintlify
      ```

      ```bash pnpm
      pnpm add -g mintlify
      ```
    </CodeGroup>
  </Step>

  <Step title="Start Development Server">
    Navigate to your docs directory (where the `mint.json` file is located) and run:

    ```bash
    mintlify dev
    ```

    Alternatively, if you prefer not to install Mintlify globally, use a run script:

    <CodeGroup>
      ```bash npm
      npx mintlify dev
      ```

      ```bash yarn
      yarn dlx mintlify dev
      ```

      ```bash pnpm
      pnpm dlx mintlify dev
      ```
    </CodeGroup>

    <Warning>
      Yarn's "dlx" run script requires yarn version >2. See [here](https://yarnpkg.com/cli/dlx) for more information.
    </Warning>
  </Step>

  <Step title="View Local Preview">
    Once the development server starts, view your documentation at `http://localhost:3000`
  </Step>
</Steps>

## Configuration Options

### Custom Ports

By default, Mintlify uses port 3000. You can customize the port using the `--port` flag:

```bash
mintlify dev --port 3333
```

If the specified port is already in use, Mintlify will automatically use the next available port:

```md
Port 3000 is already in use. Trying 3001 instead.
```

### Updating Mintlify

Keep your CLI in sync with the latest Mintlify version by updating regularly:

<CodeGroup>
  ```bash npm
  npm i -g mintlify@latest
  ```

  ```bash yarn
  yarn global upgrade mintlify
  ```

  ```bash pnpm
  pnpm up --global mintlify
  ```
</CodeGroup>

## Development Tools

### Link Validation

Check for broken links in your documentation using:

```bash
mintlify broken-links
```

### Code Formatting

We recommend using these VSCode extensions for the best development experience:

- [MDX VSCode extension](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx) for syntax highlighting
- [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) for code formatting

## Troubleshooting Guide

<AccordionGroup>
  <Accordion title="Sharp Module Error (Darwin/ARM64)">
    If you encounter the error: "Could not load the 'sharp' module using the darwin-arm64 runtime", follow these steps:

    1. Remove the current Mintlify installation:
    ```bash
    npm remove -g mintlify
    ```

    2. Upgrade Node.js to version 19 or higher

    3. Reinstall Mintlify:
    ```bash
    npm install -g mintlify
    ```
  </Accordion>

  <Accordion title="Unknown Errors">
    If you encounter an unknown error:

    1. Navigate to your device's root directory
    2. Delete the `~/.mintlify` folder
    3. Run `mintlify dev` again
  </Accordion>

  <Accordion title="Port Already in Use">
    If you see "Port already in use" errors:

    1. Try specifying a different port:
    ```bash
    mintlify dev --port 3333
    ```
    
    2. Or close any other development servers that might be running
  </Accordion>
</AccordionGroup>
