<Info>
**Prerequisite**: Node.js version 19 or higher
</Info>

## Quick Start

1. Install Mintlify CLI:
<CodeGroup>
```bash npm
npm i -g mintlify
```

```bash yarn
yarn global add mintlify
```
</CodeGroup>

2. Start the development server:
```bash
mintlify dev
```

Your docs will be available at `http://localhost:3000`.

## Common Tasks

### Using a Different Port
```bash
mintlify dev --port <port-number>
```

### Updating Mintlify
<CodeGroup>
```bash npm
npm i -g mintlify@latest
```

```bash yarn
yarn global upgrade mintlify
```
</CodeGroup>

### Checking for Broken Links
```bash
mintlify broken-links
```

## Troubleshooting

<AccordionGroup>
  <Accordion title='Error: Could not load the "sharp" module'>
    1. Remove mintlify: `npm remove -g mintlify`
    2. Upgrade to Node v19+
    3. Reinstall: `npm install -g mintlify`
  </Accordion>

  <Accordion title="Unknown errors">
    Delete the ~/.mintlify folder and run `mintlify dev` again.
  </Accordion>
</AccordionGroup>
