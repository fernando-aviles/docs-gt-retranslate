<Info>
**Prerequisite**: Node.js (version 19+) required
</Info>

## Setup and Preview

1. Install Mintlify:
<CodeGroup>
```bash npm
npm i -g mintlify
```
```bash yarn
yarn global add mintlify
```
</CodeGroup>

2. Run the development server:
```bash
mintlify dev
```
Your docs will be available at `http://localhost:3000`.

## Additional Configuration

### Custom Port
Use `--port` flag to specify a different port:
```bash
mintlify dev --port 3333
```

### Update CLI
Keep your local version in sync with production:
<CodeGroup>
```bash npm
npm i -g mintlify@latest
```
```bash yarn
yarn global upgrade mintlify
```
</CodeGroup>

### Validate Links
Check for broken links:
```bash
mintlify broken-links
```

## Troubleshooting
<AccordionGroup>
  <Accordion title='Sharp Module Error (darwin-arm64)'>
    1. Remove mintlify: `npm remove -g mintlify`
    2. Upgrade to Node v19+
    3. Reinstall: `npm install -g mintlify`
  </Accordion>
  <Accordion title="Unknown Error">
    Delete ~/.mintlify folder and run `mintlify dev` again
  </Accordion>
</AccordionGroup>
