<Info>
**Prerequisite**: Node.js (version 19 or higher)
</Info>

## Quick Setup

1. Install Mintlify:
```bash
npm i -g mintlify
```

2. Run the development server:
```bash
mintlify dev
```

Your docs will be available at `http://localhost:3000`.

## Common Commands

```bash
# Start on a different port
mintlify dev --port 3333

# Update Mintlify CLI
npm i -g mintlify@latest

# Check for broken links
mintlify broken-links
```

## Troubleshooting

<AccordionGroup>
  <Accordion title='Error: Could not load "sharp" module'>
    1. Remove mintlify: `npm remove -g mintlify`
    2. Upgrade to Node v19+
    3. Reinstall: `npm install -g mintlify`
  </Accordion>

  <Accordion title="Unknown errors">
    Delete the ~/.mintlify folder and run `mintlify dev` again.
  </Accordion>
</AccordionGroup>
