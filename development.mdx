<Info>
**Prerequisite**: Node.js version 19 or higher
</Info>

## Installation

Install Mintlify globally:

<CodeGroup>
```bash npm
npm i -g mintlify
```

```bash yarn
yarn global add mintlify
```

```bash pnpm
pnpm add -g mintlify
```
</CodeGroup>

## Run Locally

1. Go to your docs directory (where `mint.json` is)
2. Run the dev server:

```bash
mintlify dev
```

Or without global installation:

<CodeGroup>
```bash npm
npx mintlify dev
```

```bash yarn
yarn dlx mintlify dev
```

```bash pnpm
pnpm dlx mintlify dev
```
</CodeGroup>

Your docs will be available at `http://localhost:3000`.

## Customize Port

Use `--port` to run on a different port:

```bash
mintlify dev --port 3333
```

## Update CLI

Keep your CLI up to date:

<CodeGroup>
```bash npm
npm i -g mintlify@latest
```

```bash yarn
yarn global upgrade mintlify
```

```bash pnpm
pnpm up --global mintlify
```
</CodeGroup>

## Useful Commands

Check for broken links:
```bash
mintlify broken-links
```

## Editor Setup

For VSCode users:
- Install [MDX extension](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx) for syntax highlighting
- Install [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) for formatting

## Troubleshooting

<AccordionGroup>
<Accordion title='Sharp Module Error (darwin-arm64)'>
1. Remove mintlify: `npm remove -g mintlify`
2. Upgrade to Node v19+
3. Reinstall: `npm install -g mintlify`
</Accordion>

<Accordion title="Unknown Errors">
Delete ~/.mintlify folder and run `mintlify dev` again.
</Accordion>
</AccordionGroup>
