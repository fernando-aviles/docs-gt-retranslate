El `Update` componente se usa para mantener un registro de cambios y actualizaciones.

<Update label="2024-10-12" description="v0.1.1">
  <Frame>
    <img className="block" src="https://mintlify-assets.b-cdn.net/hero-light.png" />
  </Frame>

  ## Registro de cambios

  Puedes agregar cualquier cosa aquí, como una captura de pantalla, un fragmento de código o una lista de cambios.

  #### Características

  * Diseño responsivo
  * Sección fija para cada registro de cambios
</Update>

<Update label="2024-10-11" description="v0.1.0">
  ### Cómo usar

  ```md
  <Update label="2024-10-12" description="v0.1.1">
    This is how you use a changelog with a label 
    and a description.
  </Update>
  <Update label="2024-10-11" description="v0.1.0">
    This is how you use a changelog with a label 
    and a description.
  </Update>
  ```

  Puedes usar múltiples `Update` componentes para crear registros de cambios.
</Update>

<Tip>
  Cada `label` crea un ancla y también aparece en la tabla de contenidos a la derecha.
</Tip>

## Props

<ResponseField name="label" type="string" required>
  Etiqueta en el registro de cambios, en el lado izquierdo fijo.
</ResponseField>

<ResponseField name="description" type="string">
  Descripción debajo de la etiqueta en la vista previa del Registro de cambios.
</ResponseField>

<ResponseField name="tags" type="string[]">
  Etiquetas para el registro de cambios, se mostrarán como filtros en el panel del lado derecho
</ResponseField>
