El `<RequestExample>` y `<ResponseExample>` fijan bloques de código en la parte superior derecha de una página incluso mientras te desplazas. Los componentes funcionan en todas las páginas incluso si no usas un área de pruebas de API.

`<RequestExample>` y `<ResponseExample>` se muestran como bloques de código regulares en móvil.

## Ejemplo de Solicitud

El `<RequestExample>` componente funciona similar a [CodeGroup](/content/components/code-groups), pero muestra el contenido de la solicitud en la barra lateral derecha. Por lo tanto, puedes poner múltiples bloques de código dentro de `<RequestExample>`.

Por favor, establece un nombre en cada bloque de código que pongas dentro de RequestExample.

<RequestExample>
  ````md RequestExample Example
  <RequestExample>

  ```bash Request
    curl --request POST \
      --url https://dog-api.kinduff.com/api/facts
  ```

  </RequestExample>
  ````
</RequestExample>

## Ejemplo de Respuesta

El `<ResponseExample>` componente es el mismo que `<RequestExample>` pero se mostrará debajo de él.

<ResponseExample>
  ````md ResponseExample Example
  <ResponseExample>

  ```json Response
  { "status": "success" }
  ```

  </ResponseExample>
  ````
</ResponseExample>
