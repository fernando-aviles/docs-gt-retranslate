El componente `<ResponseField>` está diseñado para definir los valores de retorno de una API. Muchos documentos también usan `<ResponseField>` en páginas cuando necesita listar los tipos de algo.

<ResponseField name="response" type="string" required>
  Un ejemplo de campo de respuesta
</ResponseField>

<RequestExample>
  ```jsx ResponseField Example
  <ResponseField name="response" type="string" required>
    A response field example
  </ResponseField>
  ```
</RequestExample>

## Props

<ResponseField name="name" type="string" required>
  El nombre del valor de respuesta.
</ResponseField>

<ResponseField name="type" type="string" required>
  Tipo esperado del valor de respuesta
</ResponseField>

<ResponseField name="default" type="string">
  El valor predeterminado.
</ResponseField>

<ResponseField name="required" type="boolean">
  Mostrar "required" junto al nombre del campo.
</ResponseField>
