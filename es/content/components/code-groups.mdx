---
title: Grupos de Código
description: El componente CodeGroup te permite combinar bloques de código en una visualización
  separada por pestañas
icon: rectangle-code
---

Necesitarás hacer [Bloques de Código](/content/components/code) luego añadir el `<CodeGroup>` componente alrededor de ellos. Cada Bloque de Código debe tener un nombre de archivo porque usamos los nombres para los botones de las pestañas.

Mira abajo un ejemplo del resultado final.

<CodeGroup>
  ```javascript helloWorld.js
  console.log("Hello World");
  ```

  ```python hello_world.py
  print('Hello World!')
  ```

  ```java HelloWorld.java
  class HelloWorld {
      public static void main(String[] args) {
          System.out.println("Hello, World!");
      }
  }
  ```
</CodeGroup>
