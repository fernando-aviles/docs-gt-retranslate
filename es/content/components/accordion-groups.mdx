---
title: Grupos de Acordeón
description: Agrupa múltiples acordeones en una sola visualización.
icon: table-rows
---

Simplemente agrega`<AccordionGroup>`alrededor de tus existentes`<Accordion>`componentes.

<AccordionGroup>
  <Accordion title="FAQ without Icon">
    Puedes poner otros componentes dentro de los Acordeones.

    ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
  </Accordion>

  <Accordion title="FAQ with Icon" icon="alien-8bit">
    Consulta la documentación de[Accordion](/content/components/accordions)para todas las propiedades soportadas.
  </Accordion>

  <Accordion title="FAQ without Icon">
    Consulta la documentación de[Accordion](/content/components/accordions)para todas las propiedades soportadas.
  </Accordion>
</AccordionGroup>
