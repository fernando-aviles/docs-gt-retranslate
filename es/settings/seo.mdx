## Meta etiquetas predeterminadas

Puedes establecer meta etiquetas predeterminadas agregando el campo`metadata` a tu`docs.json`.

```json
"seo": {
    "metatags": {
        "og:image": "link to your default meta tag image"
    }
}
```

## Cambiar meta etiquetas en una página

Agrega tus meta etiquetas a los metadatos de la página de la siguiente manera:

```md
---
title: 'Your Example Page Title'
'og:image': 'link to your meta tag image'
---
```

<Warning>
  Las meta etiquetas con dos puntos necesitan escribirse entre comillas cuando las colocas en los metadatos de la página.
</Warning>

## Mapas del sitio

Mintlify genera automáticamente un archivo`sitemap.xml` y un archivo`robots.txt`. Simplemente agrega`/sitemap.xml` al final de la URL de tu sitio web para ver tu mapa del sitio. Recomendamos enviar el mapa del sitio a los motores de búsqueda para mejorar el SEO.

Solo las páginas incluidas en tu`docs.json` están incluidas por defecto. Para incluir enlaces ocultos, agrega`seo.indexing` a tu`docs.json`:

```json
"seo": {
    "indexing": all
}
```

## Desactivar la indexación

Si deseas evitar que una página sea indexada por los motores de búsqueda, puedes incluir lo siguiente en el[frontmatter](/page) de tu página:

```
---
noindex: true
---
```

También puedes especificar`noindex` para todas las páginas en tu documentación configurando el campo`metadata.robots` como`"noindex"` en tu`docs.json`:

```json
"seo": {
    "metatags": {
      "robots": "noindex"
    }
  }
```
