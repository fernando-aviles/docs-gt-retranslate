La plataforma genera automáticamente la mayoría de las etiquetas meta.

Sin embargo, puede personalizarlas completamente agregando el campo [metadata](settings#param-metadata) a su `docs.json` o al frontmatter de la página.

## Etiquetas meta globales

Puede establecer etiquetas meta predeterminadas agregando el campo `metadata` a su `docs.json`.

```json
"seo": {
    "metatags": {
        "og:image": "link to your default meta tag image"
    }
}
```

## Etiquetas meta específicas de página

Agregue sus etiquetas meta a los metadatos de la página de la siguiente manera:

```md
---
title: 'Your Example Page Title'
'og:image': 'link to your meta tag image'
---
```

<Warning>
  Las etiquetas meta con dos puntos deben escribirse entre comillas cuando las coloca en los metadatos de la página.
</Warning>

## Mapas del sitio

Mintlify genera automáticamente un archivo `sitemap.xml` y un archivo `robots.txt`. Simplemente navegue a `/sitemap.xml` para ver su mapa del sitio.

Solo las páginas incluidas en su `docs.json` se incluyen por defecto. Para incluir enlaces ocultos, agregue `seo.indexing` a su `docs.json`:

```json
"seo": {
    "indexing": all
}
```

## Desactivar la indexación

Si desea evitar que una página sea indexada por los motores de búsqueda, puede incluir lo siguiente en el [frontmatter](pages) de su página:

```
---
noindex: true
---
```

También puede especificar `noindex` para todas las páginas en su documentación estableciendo el campo `metadata.robots` en `"noindex"` en su `docs.json`:

```json
"seo": {
    "metatags": {
      "robots": "noindex"
    }
  }
```
