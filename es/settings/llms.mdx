Todas las páginas de documentación se alojan automáticamente en varios formatos markdown para facilitar:

* Que los usuarios finales alimenten tus documentos en modelos de lenguaje grande (LLMs) para recuperación específica de información
* Que los LLMs indexen tu contenido eficientemente sin analizar HTML o JavaScript

## Formatos soportados

Soportamos estándares de la industria como [/llms.txt](https://llmstxt.org) así como formatos markdown adicionales.

* **/llms.txt**: Un archivo markdown de un índice estructurado de tus documentos se aloja automáticamente en /llms.txt. Ayuda a los LLMs de propósito general a indexar más eficientemente (por ejemplo, un mapa del sitio para IA).
* **/llms-full.txt**: Un archivo markdown de todo el contenido de tus documentos se aloja automáticamente en /llms-full.txt. Ayuda a los usuarios a cargar el contexto completo en herramientas de IA con un solo enlace.
* **Soporte .md**: Agrega .md a una página individual de documentos para ver una versión markdown de esa página. Ayuda a cargar páginas individuales en herramientas de IA con mejor eficiencia de tokens y tiempo de inferencia.
* **Atajo ⌘ + c**: Copia el código fuente markdown de cualquier página usando comando + c.
