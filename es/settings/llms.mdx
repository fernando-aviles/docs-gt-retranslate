Todas las páginas de documentación se alojan automáticamente en una variedad de formatos markdown para facilitar:

* Usuarios finales para alimentar sus documentos en modelos de lenguaje grande (LLMs) para recuperación específica de información
* LLMs para indexar eficientemente su contenido sin analizar HTML o JavaScript

## Formatos soportados

Soportamos estándares de la industria como [/llms.txt](https://llmstxt.org) así como formatos markdown adicionales.

* **/llms.txt**: Un archivo markdown de un índice estructurado de sus documentos se aloja automáticamente en /llms.txt. Ayuda a los LLMs de propósito general a indexar más eficientemente (por ejemplo, un mapa del sitio para IA).
* **/llms-full.txt**: Un archivo markdown de todo el contenido de sus documentos se aloja automáticamente en /llms-full.txt. Ayuda a los usuarios a cargar el contexto completo en herramientas de IA en un solo enlace.
* **soporte .md**: Agregue .md a una página de documentos individual para ver una versión markdown de esa página. Ayuda a cargar páginas individuales en herramientas de IA con mejor eficiencia de tokens y tiempo de inferencia.
* **atajo ⌘ + c**: Copie el código fuente markdown de cualquier página usando comando + c.
