La `navigation` propiedad controla la jerarquía de tu documentación.

## Páginas

Si no quieres ninguna jerarquía, puedes simplemente definir páginas dentro de tu `navigation` campo.

Cada entrada del `pages` array debe ser una ruta a un archivo que exista dentro de tu repositorio.
Nota que no necesitas agregar `.mdx` a las rutas de archivo.

```json
{
  "navigation": {
    "pages": [
      "overview",
      "quickstart",
      "advanced/components",
      "advanced/integrations"
    ]
  }
}
```

## Grupos

Los grupos te permiten agrupar tus páginas. Los grupos también pueden anidarse entre sí.

```json
{
  "navigation": {
    "groups": [
      {
        "group": "Getting Started",
        "pages": [
          "quickstart",
          {
            "group": "Editing",
            "icon": "pen-paintbrush",
            "pages": ["development", "web-editor"]
          }
        ]
      },
      {
        "group": "Writing Content",
        "pages": ["writing-content/page", "writing-content/text"]
      }
    ]
  }
}
```

## Pestañas

Las pestañas ayudan a distinguir entre diferentes temas o secciones de tu documentación.

<Frame>
  <img className="block dark:hidden rounded-md" src="/images/tabs-light.png" />

  <img className="hidden dark:block rounded-md" src="/images/tabs-dark.png" />
</Frame>

```json docs.json
"navigation": {
  "tabs": [
    {
      "tab": "API References",
      "pages": [
        "api-reference/get",
        "api-reference/post",
        "api-reference/delete"
      ]
    },
    {
      "tab": "SDKs",
      "pages": [
        "sdk/fetch",
        "sdk/create",
        "sdk/delete",
      ]
    },
    {
      "tab": "Blog",
      "href": "https://external-link.com/blog"
    }
  ]
}
```

## Anclas

Las anclas son otra forma de seccionar tu contenido.

<Frame>
  <img className="block dark:hidden rounded-md" src="/images/anchor-light.png" />

  <img className="hidden dark:block rounded-md" src="/images/anchor-dark.png" />
</Frame>

La configuración es muy similar a la configuración de pestañas.

```json docs.json
"navigation": {
  "anchors": [
    {
      "anchor": "API References",
      "pages": [
        "api-reference/get",
        "api-reference/post",
        "api-reference/delete"
      ]
    },
    {
      "anchor": "SDKs",
      "pages": [
        "sdk/fetch",
        "sdk/create",
        "sdk/delete",
      ]
    },
    {
      "anchor": "Blog",
      "href": "https://external-link.com/blog"
    }
  ]
}
```

Puedes usar tanto anclas como pestañas juntas - cualquiera puede anidarse dentro de la otra de manera intercambiable.

## Enlaces Globales

Si quieres usar pestañas o anclas únicamente con el propósito de listar enlaces externos usa la `navigation.global` propiedad.

```json
"navigation": {
  "global": {
    "anchors": [
      {
        "anchor": "Slack",
        "icon": "slack",
        "href": "https://slack.com
      },
      {
        "anchor": "API Reference",
        "icon": "code",
        "href": "https://api-reference.com"
      }
    ]
  }
}
```

### Páginas Ocultas

Los archivos MDX no incluidos en `docs.json` no aparecerán en la barra lateral pero son accesibles enlazando directamente a ellos.

Las páginas ocultas no están indexadas para búsqueda dentro de tu documentación por defecto. Si quieres anular este comportamiento, puedes establecer el atributo[`seo.indexing`](/settings/global#param-indexing) en tu `docs.json` a `navigable`.

## Barra de Navegación

### Enlaces

Agrega enlaces a la barra superior con el `links` campo en el `docs.json` archivo.

<Frame>
  <img className="block rounded-md" src="/images/topbar-links.png" />
</Frame>

El `links` campo admite los siguientes campos: `label`, `href`.

```json
"navbar": {
  "links": [
    {
      "label": "Community",
      "href": "https://mintlify.com/community"
    }
  ]
}
```

### Botón Primario

Personaliza el botón primario en la barra superior usando el `primary`
campo.

<Frame>
  <img className="block rounded-md" src="/images/topbar-cta-get-started.png" />
</Frame>

El `primary` campo admite los siguientes campos: `label`, `href`, `type`. Para más información sobre las opciones para estos campos, consulta el [esquema docs.json](/settings/global#param-navbar).

```json
"navbar": {
  "primary": {
    "type": "button",
    "label": "Get Started",
    "href": "https://mintlify.com/get-started"
  }
}
```

#### GitHub

También puedes configurar el botón CTA para enlazar directamente a tu repositorio de GitHub. Usa el `primary` campo con el `type` establecido en `github`.

<Frame>
  <img className="block rounded-md" src="/images/topbar-cta-github.png" />
</Frame>

```json
"navbar": {
  "primary": {
    "type": "github",
    "label": "GitHub",
    "href": "https://github.com/mintlify/docs"
  }
}
```
