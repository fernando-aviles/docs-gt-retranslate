- title: Heap

Añade lo siguiente a tu archivo `docs.json` para enviar analíticas a Heap.

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "heap": {
          "appId": "required"
      }
  }
  ```

  ```json Example
  "integrations": {
      "heap": {
          "appId": "1234567890"
      }
  }
  ```
</CodeGroup>
