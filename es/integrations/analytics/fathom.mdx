---
- title: Fathom
---

Agrega lo siguiente a tu archivo `docs.json` para enviar analíticas a Fathom.

Puedes obtener el `siteId` de la configuración de tu script.

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "fathom": {
          "siteId": "required"
      }
  }
  ```

  ```json Example
  "integrations": {
      "fathom": {
          "siteId": "YSVMSDAY"
      }
  }
  ```
</CodeGroup>
