---
title: Fathom
---

Agregue lo siguiente a su `docs.json` archivo para enviar análisis a Fathom.

Puede obtener el `siteId` desde la configuración de su script.

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "fathom": {
          "siteId": "required"
      }
  }
  ```

  ```json Example
  "integrations": {
      "fathom": {
          "siteId": "YSVMSDAY"
      }
  }
  ```
</CodeGroup>
