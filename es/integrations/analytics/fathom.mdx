---
title: Fathom
---

Agrega lo siguiente a tu `docs.json` archivo para enviar análisis a Fathom.

Puedes obtener el `siteId` desde la configuración de tu script.

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "fathom": {
          "siteId": "required"
      }
  }
  ```

  ```json Example
  "integrations": {
      "fathom": {
          "siteId": "YSVMSDAY"
      }
  }
  ```
</CodeGroup>
