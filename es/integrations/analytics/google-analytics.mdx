Necesitarás generar una nueva propiedad<Tooltip tip="Google Analytics 4">GA4</Tooltip>para usar con Mintlify. Los datos recopilados irán al mismo proyecto que tus otros datos de Google Analytics.

Si estás usando la versión antigua de Google Analytics, Universal Analytics, aún podrás generar una <Tooltip tip="Google Analytics 4">GA4</Tooltip> propiedad. <Tooltip tip="Google Analytics 4">GA4</Tooltip> los datos son ligeramente diferentes de los datos de UA pero aún se recopilan en el mismo proyecto.

## Cómo Conectar GA4 a Mintlify

### Crear un Stream Web

Necesitarás crear un stream web para obtener el ID de Medición para poner en Mintlify.

Haz clic en el engranaje en la parte inferior izquierda de la pantalla de Google Analytics. Luego haz clic en Flujos de datos.

<Frame>![](/images/ga4-web-streams.png)</Frame>

Crea un Stream Web y coloca la URL de tu sitio de documentación de Mintlify como URL del stream.

Tu ID de Medición se ve como `G-XXXXXXX` y aparecerá bajo Detalles del Stream inmediatamente después de crear el Stream Web.

### Colocar ID de Medición en docs.json

Agrega tu ID de Medición a tu `docs.json` archivo así:

```json docs.json
"integrations": {
    "ga4": {
        "measurementId": "G-XXXXXXX"
    }
}
```

### Esperar

Google Analytics tarda de dos a tres días en mostrar tus datos.

Puedes usar el [Depurador de Google Analytics](https://chrome.google.com/webstore/detail/google-analytics-debugger/jnkmfdileelhofjcijamephohjechhna?hl=en) para verificar que los análisis estén habilitados correctamente. La extensión registrará en la consola de tu navegador cada vez que GA4 realice una solicitud.
