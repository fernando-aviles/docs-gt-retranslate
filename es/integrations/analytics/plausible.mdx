---
- title: Plausible
---

Añade el dominio de tu sitio a `docs.json` para enviar analíticas a Plausible.

<Info>
  No incluyas `https://` para el dominio o servidor.
</Info>

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "plausible": {
          "domain": "required",
          "server": "optional"
      }
  }
  ```

  ```json Example
  "integrations": {
      "plausible": {
          "domain": "docs.domain.com"
      }
  }
  ```
</CodeGroup>
