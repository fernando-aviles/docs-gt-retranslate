Añade lo siguiente a tu archivo `docs.json` para enviar analíticas a Amplitude.

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "amplitude": {
          "apiKey": "required"
      }
  }
  ```

  ```json Example
  "integrations": {
      "amplitude": {
          "apiKey": "76bb138bf3fbf58186XXX00000"
      }
  }
  ```
