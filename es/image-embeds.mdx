## Imagen

### Usando Markdown

La [sintaxis markdown](https://www.markdownguide.org/basic-syntax/#images) te permite agregar imágenes usando el siguiente código

```md
![title](/path/image.jpg)
```

Ten en cuenta que el tamaño del archivo de imagen debe ser menor a 5MB. De lo contrario, recomendamos alojarlo en un servicio como [Cloudinary](https://cloudinary.com/) o [S3](https://aws.amazon.com/s3/). Luego puedes usar esa URL e incrustarla.

### Usando Incrustaciones

Para obtener más personalización con las imágenes, también puedes usar incrustaciones para agregar imágenes.

```html
<img height="200" src="/path/image.jpg" />
```

### Deshabilitar Zoom de Imagen

Para deshabilitar el zoom predeterminado al hacer clic en las imágenes, agrega la propiedad noZoom a las incrustaciones de imagen.

```html
<img height="200" noZoom src="/path/image.jpg" />
```

### Enlazando Imágenes

Para enlazar una imagen, por ejemplo para crear un botón en tu documentación, envuelve la imagen en un enlace con la`noZoom` propiedad. Las imágenes en`a` etiquetas tendrán automáticamente un cursor de puntero.

```html
<a href="https://mintlify.com" target="_blank">
  <img height="200" noZoom src="/path/image.jpg" />
</a>
```

### Modo Oscuro

Para usar imágenes diferentes para modo claro y oscuro, usa Tailwind CSS para ocultar y mostrar imágenes.

```html
<img className="block h-32 dark:hidden" src="/path/image-light.jpg" />
<img className="hidden h-32 dark:block" src="/path/image-dark.jpg" />
```

## Videos

<iframe className="w-full aspect-video rounded-xl" src="https://www.youtube.com/embed/4KzFe50RQkQ" title="YouTube video player" frameBorder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowFullScreen />

<br />

<Tip>
  Mintlify admite [etiquetas HTML en Markdown](https://www.markdownguide.org/basic-syntax/#html). Esto es útil si prefieres etiquetas HTML a la sintaxis Markdown, y te permite crear documentación con flexibilidad infinita.
</Tip>

Para videos de YouTube usa:

```html
<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/4KzFe50RQkQ"
  title="YouTube video player"
  frameborder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowfullscreen
></iframe>
```

Para otros videos, usa:

```html
<video
  controls
  className="w-full aspect-video"
  src="link-to-your-video.com"
></video>
```

Para reproducir el video automáticamente, usa:

```html
<video
  autoPlay
  muted
  loop
  playsInline
  className="w-full aspect-video"
  src="link-to-your-video.com"
></video>
```

<Warning>
  Ya que Mintlify necesita adherirse a la sintaxis JSX, los atributos de doble palabra deberán
  escribirse en camelCase: autoPlay, playsInline.
</Warning>

## iFrames

Carga otra página HTML dentro del documento.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```

<Tip>
  Aunque no es obligatorio, recomendamos agregar los`alt` y `title` atributos a
  las imágenes para mejor SEO y accesibilidad. Aprende más en [imagen
  SEO](https://yoast.com/image-seo-alt-tag-and-title-tag-optimization/).
</Tip>
