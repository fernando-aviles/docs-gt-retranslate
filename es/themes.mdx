Ofrecemos temas prediseñados para que puedas diferenciar el aspecto y la sensación de tu documentación según tu marca.

## Configuración

Puedes configurar el tema en el `docs.json` archivo.

Por ejemplo, para usar el `maple` tema, puedes agregar lo siguiente a tu `docs.json` archivo:

```json
{
  "theme": "maple"
}
```
