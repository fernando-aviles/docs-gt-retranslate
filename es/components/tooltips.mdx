Los tooltips son una forma de mostrar una definición cuando pasas el cursor sobre el texto.

<Tooltip tip="¡Esto es un tooltip!">Pasa el cursor sobre mí</Tooltip> y verás un tooltip en acción

<RequestExample>
  ```jsx Tooltip Example
  <Tooltip tip="This is a tooltip!">Hover over me</Tooltip>
  ```
</RequestExample>
