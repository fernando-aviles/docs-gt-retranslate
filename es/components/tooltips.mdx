---
title: "Tooltips"
description: "Muestra una definición al pasar el cursor sobre el texto"
icon: "message-square"
---

Usa los tooltips para proporcionar contexto adicional o definiciones cuando un usuario pasa el cursor sobre un fragmento de texto. Los tooltips pueden incluir enlaces opcionales con llamadas a la acción.

**Ejemplo**: <Tooltip tip="Interfaz de programación de aplicaciones: un conjunto de protocolos para que las aplicaciones de software se comuniquen." cta="Lee nuestra guía de API" href="/es/api-reference">API</Tooltip> La documentación ayuda a los desarrolladores a entender cómo integrarse con tu servicio.

```mdx Ejemplo de ajuste de tooltip
<Tooltip tip="Interfaz de programación de aplicaciones: conjunto de protocolos que permiten la comunicación entre aplicaciones de software." cta="Lee nuestra guía de API" href="/api-reference">La documentación de la API</Tooltip> ayuda a los desarrolladores a entender cómo integrarse con tu servicio.
```

<div id="properties">
  ## Propiedades
</div>

<ResponseField name="tip" type="string" required>
  El texto que se muestra en el globo de ayuda.
</ResponseField>

<ResponseField name="cta" type="string">
  El texto del llamado a la acción para un enlace dentro del globo de ayuda.
</ResponseField>

<ResponseField name="href" type="string">
  URL del enlace del llamado a la acción. Obligatorio cuando se utiliza `cta`.
</ResponseField>
