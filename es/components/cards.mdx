Usa tarjetas para crear contenedores visuales de contenido. Las tarjetas son contenedores flexibles que pueden incluir texto, iconos, imágenes y enlaces.

<div id="basic-card">
  ## Tarjeta básica
</div>

<Card title="Card title" icon="text" href="/es/components/columns">
  Así se usa una tarjeta con un icono y un enlace. Al hacer clic en esta tarjeta,
  irás a la página Columns.
</Card>

```mdx Ejemplo de tarjeta
<Card title="Título de la tarjeta" icon="text" href="/components/columns">
  Así es como usar una tarjeta con un icono y un enlace. Al hacer clic en esta tarjeta,
  irás a la página de Columnas.
</Card>
```

<div id="card-variations">
  ## Variaciones de tarjetas
</div>

Las tarjetas ofrecen varias opciones de diseño y estilo para adaptarse a distintas necesidades de contenido.

<div id="horizontal-layout">
  ### Diseño horizontal
</div>

Añade la propiedad `horizontal` para mostrar las tarjetas en un diseño más compacto y horizontal.

<Card title="Tarjeta horizontal" icon="text" horizontal>
  Este es un ejemplo de una tarjeta horizontal.
</Card>

```mdx Ejemplo de tarjeta horizontal
<Card title="Tarjeta horizontal" icon="text" horizontal>
  Este es un ejemplo de tarjeta horizontal.
</Card>
```

<div id="image-cards">
  ### Tarjetas con imagen
</div>

Agrega la propiedad `img` para mostrar una imagen en la parte superior de la tarjeta.

<Card title="Tarjeta con imagen" img="https://mintlify-assets.b-cdn.net/yosemite.jpg">
  Este es un ejemplo de una tarjeta con imagen.
</Card>

```mdx Ejemplo de tarjeta con imagen
<Card title="Tarjeta con imagen" img="/images/card-with-image.png">
  Este es un ejemplo de una tarjeta con imagen.
</Card>
```

<div id="link-cards-with-custom-ctas">
  ### Tarjetas de enlace con CTA personalizados
</div>

Puedes personalizar el texto del llamado a la acción y controlar si aparece una flecha. De forma predeterminada, las flechas solo se muestran en los enlaces externos.

<Card title="Link card" icon="link" href="/es/components/columns" arrow="true" cta="Click here">
  Este es un ejemplo de una tarjeta con un ícono y un enlace. Al hacer clic en esta tarjeta, irás a la página Columns.
</Card>

```mdx Ejemplo de tarjeta de enlace
<Card
  title="Tarjeta con enlace"
  icon="link"
  href="/components/columns"
  arrow="true"
  cta="Haz clic aquí"
>
  Este es un ejemplo de una tarjeta con un icono y un enlace. Al hacer clic en esta tarjeta, irás a la página de Columns.
</Card>
```

<div id="grouping-cards">
  ## Agrupar tarjetas
</div>

Usa el [componente Columns](/es/components/columns) para organizar varias tarjetas una junto a la otra.

<Columns cols={2}>
  <Card title="Primera tarjeta" icon="panel-left-close">
    Esta es la primera tarjeta.
  </Card>

  <Card title="Segunda tarjeta" icon="panel-right-close">
    Esta es la segunda tarjeta.
  </Card>
</Columns>

```mdx Ejemplo de columnas
<Columns cols={2}>
  <Card title="Primera tarjeta" icon="panel-left-close">
    Esta es la primera tarjeta.
  </Card>
  <Card title="Segunda tarjeta" icon="panel-right-close">
    Esta es la segunda tarjeta.
  </Card>
</Columns>
```

<div id="properties">
  ## Propiedades
</div>

<ResponseField name="title" type="string" required>
  Título que se muestra en la tarjeta
</ResponseField>

<IconsOptional />

<ResponseField name="color" type="string">
  Color del icono como código hexadecimal (p. ej., `#FF6B6B`).
</ResponseField>

<ResponseField name="href" type="string">
  URL a la que se navega al hacer clic en la tarjeta.
</ResponseField>

<ResponseField name="horizontal" type="boolean">
  Muestra la tarjeta en un diseño horizontal compacto.
</ResponseField>

<ResponseField name="img" type="string">
  URL o ruta local de una imagen que se muestra en la parte superior de la tarjeta.
</ResponseField>

<ResponseField name="cta" type="string">
  Texto personalizado para el botón de acción.
</ResponseField>

<ResponseField name="arrow" type="boolean">
  Mostrar u ocultar el icono de flecha del enlace.
</ResponseField>
