---
- title: Campos de Respuesta
- description: Muestra valores de respuesta de API
---

El componente `<ResponseField>` está diseñado para definir los valores de retorno de una API. Muchos documentos también usan `<ResponseField>` en las páginas cuando necesitas listar los tipos de algo.

<ResponseField name="response" type="string" required>
  Un ejemplo de campo de respuesta
</ResponseField>

```jsx
<ResponseField name="response" type="string" required>
  A response field example
</ResponseField>
```

## Propiedades

<ResponseField name="name" type="string" required>
  El nombre del valor de respuesta.
</ResponseField>

<ResponseField name="type" type="string" required>
  Tipo esperado del valor de respuesta - esto puede ser cualquier cadena arbitraria.
</ResponseField>

<ResponseField name="default" type="string">
  El valor por defecto.
</ResponseField>

<ResponseField name="required" type="boolean">
  Muestra "requerido" junto al nombre del campo.
</ResponseField>

<ResponseField name="deprecated" type="boolean">
  Si un campo está obsoleto o no.
</ResponseField>

<ResponseField name="pre" type="string[]">
  Etiquetas que se muestran antes del nombre del campo
</ResponseField>

<ResponseField name="post" type="string[]">
  Etiquetas que se muestran después del nombre del campo
</ResponseField>
