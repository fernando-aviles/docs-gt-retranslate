`<RequestExample>` 和 `<ResponseExample>` 组件会在右侧边栏显示代码块，形成双栏布局，让示例在用户滚动浏览内容时始终可见。虽然这两个组件主要用于 API 文档，但同样适用于所有页面。

常见用法：

* 带有请求与响应示例的 API 端点文档
* 与说明文字并列的配置示例
* 供用户在操作步骤中参考的代码样例
* 教程中的前后对比示例

在移动设备上，`<RequestExample>` 和 `<ResponseExample>` 组件会显示为常规代码块，可直接滑动越过。

<RequestExample>
  ```bash Request
    curl --request POST \
      --url https://dog-api.kinduff.com/api/facts
  ```
</RequestExample>

<ResponseExample>
  ```json Response
  { "status": "success" }
  ```
</ResponseExample>

<div id="requestexample">
  ## RequestExample
</div>

使用 `<RequestExample>` 将代码示例固定在右侧边栏。该组件与 [CodeGroup](/zh/components/code-groups) 组件类似，但会把代码显示在侧边栏中，而不是内联显示。

你可以在单个 `<RequestExample>` 中包含多个代码块。每个代码块都必须带有一个 title 属性。

````mdx 请求示例
<RequestExample>

```bash 请求
  curl --request POST \
    --url https://dog-api.kinduff.com/api/facts
```

</RequestExample>
````

<div id="responseexample">
  ## ResponseExample
</div>

`<ResponseExample>` 组件会将代码示例固定在同一页面右侧边栏中所有 `<RequestExample>` 内容的下方。

````mdx 响应示例
<ResponseExample>

```json 响应
{ "status": "success" }
```

</ResponseExample>
````
