---
title: "工具提示"
description: "鼠标悬停文本时显示定义"
icon: "message-square"
---

使用工具提示可在用户将鼠标悬停在一段文本上时提供额外的上下文或定义。工具提示还可以包含可选的行动链接。

**示例**：<Tooltip tip="应用程序编程接口：用于使软件应用程序相互通信的一组协议。" cta="阅读我们的 API 指南" href="/zh/api-reference">API</Tooltip> 文档有助于开发者了解如何与您的服务集成。

```mdx 工具提示示例包装
<Tooltip tip="应用程序编程接口：用于使软件应用相互通信的一组协议。" cta="阅读我们的 API 指南" href="/api-reference">API</Tooltip> 文档可帮助开发者了解如何将您的服务集成到他们的应用中。
```

<div id="properties">
  ## 属性
</div>

<ResponseField name="tip" type="string" required>
  工具提示中显示的文本。
</ResponseField>

<ResponseField name="cta" type="string">
  工具提示内链接的行动号召文案。
</ResponseField>

<ResponseField name="href" type="string">
  行动号召链接的 URL。使用 `cta` 时必填。
</ResponseField>
