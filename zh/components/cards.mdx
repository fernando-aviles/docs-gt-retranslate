使用卡片为内容创建可视化容器。卡片是一种灵活的容器，可包含文本、图标、图像和链接。

  会跳转到 Columns 页面。

<div id="card-variations">
  ## 卡片变体
</div>

卡片支持多种布局和样式选项，以满足不同的内容需求。

<div id="horizontal-layout">
  ### 水平布局
</div>

添加 `horizontal` 属性即可将卡片以更紧凑的水平布局展示。

<Card title="Horizontal card" icon="text" horizontal>
  这是一个水平卡片的示例。
</Card>

```mdx 横向卡片示例
<Card title="横向卡片" icon="text" horizontal>
  这是一个横向卡片示例。
</Card>
```

<div id="image-cards">
  ### 图片卡片
</div>

添加 `img` 属性即可在卡片顶部显示图片。

<Card title="图片卡片" img="https://mintlify-assets.b-cdn.net/yosemite.jpg">
  这是一个带图片的卡片示例。
</Card>

```mdx 图片卡片示例
<Card title="图片卡片" img="/images/card-with-image.png">
  这是一个包含图片的卡片示例。
</Card>
```

<div id="link-cards-with-custom-ctas">
  ### 带自定义 CTA 的链接卡片
</div>

你可以自定义行动号召文本，并控制是否显示箭头。默认情况下，箭头只会显示在外部链接上。

<Card title="Link card" icon="link" href="/zh/components/columns" arrow="true" cta="Click here">
  这是一个带有图标和链接的卡片示例。点击此卡片将跳转到 Columns 页面。
</Card>

```mdx 链接卡片示例
<Card
  title="链接卡片"
  icon="link"
  href="/components/columns"
  arrow="true"
  cta="点击此处"
>
  这是一个带有图标和链接的卡片示例。点击该卡片将带你前往 Columns 页面。
</Card>
```

<div id="grouping-cards">
  ## 分组卡片
</div>

使用 [Columns 组件](/zh/components/columns) 将多个卡片并排排列。

<Columns cols={2}>
  <Card title="第一张卡片" icon="panel-left-close">
    这是第一张卡片。
  </Card>

  <Card title="第二张卡片" icon="panel-right-close">
    这是第二张卡片。
  </Card>
</Columns>

```mdx Columns 示例
<Columns cols={2}>
  <Card title="第一张卡片" icon="panel-left-close">
    这是第一张卡片。
  </Card>
  <Card title="第二张卡片" icon="panel-right-close">
    这是第二张卡片。
  </Card>
</Columns>
```

<div id="properties">
  ## 属性
</div>

<ResponseField name="title" type="string" required>
  卡片上显示的标题
</ResponseField>

<IconsOptional />

<ResponseField name="color" type="string">
  图标颜色的十六进制值（例如：`#FF6B6B`）。
</ResponseField>

<ResponseField name="href" type="string">
  点击卡片后跳转的URL。
</ResponseField>

<ResponseField name="horizontal" type="boolean">
  以紧凑的横向布局显示卡片。
</ResponseField>

<ResponseField name="img" type="string">
  卡片顶部图片的URL或本地路径。
</ResponseField>

<ResponseField name="cta" type="string">
  操作按钮的自定义文本。
</ResponseField>

<ResponseField name="arrow" type="boolean">
  显示或隐藏链接箭头图标。
</ResponseField>
