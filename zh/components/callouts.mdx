---
title: '标注'
description: '使用标注为内容添加醒目的上下文'
icon: 'info'
---

标注可以设置为 Note、Warning、Info、Tip、Check、Danger，或创建你自己的标注：

<Note>这会在内容中添加一条说明</Note>

```mdx
<Note>这会在内容中添加一条备注</Note>
```

<Warning>这是一个需要留意的警示</Warning>

```mdx
<Warning>这是一条需要注意的警告</Warning>
```

<Info>用于突出显示重要信息</Info>

```mdx
<Info>用于强调重要信息</Info>
```

<Tip>这是一个有用的提示</Tip>

```mdx
<Tip>这是一个实用提示</Tip>
```

<Check>这会呈现一个已勾选状态</Check>

```mdx
<Check>这会呈现“已勾选”状态</Check>
```

<Danger>这是一个危险警示</Danger>

```mdx
<Danger>这是一个危险警示</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular"> 这是一个自定义提醒</Callout>

```mdx 换行
<Callout icon="key" color="#FFC107" iconType="regular">这是一个自定义标注</Callout>
```
