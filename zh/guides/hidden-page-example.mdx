此页面为隐藏状态！它未包含在 `docs.json` 的导航中，因此只能通过其 URL 访问。

请参阅[隐藏页面](/zh/guides/hidden-pages)了解更多信息。

<div id="common-use-cases-for-hidden-pages">
  ## 隐藏页面的常见用例
</div>

<Note>
  默认情况下，隐藏页面可公开访问，但不会在导航中被发现。若要限制对隐藏页面的访问，必须配置[身份验证](/zh/authentication-personalization/authentication-setup)。
</Note>

* **Beta 文档**：内容可以公开，但不应通过导航被发现。
* **AI 工具上下文**：如果隐藏页面被索引，AI 工具可以引用其中内容作为上下文。将对用户不重要但有助于 AI 工具提供更准确回答的内容放在隐藏页面中。
* **旧版页面**：通过直接链接保留旧内容可访问，同时将其从导航菜单中移除。
* **内部工具**：记录内部工具、预发布环境 API 或开发流程。
