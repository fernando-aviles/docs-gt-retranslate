<div id="lists">
  ## 列表
</div>

列表遵循官方的 [Markdown 语法](https://www.markdownguide.org/basic-syntax/#lists-1)。

<div id="ordered-list">
  ### 有序列表
</div>

要创建有序列表，在每个列表项前加上数字并紧跟一个句点。

1. 第一项
2. 第二项
3. 第三项
4. 第四项

```mdx
1. First item
2. Second item
3. Third item
4. Fourth item
```

<div id="unordered-list">
  ### 无序列表
</div>

要创建无序列表，在每个列表项前加上短横线（`-`）、星号（`*`）或加号（`+`）。

- 第一项
- 第二项
- 第三项
- 第四项

```mdx
- First item
- Second item
- Third item
- Fourth item
```

<div id="nested-list">
  ### 嵌套列表
</div>

通过缩进来创建嵌套的列表项。

- 第一项
- 第二项
  - 附加项
  - 附加项
- 第三项

```mdx
- First item
- Second item
  - Additional item
  - Additional item
- Third item
```



<div id="tables">
  ## 表格
</div>

表格遵循官方的 [Markdown 语法](https://www.markdownguide.org/extended-syntax/#tables)。

要添加表格，请使用三个或更多连字符（`---`）来创建各列的表头，并使用竖线（`|`）分隔每一列。为提高兼容性，建议在每行的开头和结尾都添加一个竖线。

| 属性     | 说明                                   |
| -------- | -------------------------------------- |
| Name     | 用户全名                               |
| Age      | 填报年龄                               |
| Joined   | 是否已加入社区                         |

```mdx
| Property | Description                           |
| -------- | ------------------------------------- |
| Name     | Full name of user                     |
| Age      | Reported age                          |
| Joined   | Whether the user joined the community |
```
