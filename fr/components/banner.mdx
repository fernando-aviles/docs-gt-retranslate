title: "Bannière"
description: "Ajouter une bannière pour afficher des annonces et notifications importantes à l’échelle du site"
icon: "layout-panel-top"
---

Utilisez des bannières pour afficher des annonces, mises à jour ou notifications importantes sur l’ensemble de votre site de documentation. Les bannières apparaissent en haut de chaque page, prennent en charge la mise en forme Markdown et peuvent être rendues ignorables.

Pour ajouter une bannière, utilisez la propriété `banner` dans votre `docs.json` :

<CodeGroup>
  ```json Product announcements wrap
  "banner": {
    "content": "🚀 La version 2.0 est maintenant disponible ! Consultez notre [journal des modifications](/changelog) pour plus de détails.",
    "dismissible": true 
  }
  ```

  ```json Maintenance notices wrap
  "banner": {
    "content": "⚠️ Maintenance planifiée : l’API sera indisponible le 15 décembre de 2 h à 4 h UTC",
    "dismissible": false
  }
  ```

  ```json Required actions wrap
  "banner": {
    "content": "**Action requise :** migrez vers notre nouvelle version avant le 1er janvier. [Guide de migration](/migration)",
    "dismissible": true
  }
  ```
</CodeGroup>

<div id="properties">
  ## Propriétés
</div>

<ResponseField name="content" type="string" required>
  Le message de la bannière. Prend en charge le texte simple et la mise en forme Markdown.
</ResponseField>

<ResponseField name="dismissible" type="boolean">
  Indique si les utilisateurs peuvent ignorer la bannière. Lorsque `true`, ils peuvent fermer la bannière et elle ne réapparaît pas pendant leur session. Par défaut : `false`.
</ResponseField>
