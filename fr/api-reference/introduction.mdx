L’API REST de Mintlify vous permet d’interagir avec votre documentation par programmation, de déclencher des mises à jour et d’intégrer des expériences de chat propulsées par l’IA.

<div id="endpoints">
  ## Endpoints
</div>

* [Déclencher une mise à jour](/fr/api-reference/update/trigger): Déclenchez une mise à jour de votre site à tout moment.
* [Obtenir l’état de la mise à jour](/fr/api-reference/update/status): Obtenez l’état d’une mise à jour et d’autres informations sur votre documentation.
* [Générer un message de l’Assistant](/fr/api-reference/assistant/create-assistant-message): Intégrez l’Assistant, entraîné sur votre documentation, dans l’application de votre choix.
* [Rechercher dans la documentation](/fr/api-reference/assistant/search): Recherchez dans votre documentation.

<div id="authentication">
  ## Authentification
</div>

Vous pouvez générer une clé API depuis [le tableau de bord](https://dashboard.mintlify.com/settings/organization/api-keys). Les clés API sont associées à l’ensemble de l’organisation et peuvent être utilisées sur plusieurs déploiements.

<div id="admin-api-key">
  ### Clé API d’administrateur
</div>

La clé API d’administrateur est utilisée pour les endpoints [Déclencher une mise à jour](/fr/api-reference/update/trigger) et [Obtenir l’état de la mise à jour](/fr/api-reference/update/status).

Les clés API d’administrateur commencent par le préfixe `mint_`. Gardez vos clés API d’administrateur secrètes.

<div id="assistant-api-key">
  ### Clé API de l’Assistant
</div>

La clé API de l’Assistant est utilisée pour les endpoints [Générer un message de l’Assistant](/fr/api-reference/assistant/create-assistant-message) et [Rechercher dans la documentation](/fr/api-reference/assistant/search).

Les clés API de l’Assistant commencent par le préfixe `mint_dsc_`.

La **clé** API de l’Assistant est un jeton côté serveur qui doit rester secret.

Le **jeton** API de l’Assistant est un jeton public qui peut être référencé dans votre code frontend.

<Note>
  Les appels utilisant le jeton API de l’Assistant peuvent entraîner des coûts : soit en utilisant vos crédits d’Assistant IA, soit en engendrant des dépassements.
</Note>
