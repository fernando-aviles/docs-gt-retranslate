---
title: "Page 404 personnalisée"
description: "Personnalisez le titre et la description de votre page d’erreur 404"
icon: "file-warning"
---

Vous pouvez définir le titre et la description de la page d’erreur 404 qui s’affiche lorsque les utilisateurs suivent des liens rompus ou introuvables.

Lors de la personnalisation de votre page 404, utilisez la description pour orienter les utilisateurs vers des ressources utiles ou des liens dans votre documentation afin de les aider à trouver ce qu’ils recherchent.

<div id="configuration">
  ## Configuration
</div>

Configurez votre page 404 dans la section `errors.404` de votre fichier `docs.json` :

```json
"errors": {
  "404": {
    "redirect": false,
    "title": "Introuvable",
    "description": "Qu’est-il donc **arrivé** à cette _page_ ?"
  }
}
```


<div id="parameters">
  ## Paramètres
</div>

<ResponseField name="redirect" type="boolean">
  Indique s’il faut rediriger automatiquement vers la page d’accueil lorsqu’une page est introuvable.
  
  Définir sur `true` pour rediriger vers la page d’accueil.

  Définir sur `false` pour afficher la page 404 personnalisée.
</ResponseField>

<ResponseField name="title" type="string">
  Titre personnalisé pour la page d’erreur 404. Remplace le titre par défaut « Page not found ».
</ResponseField>

<ResponseField name="description" type="string">
  Description personnalisée pour la page d’erreur 404. Prend en charge la mise en forme Markdown.
</ResponseField>
