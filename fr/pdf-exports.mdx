<Info>
Les exportations PDF sont disponibles avec les [offres Custom](https://mintlify.com/pricing).
</Info>

Vous pouvez exporter votre documentation dans un seul fichier PDF pour une consultation hors ligne. Le PDF inclura l’intégralité du contenu de la documentation, y compris les images et les liens, ainsi qu’une table des matières navigable.

<div id="exporting-a-pdf">
  ## Exporter un PDF
</div>

1. Accédez à la page [General](https://dashboard.mintlify.com/settings/deployment/general) dans la section Project Settings de votre tableau de bord.
2. Sélectionnez le bouton **Export all content**.
3. Facultativement, personnalisez les options d’export :
    - **Page format** : Choisissez la taille de page du PDF.
    - **Scale percentage** : Ajustez l’échelle du PDF.
    - **Include footer** : Incluez un pied de page avec le numéro de page et le nombre total de pages.
4. Sélectionnez **Export** pour lancer l’export.
5. Une fois l’export terminé, vous recevrez un e‑mail avec un lien de téléchargement.
6. Cliquez sur le lien pour télécharger le fichier PDF.
