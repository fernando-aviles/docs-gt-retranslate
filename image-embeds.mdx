<Info>
  Mintlify supports files up to 5 MB. For larger files, use services like Cloudinary.
</Info>

## Images

### Markdown Syntax
```md
![title](/path/image.jpg)
```

### HTML Embed
```html
<img height="200" src="/path/image.jpg" />
```

### Additional Image Features
- Disable zoom: Add `noZoom` property
- Link images: Wrap in `<a>` tag
- Dark mode support: Use Tailwind classes (`dark:hidden`/`dark:block`)

## Videos

```html
<iframe
  className="w-full aspect-video"
  src="https://www.youtube.com/embed/4KzFe50RQkQ"
  title="YouTube video player"
  frameBorder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowFullScreen
></iframe>
```

For other videos:
```html
<video
  controls
  className="w-full aspect-video"
  src="link-to-your-video.com"
></video>
```

<Warning>
Use camelCase for multi-word attributes (e.g., autoPlay, playsInline).
</Warning>

<Tip>
Add `alt` and `title` attributes to images for better SEO and accessibility.
</Tip>
