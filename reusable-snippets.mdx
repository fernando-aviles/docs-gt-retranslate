Reusable snippets help you maintain consistent content across your documentation. Instead of copying and pasting the same content multiple times, create a snippet once and reuse it anywhere.

## Creating Custom Snippets

All snippets must be created in the `snippets` directory. Files in this directory won't generate standalone pages unless imported into another file.

### Basic Snippet

1. Create your snippet file:

```typescript snippets/my-snippet.mdx
Hello world! This is my reusable content.
```

2. Import and use the snippet:

```typescript destination-file.mdx
---
title: My title
description: My Description
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

<MySnippet/>
```

### Snippets with Variables

1. Create a snippet with variables:

```typescript snippets/my-snippet.mdx
My keyword of the day is {word}.
```

2. Use the snippet with variables:

```typescript destination-file.mdx
import MySnippet from '/snippets/path/to/my-snippet.mdx';

<MySnippet word="bananas" />
```

### Reusable Variables

1. Export variables:

```typescript snippets/path/to/custom-variables.mdx
export const myName = 'my name';
export const myObject = { fruit: 'strawberries' };
```

2. Import and use variables:

```typescript destination-file.mdx
import { myName, myObject } from '/snippets/path/to/custom-variables.mdx';

Hello, my name is {myName} and I like {myObject.fruit}.
```

### Reusable Components

1. Create a component:

```typescript snippets/custom-component.mdx
export const MyComponent = ({ title }) => (
  <div>
    <h1>{title}</h1>
    <p>... snippet content ...</p>
  </div>
);
```

<Warning>
  Use HTML syntax in arrow functions instead of MDX. For MDX syntax, use default exports.
</Warning>

2. Use the component:

```typescript destination-file.mdx
import { MyComponent } from '/snippets/custom-component.mdx';

<MyComponent title={'Custom title'} />
```

### Client-Side Content

For client-side rendering, check for the `document` object before rendering:

```typescript snippets/client-component.mdx
export const ClientComponent = () => {
  if (typeof document === "undefined") {
    return null;
  }

  setTimeout(() => {
    const clientComponent = document.getElementById("client-component");
    if (clientComponent) {
      clientComponent.innerHTML = "Hello, client-side component!";
    }
  }, 1);

  return <div id="client-component"></div>
}
```
