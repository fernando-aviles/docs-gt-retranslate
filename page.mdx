## Page Configuration

Each MDX file requires metadata between `---` lines at the start:

<CodeGroup>

```md Basic
---
title: "Your Page Title"
description: "Brief description of the page"
---
```

```md Extended
---
title: "Your Page Title"
description: "Brief description"
sidebarTitle: "Short Navigation Title" # Optional shorter title for sidebar
icon: "rectangle-code" # Font Awesome icon name
iconType: "solid" # Optional: defaults to regular
url: "https://external-link.com" # Optional: for external links
---
```

</CodeGroup>

## Page Modes

Choose from three display modes:

- **Default**: Standard layout with table of contents
- **Wide**: Hides table of contents (`mode: "wide"`)
- **Custom**: Minimal layout with only top bar (`mode: "custom"`)

## API Pages

Enable API playgrounds by adding the `api` or `openapi` property:

```md
---
api: "GET /users"
# or
openapi: "GET /users"
---
```

## SEO Optimization

Add meta tags for social sharing and SEO:

```md
---
"twitter:image": "/images/your-photo.jpg"
# Add other meta tags as needed
---
```

For all supported meta tags, see [SEO settings](/settings/seo).
