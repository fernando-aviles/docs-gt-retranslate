<Note>
  Wenn Sie derzeit die Legacy`mint.json` Konfigurationsdatei verwenden, aktualisieren Sie bitte die CLI:

  ```sh
  npm i -g mintlify@latest
  ```

  Und führen Sie den neuen`upgrade` Befehl in Ihrem Docs-Repository aus:

  ```sh
  mintlify upgrade
  ```

  Dies generiert eine`docs.json` basierend auf Ihrer`mint.json`. Bitte löschen Sie dann die`mint.json` Datei aus Ihrem Repository.
</Note>
