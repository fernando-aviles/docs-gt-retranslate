Die `<RequestExample>` und `<ResponseExample>` fixieren Codeblöcke oben rechts auf einer Seite, auch während Sie scrollen. Die Komponenten funktionieren auf allen Seiten, auch wenn Sie keinen API-Playground verwenden.

`<RequestExample>` und `<ResponseExample>` werden auf Mobilgeräten wie normale Codeblöcke angezeigt.

## Anfrage-Beispiel

Die `<RequestExample>` Komponente funktioniert ähnlich wie [CodeGroup](/content/components/code-groups), zeigt aber den Anfrage-Inhalt in der rechten Seitenleiste an. Daher können Sie mehrere Codeblöcke innerhalb von `<RequestExample>` platzieren.

Bitte setzen Sie einen Namen für jeden Codeblock, den Sie in RequestExample platzieren.

<RequestExample>
  ````md RequestExample Example
  <RequestExample>

  ```bash Request
    curl --request POST \
      --url https://dog-api.kinduff.com/api/facts
  ```

  </RequestExample>
  ````
</RequestExample>

## Antwort-Beispiel

Die `<ResponseExample>` Komponente ist die gleiche wie `<RequestExample>` wird aber darunter angezeigt.

<ResponseExample>
  ````md ResponseExample Example
  <ResponseExample>

  ```json Response
  { "status": "success" }
  ```

  </ResponseExample>
  ````
</ResponseExample>
