---
title: Antwortfelder
description: API-Antwortswerte anzeigen

---

Die `<ResponseField>` Komponente ist dafür konzipiert, die Rückgabewerte einer API zu definieren. Viele Dokumentationen verwenden auch `<ResponseField>` auf Seiten, wenn Sie die Typen von etwas auflisten müssen.

<ResponseField name="response" type="string" required>
  Ein Antwortfeld-Beispiel
</ResponseField>

<RequestExample>
  ```jsx ResponseField Example
  <ResponseField name="response" type="string" required>
    A response field example
  </ResponseField>
  ```
</RequestExample>

## Eigenschaften

<ResponseField name="name" type="string" required>
  Der Name des Antwortwertes.
</ResponseField>

<ResponseField name="type" type="string" required>
  Erwarteter Typ des Antwortwertes
</ResponseField>

<ResponseField name="default" type="string">
  Der Standardwert.
</ResponseField>

<ResponseField name="required" type="boolean">
  Zeigt "erforderlich" neben dem Feldnamen an.
</ResponseField>
