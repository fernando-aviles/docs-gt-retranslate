---
title: Hinweisfelder
description: Verwenden Sie Hinweise, um auffälligen Kontext zu Ihrem Inhalt hinzuzufügen
icon: circle-exclamation
---

### Notiz-Hinweise

<Note>Dies fügt eine Notiz im Inhalt hinzu</Note>

```jsx
<Note>This adds a note in the content</Note>
```

### Warnungs-Hinweise

<Warning>Dies weist auf eine Warnung hin, auf die zu achten ist</Warning>

```jsx
<Warning>This raises a warning to watch out for</Warning>
```

### Info-Hinweise

<Info>Dies lenkt die Aufmerksamkeit auf wichtige Informationen</Info>

```jsx
<Info>This draws attention to important information</Info>
```

### Tipp-Hinweise

<Tip>Dies schlägt einen hilfreichen Tipp vor</Tip>

```jsx
<Tip>This suggests a helpful tip</Tip>
```

### Häkchen-Hinweise

<Check>Dies zeigt uns einen geprüften Status</Check>

```jsx
<Check>This brings us a checked status</Check>
```
