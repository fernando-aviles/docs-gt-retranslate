Fügen Sie einfach`<AccordionGroup>`um Ihre bestehenden`<Accordion>`Komponenten hinzu.

<AccordionGroup>
  <Accordion title="FAQ without Icon">
    Sie können andere Komponenten innerhalb von Akkordeons platzieren.

    ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
  </Accordion>

  <Accordion title="FAQ with Icon" icon="alien-8bit">
    Schauen Sie sich die[Akkordeon](/content/components/accordions)Dokumentation für alle unterstützten Eigenschaften an.
  </Accordion>

  <Accordion title="FAQ without Icon">
    Schauen Sie sich die[Akkordeon](/content/components/accordions)Dokumentation für alle unterstützten Eigenschaften an.
  </Accordion>
</AccordionGroup>
