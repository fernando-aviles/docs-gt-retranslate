Mintlify integriert sich mit der GitHub API und ermöglicht die Synchronisation zwischen Ihren
Docs und Ihrem GitHub-Repository. Diese Integration wird durch die
Verwendung von[GitHub Apps](https://docs.github.com/en/developers/apps/getting-started-with-apps/about-apps#about-github-apps).

## Installation

Zunächst müssen Sie die Mintlify GitHub App in dem GitHub-Konto installieren,
in dem sich Ihr Docs-Repository befindet. Die Installation einer GitHub App erfordert entweder
Organisations-Besitz oder Admin-Berechtigungen in einem Repository. Wenn Sie nicht über die
erforderlichen Berechtigungen verfügen, muss der Repository-Besitzer die Anfrage genehmigen.
Sie können auf die Installationsseite zugreifen, indem Sie sich im[Mintlify Dashboard](https://dashboard.mintlify.com).

Bei GitHub Apps können Sie die Berechtigungen auf ein einzelnes Repository beschränken.
Wir empfehlen dies ausdrücklich, da wir nur Zugriff auf das Repository benötigen, in dem
Ihre Docs gehostet werden.

## Berechtigungen

Während der Installation unserer GitHub App werden Sie aufgefordert, bestimmte
Berechtigungen zu erteilen:

* Lese- und Schreibzugriff auf `checks`, `contents`, `deployments`, `pull requests`,
  und `workflows`

Diese Berechtigungen werden genutzt, um eine nahtlose Erfahrung bei der Verwaltung
Ihrer Docs zu bieten.

Wenn Sie einen Commit in dem Branch machen, den Sie als Ihren Docs-Deployment-
Branch konfiguriert haben, rufen wir die Inhalte der geänderten Dateien ab, um Ihre Docs zu aktualisieren. Dafür benötigen wir Lesezugriff auf Ihre `contents`.

Wenn Pull Requests erstellt werden, erstellen wir eine Überprüfung und eine Vorschau-Bereitstellung, weshalb wir Schreibzugriff benötigen auf`checks` und `deployments`.

Im Mintlify Web Editor arbeitet Mintlify in Ihrem Namen, um Branches und Pull Requests zu erstellen, weshalb wir Schreibzugriff benötigen auf`pull requests` und
`workflows`.

<Info>
  Wenn Sie Bedenken bezüglich der Schreibberechtigungen haben, hat die GitHub App nur Zugriff auf die Repositories, denen Sie Zugriff gewähren*und* wenn Sie Branch-Schutz aktiviert haben*kann sie* nicht direkt in Ihre Branches pushen - sie hält sich an Ihre Branch-Schutzregeln.
</Info>

## Repository-Installationen

Bei der Installation unserer GitHub App werden Sie aufgefordert, alle Repositories oder eine Teilmenge davon auszuwählen. Diese Auswahl kann jederzeit geändert werden, indem Sie zu den
[GitHub App-Einstellungen](https://github.com/apps/mintlify/installations/new).
