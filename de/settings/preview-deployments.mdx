Mit Vorschau-Deployments können Sie Änderungen an Ihrer Dokumentation in einem Live-Deployment vorschauen, ohne diese Änderungen in Ihren Produktionszweig zu übernehmen.

Wenn Sie bei der Verwendung von Git einen Pull- oder Merge-Request erstellt haben, wird die generierte Vorschau-URL als Kommentar vom Mintlify-Bot im PR verfügbar sein. Diese URL wird immer die neuesten Deployment-Änderungen widerspiegeln.

## Vorschau-Deployments teilen

Nach dem Generieren einer Vorschau-URL können Sie diesen Link mit jedem teilen, damit sie ihn ansehen können.

### Authentifizierte Vorschau-Deployments

Wenn Sie den Zugriff auf Ihre Vorschau-Deployments einschränken möchten, können Sie die Vorschau-Deployment-Authentifizierung einrichten. Bei dieser Authentifizierungsmethode können nur authentifizierte Mintlify-Benutzer auf Ihre Vorschau-Deployments zugreifen.

Um die Vorschau-Deployment-Authentifizierung einzurichten, bitte [kontaktieren Sie den Vertrieb](mailto:sales@mintlify.com).
