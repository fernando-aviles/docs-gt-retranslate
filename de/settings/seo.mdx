  * og:image:height
  * robots

## Standard Meta-Tags

Sie können Standard-Meta-Tags festlegen, indem Sie das`metadata` Feld zu Ihrer`docs.json`.

```json
"seo": {
    "metatags": {
        "og:image": "link to your default meta tag image"
    }
}
```

## Meta-Tags auf einer Seite ändern

Fügen Sie Ihre Meta-Tags wie folgt zu den Metadaten der Seite hinzu:

```md
---
title: 'Your Example Page Title'
'og:image': 'link to your meta tag image'
---
```

<Warning>
  Meta-Tags mit Doppelpunkten müssen in Anführungszeichen gesetzt werden, wenn Sie sie in die Seiten-Metadaten einfügen.
</Warning>

## Sitemaps

Mintlify generiert automatisch eine`sitemap.xml` Datei und eine`robots.txt` Datei. Fügen Sie einfach`/sitemap.xml` am Ende Ihrer Website-URL hinzu, um Ihre Sitemap zu sehen. Wir empfehlen, die Sitemap bei Suchmaschinen einzureichen, um die SEO zu verbessern.

Nur Seiten, die in Ihrer`docs.json` enthalten sind, werden standardmäßig einbezogen. Um versteckte Links einzubeziehen, fügen Sie`seo.indexing` zu Ihrer`docs.json`:

```json
"seo": {
    "indexing": all
}
```

## Indexierung deaktivieren

Wenn Sie verhindern möchten, dass eine Seite von Suchmaschinen indexiert wird, können Sie Folgendes in den[Frontmatter](/page) Ihrer Seite einfügen:

```
---
noindex: true
---
```

Sie können auch`noindex` für alle Seiten in Ihren Docs festlegen, indem Sie das`metadata.robots` Feld auf`"noindex"` in Ihrer`docs.json`:

```json
"seo": {
    "metatags": {
      "robots": "noindex"
    }
  }
```
