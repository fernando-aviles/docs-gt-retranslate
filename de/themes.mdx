Wir bieten vorgefertigte Themes an, damit Sie das Aussehen und die Handhabung Ihrer Dokumentation entsprechend Ihrer Marke differenzieren können.

## Konfiguration

Sie können das Theme in der `docs.json` Datei konfigurieren.

Um zum Beispiel das `maple` Theme zu verwenden, können Sie Folgendes zu Ihrer `docs.json` Datei hinzufügen:

```json
{
  "theme": "maple"
}
```
