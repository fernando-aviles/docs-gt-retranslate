---
title: Front
---

Fügen Sie Folgendes zu Ihrer`docs.json` Datei hinzu, um ein[Front Chat](https://front.com) Widget hinzuzufügen.

<CodeGroup>
  ```json Integration options in docs.json
  "integrations": {
      "frontchat": "CHAT_ID"
  }
  ```

  ```json Example
  "integrations": {
      "frontchat": "1365d046d7c023e9b030ce90d02d093a"
  }
  ```
</CodeGroup>
