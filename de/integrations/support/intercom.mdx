---
title: Intercom
---

Fügen Sie Folgendes zu Ihrer`docs.json` Datei hinzu, um ein[Intercom](https://www.intercom.com) Widget hinzuzufügen.

<CodeGroup>
  ```json Integration options in docs.json
  "integrations": {
        "intercom": {
              "appId": "APP_ID"
        }
  }
  ```

  ```json Example
  "integrations": {
        "intercom": {
              "appId": "APP_ID"
        }
  }
  ```
</CodeGroup>
