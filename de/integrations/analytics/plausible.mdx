---
title: Plausible
---

Fügen Sie die Domain Ihrer Website zu `docs.json` um Analytics an Plausible zu senden.

<Info>
  Nicht einschließen`https://` für die Domain oder den Server.
</Info>

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "plausible": {
          "domain": "required",
          "server": "optional"
      }
  }
  ```

  ```json Example
  "integrations": {
      "plausible": {
          "domain": "docs.domain.com"
      }
  }
  ```
</CodeGroup>
