---
title: Fathom
---

Fügen Sie Folgendes zu Ihrer`docs.json` Datei hinzu, um Analytics an Fathom zu senden.

Sie können die`siteId` aus Ihren Skript-Einstellungen erhalten.

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "fathom": {
          "siteId": "required"
      }
  }
  ```

  ```json Example
  "integrations": {
      "fathom": {
          "siteId": "YSVMSDAY"
      }
  }
  ```
</CodeGroup>
