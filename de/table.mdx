| Eigenschaft | Beschreibung                                  |
| ----------- | --------------------------------------------- |
| Name        | Vollständiger Name des Benutzers              |
| Alter       | Angegebenes Alter                             |
| Beigetreten | Ob der Benutzer der Community beigetreten ist |

### Eine Tabelle erstellen

<Tip>
  Die Tabellen-Komponente folgt der offiziellen [Markdown-Syntax](https://www.markdownguide.org/extended-syntax/#tables).
</Tip>

<span className="hover:bg-cyan-500">
  Um eine Tabelle hinzuzufügen
</span>

, verwenden Sie drei oder mehr Bindestriche (`---`), um die Kopfzeile jeder Spalte zu erstellen, und verwenden Sie Pipes (`|`), um jede Spalte zu trennen. Für die Kompatibilität sollten Sie auch eine Pipe an beiden Enden der Zeile hinzufügen.

```md
| Property | Description                           |
| -------- | ------------------------------------- |
| Name     | Full name of user                     |
| Age      | Reported age                          |
| Joined   | Whether the user joined the community |
```
