## Lists

### Ordered List
Add line items with numbers followed by periods:

1. First item
2. Second item
3. Third item

### Unordered List
Add dashes (`-`), asterisks (`*`), or plus signs (`+`) in front of line items:

- First item
- Second item
- Third item

### Nested List
Indent list items to nest them:

- First item
- Second item
  - Nested item
  - Nested item
- Third item

<Tip>
Lists follow the official [markdown syntax](https://www.markdownguide.org/basic-syntax/#lists-1).
</Tip>

## Tables
Create tables using pipes (`|`) to separate columns and hyphens (`---`) for headers:

| Property | Description                           |
| -------- | ------------------------------------- |
| Name     | Full name of user                     |
| Age      | Reported age                          |
| Joined   | Whether the user joined the community |

<Tip>
Tables follow the official [markdown syntax](https://www.markdownguide.org/extended-syntax/#tables).
</Tip>
