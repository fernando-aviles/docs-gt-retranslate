## Lists

### Ordered List
Add line items with numbers followed by periods:

```md
1. First item
2. Second item
3. Third item
```

### Unordered List
Add dashes (`-`), asterisks (`*`), or plus signs (`+`) in front of line items:

```md
- First item
- Second item
- Third item
```

### Nested List
Indent list items to nest them:

```md
- First item
- Second item
  - Nested item
  - Nested item
- Third item
```

## Tables
Create tables using pipes (`|`) to separate columns and hyphens (`---`) for headers:

```md
| Property | Description          |
| -------- | -------------------- |
| Name     | Full name of user    |
| Age      | Reported age         |
```

<Tip>
Lists and tables follow the official [markdown syntax](https://www.markdownguide.org/basic-syntax/#lists-1).
</Tip>
