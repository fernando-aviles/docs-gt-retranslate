## Documentation Hosting

Mintlify provides enterprise-grade documentation hosting with:

<Steps>
  <Step title="Automatic Deployments">
    Changes are automatically deployed when you push to your repository, with preview deployments for pull requests.
  </Step>
  <Step title="Custom Domains">
    Host your documentation on your own domain or subdomain with SSL certificates included.
  </Step>
  <Step title="Global CDN">
    Documentation served through a global CDN for fast loading times worldwide.
  </Step>
  <Step title="Version Control">
    Built-in versioning system to maintain multiple versions of your documentation.
  </Step>
</Steps>
