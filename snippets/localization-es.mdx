# Ejemplo de Localización en Español

Para configurar el contenido en español, siga estos pasos:

1. Configure la versión en español en su archivo `mint.json`:

```json
"versions": [
  {
    "name": "English",
    "locale": "en"
  },
  {
    "name": "Español",
    "locale": "es"
  }
]
```

2. Cree una carpeta para su contenido en español (por ejemplo, `es`) y coloque allí sus archivos MDX traducidos.

3. Especifique la versión en español en su navegación:

```json
"navigation": [
  {
    "group": "Comenzar",
    "version": "es",
    "pages": ["introduccion", "guia-rapida"]
  }
]
```

Mintlify automáticamente traducirá los elementos de la interfaz de usuario como "¿Te resultó útil esta página?" al español.
