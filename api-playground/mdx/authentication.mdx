## Authentication Methods

Authentication can be configured either globally in your `mint.json` file or on a per-page basis through page metadata. Page-level settings will override the global configuration.

### Available Methods

<CodeGroup>

```json Bearer Token (mint.json)
"api": {
    "auth": {
        "method": "bearer"
    }
}
```

```json Basic Auth (mint.json)
"api": {
    "auth": {
        "method": "basic"
    }
}
```

```json API Key (mint.json)
"api": {
    "auth": {
        "method": "key",
        "name": "x-api-key"
    }
}
```

</CodeGroup>

To set authentication at the page level, add the `authMethod` property to your page's front matter:

```md Page Metadata
---
title: "Your page title"
authMethod: "bearer" | "basic" | "key" | "none"
---
```

Note: Use `authMethod: "none"` to disable authentication for specific pages when a global auth method is set.
