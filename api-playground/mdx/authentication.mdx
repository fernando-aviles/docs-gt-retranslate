## Authentication Methods

You can configure authentication either globally in `mint.json` or per-page in the page metadata. Page-level settings override global settings.

<CodeGroup>

```json Bearer Token
"api": {
    "auth": {
        "method": "bearer"
    }
}
```

```json Basic Auth
"api": {
    "auth": {
        "method": "basic"
    }
}
```

```json API Key
"api": {
    "auth": {
        "method": "key",
        "name": "x-api-key"
    }
}
```

```md Disable Authentication
---
authMethod: "none"
---
```

</CodeGroup>

To set authentication for a specific page, add `authMethod: "[method]"` to the page's front matter.
