---
title: 'MDX Setup'
description: 'Generate docs pages for your API endpoints using MDX'
---

Mintlify generates an API playground with request and response examples from your endpoint definitions. Here's how to set it up:

<Steps>
  <Step title="Configure your API">
    In `mint.json`, set your base URL and auth method:

    ```json
    {
      "api": {
        "baseUrl": "https://mintlify.com/api",
        "auth": {
          "method": "bearer"
        }
      }
    }
    ```

    To hide the playground, use:
    ```json
    {
      "api": {
        "playground": {
          "mode": "hide"
        }
      }
    }
    ```

    See all API configurations [here](/settings/global#api-configurations).
  </Step>
  
  <Step title="Create endpoint pages">
    For each endpoint, create an MDX file with:

    ```md
    ---
    title: 'Create new user'
    api: 'POST https://api.mintlify.com/user'
    ---
    ```

    For path parameters, use `{paramName}` syntax: `https://api.example.com/v1/endpoint/{userId}`

    <Note>
    With configured `baseUrl`, you can use relative paths like `/v1/endpoint`.
    </Note>
  </Step>

  <Step title="Add to navigation">
    Add your endpoint pages to the `navigation` field in `mint.json`. [Learn more](/settings/navigation).
  </Step>
</Steps>
