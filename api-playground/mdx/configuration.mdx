---
title: 'MDX Setup'
description: 'Generate docs pages for your API endpoints using MDX'
---

Mintlify generates API playgrounds and examples from endpoints defined through `mint.json` configuration, MDX metadata, and the `<ParamFields />` component.

<Steps>
  <Step title="Configure your API">
    In `mint.json`, set your base URL and auth method:

    ```json
    {
      "api": {
        "baseUrl": "https://mintlify.com/api",
        "auth": {
          "method": "bearer"
        }
      }
    }
    ```

    To hide the API playground:

    ```json
    {
      "api": {
        "playground": {
          "mode": "hide"
        }
      }
    }
    ```

    See all API configurations [here](/settings/global#api-configurations).
  </Step>
  
  <Step title="Create endpoint pages">
    Define endpoints in MDX files with metadata:

    ```md
    ---
    title: 'Create new user'
    api: 'POST https://api.mintlify.com/user'
    ---
    ```

    Use `{parameterName}` for path parameters: `https://api.example.com/v1/endpoint/{userId}`

    <Note>
    With configured `baseUrl`, you can use relative paths like `/v1/endpoint`.
    </Note>
  </Step>

  <Step title="Add to navigation">
    Add endpoint pages to your sidebar in `mint.json`'s `navigation` field. See [navigation docs](/settings/navigation).
  </Step>
</Steps>
