---
title: Caixas de Destaque
description: Use destaques para adicionar contexto chamativo ao seu conteúdo
icon: circle-exclamation
---

### Destaques de Nota

<Note>Isto adiciona uma nota no conteúdo</Note>

```jsx
<Note>This adds a note in the content</Note>
```

### Destaques de Aviso

<Warning>Isto levanta um aviso para ficar atento</Warning>

```jsx
<Warning>This raises a warning to watch out for</Warning>
```

### Destaques de Informação

<Info>Isto chama atenção para informações importantes</Info>

```jsx
<Info>This draws attention to important information</Info>
```

### Destaques de Dica

<Tip>Isto sugere uma dica útil</Tip>

```jsx
<Tip>This suggests a helpful tip</Tip>
```

### Destaques de Verificação

<Check>Isto nos traz um status verificado</Check>

```jsx
<Check>This brings us a checked status</Check>
```
