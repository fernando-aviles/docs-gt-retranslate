Simplesmente adicione`<AccordionGroup>`ao redor dos seus`<Accordion>`componentes existentes.

<AccordionGroup>
  <Accordion title="FAQ without Icon">
    Você pode colocar outros componentes dentro dos Acordeões.

    ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
  </Accordion>

  <Accordion title="FAQ with Icon" icon="alien-8bit">
    Confira a[Acordeão](/content/components/accordions)documentação para todas as props suportadas.
  </Accordion>

  <Accordion title="FAQ without Icon">
    Confira a[Acordeão](/content/components/accordions)documentação para todas as props suportadas.
  </Accordion>
</AccordionGroup>
