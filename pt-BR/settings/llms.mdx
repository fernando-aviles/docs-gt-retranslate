Todas as páginas de documentação são auto-hospedadas em uma variedade de formatos markdown para facilitar:

* Usuários finais alimentarem seus documentos em modelos de linguagem grandes (LLMs) para recuperação específica de informações
* LLMs indexarem seu conteúdo eficientemente sem analisar HTML ou JavaScript

## Formatos suportados

Suportamos padrões da indústria como [/llms.txt](https://llmstxt.org) assim como formatos markdown adicionais.

* **/llms.txt**: Um arquivo markdown de um índice estruturado de seus documentos é automaticamente hospedado em /llms.txt. Ajuda LLMs de propósito geral a indexar mais eficientemente (por exemplo, um sitemap para IA).
* **/llms-full.txt**: Um arquivo markdown de todo o conteúdo dos seus documentos é automaticamente hospedado em /llms-full.txt. Ajuda os usuários a carregar o contexto completo em ferramentas de IA em um único link.
* **suporte .md**: Adicione .md a uma página individual de documentação para ver uma versão markdown dessa página. Ajuda a carregar páginas individuais em ferramentas de IA com melhor eficiência de tokens e tempo de inferência.
* **atalho ⌘ + c**: Copie o código-fonte markdown de qualquer página usando command + c.
