A plataforma gera automaticamente a maioria das meta tags.

No entanto, você pode personalizá-las completamente adicionando o campo [metadata](settings#param-metadata) ao seu `docs.json` ou ao frontmatter da página.

## Meta tags globais

Você pode definir meta tags padrão adicionando o campo `metadata` ao seu `docs.json`.

```json
"seo": {
    "metatags": {
        "og:image": "link to your default meta tag image"
    }
}
```

## Meta tags específicas da página

Adicione suas meta tags aos metadados da página desta forma:

```md
---
title: 'Your Example Page Title'
'og:image': 'link to your meta tag image'
---
```

<Warning>
  Meta tags com dois pontos precisam ser digitadas entre aspas quando você as coloca nos metadados da página.
</Warning>

## Sitemaps

O Mintlify gera automaticamente um arquivo `sitemap.xml` e um arquivo `robots.txt`. Simplesmente navegue para `/sitemap.xml` para ver seu sitemap.

Apenas as páginas incluídas no seu `docs.json` são incluídas por padrão. Para incluir links ocultos, adicione `seo.indexing` ao seu `docs.json`:

```json
"seo": {
    "indexing": all
}
```

## Desativando a indexação

Se você quiser impedir que uma página seja indexada pelos mecanismos de busca, você pode incluir o seguinte no [frontmatter](pages) da sua página:

```
---
noindex: true
---
```

Você também pode especificar `noindex` para todas as páginas em sua documentação definindo o campo `metadata.robots` como `"noindex"` no seu `docs.json`:

```json
"seo": {
    "metatags": {
      "robots": "noindex"
    }
  }
```
