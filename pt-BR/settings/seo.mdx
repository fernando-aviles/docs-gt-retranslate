## Meta tags padrão

Você pode definir meta tags padrão adicionando o campo`metadata` ao seu`docs.json`.

```json
"seo": {
    "metatags": {
        "og:image": "link to your default meta tag image"
    }
}
```

## Alterando meta tags em uma página

Adicione suas meta tags aos metadados da página desta forma:

```md
---
title: 'Your Example Page Title'
'og:image': 'link to your meta tag image'
---
```

<Warning>
  Meta tags com dois pontos precisam ser digitadas entre aspas quando você as coloca nos metadados da página.
</Warning>

## Sitemaps

O Mintlify gera automaticamente um`sitemap.xml` arquivo e um`robots.txt` arquivo. Simplesmente adicione`/sitemap.xml` ao final da URL do seu site para ver seu sitemap. Recomendamos enviar o sitemap para os mecanismos de busca para melhorar o SEO.

Apenas as páginas incluídas no seu`docs.json` são incluídas por padrão. Para incluir links ocultos, adicione`seo.indexing` ao seu`docs.json`:

```json
"seo": {
    "indexing": all
}
```

## Desativando indexação

Se você quiser impedir que uma página seja indexada pelos mecanismos de busca, você pode incluir o seguinte no[frontmatter](/page) da sua página:

```
---
noindex: true
---
```

Você também pode especificar`noindex` para todas as páginas em seus docs definindo o campo`metadata.robots` como`"noindex"` no seu`docs.json`:

```json
"seo": {
    "metatags": {
      "robots": "noindex"
    }
  }
```
