O Mintlify se integra com a API do GitHub, permitindo a sincronização entre sua
documentação e seu repositório GitHub. Esta integração é possível através da
utilização de
[Apps do GitHub](https://docs.github.com/en/developers/apps/getting-started-with-apps/about-apps#about-github-apps).

## Instalação

Para começar, você precisa instalar o App do GitHub do Mintlify na conta do GitHub
onde seu repositório de documentação está localizado. Instalar um App do GitHub requer
propriedade da organização ou permissões de administrador em um repositório. Se você não tiver as
permissões necessárias, o proprietário do repositório precisará aprovar a solicitação.
Você pode acessar a página de instalação fazendo login no
[painel do Mintlify](https://dashboard.mintlify.com).

Para Apps do GitHub, você pode escolher dar permissões apenas a um único repositório.
Recomendamos fortemente que você faça isso, pois só precisamos de acesso ao repositório onde
sua documentação está hospedada.

## Permissões

Durante a instalação do nosso app do GitHub, você será solicitado a conceder certas
permissões:

* Acesso de leitura e escrita para `checks`, `contents`, `deployments`, `pull requests`,
  e `workflows`

Essas permissões são aproveitadas para fornecer uma experiência perfeita ao gerenciar
sua documentação.

Quando você faz um commit no branch que configurou como seu branch de implantação
da documentação, buscamos o conteúdo dos arquivos alterados para atualizar sua documentação. Para
realizar isso, precisamos de acesso de leitura ao seu `contents`.

Quando pull requests são criados, nós criamos uma verificação e uma implantação de preview, é por isso que precisamos de acesso de escrita para`checks` e `deployments`.

Dentro do editor web do Mintlify, o Mintlify trabalha em seu nome para criar branches e pull requests, é por isso que precisamos de acesso de escrita para`pull requests` e
`workflows`.

<Info>
  Se você está preocupado com as permissões de escrita, o GitHub App só terá acesso aos repositórios que você permitir*e* se você tiver proteções de branch*não pode* fazer push diretamente para seus branches - ele segue suas regras de proteção de branch.
</Info>

## Instalações de Repositórios

Ao instalar nosso GitHub app, você será solicitado a selecionar todos os repositórios ou um subconjunto deles. Esta seleção pode ser alterada a qualquer momento indo para as
[configurações do GitHub app](https://github.com/apps/mintlify/installations/new).
