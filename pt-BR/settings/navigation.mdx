A `navigation` propriedade controla a hierarquia da sua documentação.

## Páginas

Se você não quiser nenhuma hierarquia, você pode simplesmente definir páginas dentro do seu campo `navigation`.

Cada entrada do array `pages` deve ser um caminho para um arquivo que existe dentro do seu repositório.
Note que você não precisa anexar `.mdx` aos caminhos dos arquivos.

```json
{
  "navigation": {
    "pages": [
      "overview",
      "quickstart",
      "advanced/components",
      "advanced/integrations"
    ]
  }
}
```

## Grupos

Grupos permitem que você agrupe suas páginas. Grupos também podem ser aninhados uns dentro dos outros.

```json
{
  "navigation": {
    "groups": [
      {
        "group": "Getting Started",
        "pages": [
          "quickstart",
          {
            "group": "Editing",
            "icon": "pen-paintbrush",
            "pages": ["development", "web-editor"]
          }
        ]
      },
      {
        "group": "Writing Content",
        "pages": ["writing-content/page", "writing-content/text"]
      }
    ]
  }
}
```

## Abas

Abas ajudam a distinguir entre diferentes tópicos ou seções da sua documentação.

<Frame>
  <img className="block dark:hidden rounded-md" src="/images/tabs-light.png" />

  <img className="hidden dark:block rounded-md" src="/images/tabs-dark.png" />
</Frame>

```json docs.json
"navigation": {
  "tabs": [
    {
      "tab": "API References",
      "pages": [
        "api-reference/get",
        "api-reference/post",
        "api-reference/delete"
      ]
    },
    {
      "tab": "SDKs",
      "pages": [
        "sdk/fetch",
        "sdk/create",
        "sdk/delete",
      ]
    },
    {
      "tab": "Blog",
      "href": "https://external-link.com/blog"
    }
  ]
}
```

## Âncoras

Âncoras são outra maneira de seccionar seu conteúdo.

<Frame>
  <img className="block dark:hidden rounded-md" src="/images/anchor-light.png" />

  <img className="hidden dark:block rounded-md" src="/images/anchor-dark.png" />
</Frame>

A configuração é muito similar à configuração de abas.

```json docs.json
"navigation": {
  "anchors": [
    {
      "anchor": "API References",
      "pages": [
        "api-reference/get",
        "api-reference/post",
        "api-reference/delete"
      ]
    },
    {
      "anchor": "SDKs",
      "pages": [
        "sdk/fetch",
        "sdk/create",
        "sdk/delete",
      ]
    },
    {
      "anchor": "Blog",
      "href": "https://external-link.com/blog"
    }
  ]
}
```

## Hierarquia Aninhada

Você pode usar tanto âncoras quanto abas juntas - qualquer uma pode ser aninhada dentro da outra de forma intercambiável.

## Links Globais

Se você quiser usar abas ou âncoras apenas com o propósito de listar links externos, use a propriedade `navigation.global`.

```json
"navigation": {
  "global": {
    "anchors": [
      {
        "anchor": "Slack",
        "icon": "slack",
        "href": "https://slack.com
      },
      {
        "anchor": "API Reference",
        "icon": "code",
        "href": "https://api-reference.com"
      }
    ]
  }
}
```

### Páginas Ocultas

Arquivos MDX não incluídos em `docs.json` não aparecerão na barra lateral mas são acessíveis através de links diretos.

Páginas ocultas não são indexadas para busca dentro da sua documentação por padrão. Se você quiser sobrescrever este comportamento, você pode definir o atributo [`seo.indexing`](/settings/global#param-indexing) no seu `docs.json` como `navigable`.

## Barra de Navegação

### Links

Adicione links à barra superior com o campo `links` no arquivo `docs.json`.

<Frame>
  <img className="block rounded-md" src="/images/topbar-links.png" />
</Frame>

O campo `links` suporta os seguintes campos: `label`, `href`.

```json
"navbar": {
  "links": [
    {
      "label": "Community",
      "href": "https://mintlify.com/community"
    }
  ]
}
```

### Botão Primário

Personalize o botão primário na barra superior usando o campo `primary`
.

<Frame>
  <img className="block rounded-md" src="/images/topbar-cta-get-started.png" />
</Frame>

O campo `primary` suporta os seguintes campos: `label`, `href`, `type`. Para mais informações sobre as opções para estes campos, veja o [esquema docs.json](/settings/global#param-navbar).

```json
"navbar": {
  "primary": {
    "type": "button",
    "label": "Get Started",
    "href": "https://mintlify.com/get-started"
  }
}
```

#### GitHub

Você também pode configurar o botão CTA para linkar diretamente ao seu repositório GitHub. Use o campo `primary` com o `type` definido como `github`.

<Frame>
  <img className="block rounded-md" src="/images/topbar-cta-github.png" />
</Frame>

```json
"navbar": {
  "primary": {
    "type": "github",
    "label": "GitHub",
    "href": "https://github.com/mintlify/docs"
  }
}
```
