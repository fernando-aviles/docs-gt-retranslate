## Configuração do vercel.json

Para hospedar sua documentação em um subcaminho personalizado usando Vercel, você precisa adicionar a
seguinte configuração ao seu `vercel.json` arquivo.

```json
{
  "rewrites": [
    {
      "source": "/docs",
      "destination": "https://[subdomain].mintlify.dev/docs"
    },
    {
      "source": "/docs/:match*",
      "destination": "https://[subdomain].mintlify.dev/docs/:match*"
    }
  ]
}
```

<Note>
  Para mais informações, você também pode consultar o guia oficial da Vercel sobre
  redirecionamentos: [Project Configuration:
  Rewrites](https://vercel.com/docs/projects/project-configuration#rewrites)
</Note>
