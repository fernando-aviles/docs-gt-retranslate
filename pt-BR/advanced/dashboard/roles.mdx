---
- title: Funções
- description: Controle o acesso ao seu painel com funções.
---

O Mintlify fornece dois níveis de acesso ao painel: Editor e Admin.

A seguir estão descritas as ações que são limitadas à função de Admin:

|                                   | Editor | Admin |
| --------------------------------- | :----: | :---: |
| Atualizar funções de usuário      |    ❌   |   ✅   |
| Excluir usuários                  |    ❌   |   ✅   |
| Convidar usuários admin           |    ❌   |   ✅   |
| Gerenciar e atualizar faturamento |    ❌   |   ✅   |
| Atualizar domínio personalizado   |    ❌   |   ✅   |
| Atualizar fonte Git               |    ❌   |   ✅   |
| Excluir organização               |    ❌   |   ✅   |

Outras ações no painel estão disponíveis para ambas as funções.

Você pode convidar quantos administradores quiser, mas recomendamos limitar o acesso de administrador aos usuários que precisam dele.
