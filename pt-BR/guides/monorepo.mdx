---
- title: Configuração de monorepo
- description: Implante seus documentos de um repositório que contém múltiplos projetos
- icon: folder-git
---

Configure o Mintlify para implantar documentação de um diretório específico dentro de um monorepo. Esta configuração permite que você mantenha a documentação junto com seu código em repositórios que contêm múltiplos projetos ou serviços.

## Pré-requisitos

* Acesso de administrador ao seu projeto Mintlify.
* Arquivos de documentação organizados em um diretório dedicado dentro do seu monorepo.
* Um`docs.json`válido no seu diretório de documentação.

## Configurar implantação de monorepo

<Steps>
  <Step title="Acessar configurações do Git">
    Navegue até[Configurações do Git](https://dashboard.mintlify.com/settings/deployment/git-settings)no seu painel.

    <Frame>
      <img className="block dark:hidden my-0 pointer-events-none" src="/images/monorepo-light.png" alt="The project settings panel in the Git Settings menu. The Set up as monorepo toggle button is enabled and a path to the /docs directory is specified." />

      <img className="hidden dark:block my-0 pointer-events-none" src="/images/monorepo-dark.png" alt="The project settings panel in the Git Settings menu. The Set up as monorepo toggle button is enabled and a path to the /docs directory is specified." />
    </Frame>
  </Step>

  <Step title="Defina o caminho da sua documentação">
    1. Selecione o botão de alternância**Configurar como monorepo**.
    2. Insira o caminho relativo para o seu diretório de documentação.
    3. Selecione**Salvar alterações**.
  </Step>
</Steps>
