<Note>
  Se você está atualmente usando o legado `mint.json` arquivo de configuração, por favor atualize o CLI:

  ```sh
  npm i -g mintlify@latest
  ```

  E execute o novo `upgrade` comando no seu repositório de documentação:

  ```sh
  mintlify upgrade
  ```

  Isso irá gerar um `docs.json` baseado no seu `mint.json`. Então, por favor delete o `mint.json` arquivo do seu repositório.
</Note>
