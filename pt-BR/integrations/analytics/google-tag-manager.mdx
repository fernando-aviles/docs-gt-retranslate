---
- title: Google Tag Manager
---

Adicione seu ID de tag ao `docs.json` arquivo e nós injetaremos o script do Google Tag Manager em todas as suas páginas.

Você é responsável por configurar banners de consentimento de cookies com o Google Tag Manager, se necessário.

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "gtm": {
          "tagId": "required"
      }
  }
  ```

  ```json Example
  "integrations": {
      "gtm": {
          "tagId": "GTM-MGBL4PW"
      }
  }
  ```
</CodeGroup>
