---
- title: Fathom
---

Adicione o seguinte ao seu arquivo `docs.json` para enviar análises para o Fathom.

Você pode obter o `siteId` nas configurações do seu script.

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "fathom": {
          "siteId": "required"
      }
  }
  ```

  ```json Example
  "integrations": {
      "fathom": {
          "siteId": "YSVMSDAY"
      }
  }
  ```
</CodeGroup>
