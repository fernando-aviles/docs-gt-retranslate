---
- title: Plausible
---

Adicione o domínio do seu site a `docs.json` para enviar análises para o Plausible.

<Info>
  Não inclua `https://` para o domínio ou servidor.
</Info>

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "plausible": {
          "domain": "required",
          "server": "optional"
      }
  }
  ```

  ```json Example
  "integrations": {
      "plausible": {
          "domain": "docs.domain.com"
      }
  }
  ```
</CodeGroup>
