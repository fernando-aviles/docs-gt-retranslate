- title: Front
---

Adicione o seguinte ao seu arquivo `docs.json` para adicionar um widget do [Front Chat](https://front.com).

<CodeGroup>
  ```json Integration options in docs.json
  "integrations": {
      "frontchat": "CHAT_ID"
  }
  ```

  ```json Example
  "integrations": {
      "frontchat": "1365d046d7c023e9b030ce90d02d093a"
  }
  ```
</CodeGroup>
