---
- title: Intercom
---

Adicione o seguinte ao seu arquivo `docs.json` para adicionar um widget do [Intercom](https://www.intercom.com).

<CodeGroup>
  ```json Integration options in docs.json
  "integrations": {
        "intercom": {
              "appId": "APP_ID"
        }
  }
  ```

  ```json Example
  "integrations": {
        "intercom": {
              "appId": "APP_ID"
        }
  }
  ```
</CodeGroup>
