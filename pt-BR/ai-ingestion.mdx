Mintlify gera formatos otimizados e fornece atalhos que ajudam os usuários a obter respostas mais rápidas e precisas ao usar sua documentação como contexto para LLMs e ferramentas de IA.

## Menu contextual

Forneça acesso rápido a conteúdo otimizado para IA e integrações diretas com ferramentas populares de IA a partir de um menu contextual em suas páginas.

* **Copiar página**: Copia a página atual como Markdown para colar como contexto em ferramentas de IA.
* **Ver como Markdown**: Abre a página atual como Markdown.
* **Abrir no ChatGPT**: Cria uma conversa no ChatGPT com a página atual como contexto.
* **Abrir no Claude**: Cria uma conversa no Claude com a página atual como contexto.

### Habilitando o menu contextual

Adicione o campo `contextual` ao seu `docs.json` e especifique quais opções você deseja incluir em seu menu.

```json
{
 "contextual": {
   "options": [
     "copy",
     "view",
     "chatgpt",
     "claude"
   ]
 }
}
```

## /llms.txt

O arquivo [/llms.txt](https://llmstxt.org) é um padrão da indústria que ajuda LLMs de propósito geral a indexar de forma mais eficiente, semelhante a como um sitemap ajuda os mecanismos de busca.

Cada site de documentação automaticamente hospeda um arquivo `/llms.txt` na raiz que lista todas as páginas disponíveis em sua documentação. Ferramentas de IA podem usar este arquivo para entender a estrutura da sua documentação e encontrar conteúdo relevante para as solicitações dos usuários.

<PreviewButton href="https://mintlify.com/docs/llms.txt">Abrir llms.txt para este site</PreviewButton>

## /llms-full.txt

O arquivo `/llms-full.txt` combina todo o seu site de documentação em um único arquivo como contexto para ferramentas de IA.

Cada site de documentação automaticamente hospeda um arquivo `/llms-full.txt` na raiz.

<PreviewButton href="https://mintlify.com/docs/llms-full.txt">Abrir llms-full.txt para este site</PreviewButton>

## Gerando versões Markdown das páginas

Markdown fornece texto estruturado que as ferramentas de IA podem processar de forma mais eficiente do que HTML, resultando em melhores tempos de resposta e menor uso de tokens.

### .md extension

Adicione uma `.md` à URL de uma página para exibir uma versão Markdown dessa página.

<PreviewButton href="https://mintlify.com/docs/quickstart.md">Abrir quickstart.md</PreviewButton>

### Atalho Command + C

Selecione Command + C (Ctrl + C no Windows) para copiar qualquer página como Markdown.
