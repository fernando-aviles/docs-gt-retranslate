<Update label="Fevereiro 2025">
  ## Novo esquema de configuração `docs.json`

  <Frame>
    ![docs.json screenshot](/images/changelog/docsjson.png)
  </Frame>

  Introduzimos um novo `docs.json` esquema como substituto para `mint.json`, para suportar melhor o versionamento multinível, compreensão visual mais fácil e terminologia mais consistente. Para mais informações sobre o que mudou, [confira nosso blog](https://mintlify.com/blog/refactoring-mint-json-into-docs-json).

  Atualize de `mint.json` para `docs.json` com os seguintes passos:

  1. Certifique-se de que seu CLI está na versão mais recente

  ```
  npm i mint@latest -g
  ```

  1. No repositório de seus documentos, execute

  ```
  mint upgrade
  ```

  1. Exclua seu arquivo mint.json antigo e envie suas alterações

  ```javascript Line Highlighting Example {1,3-5}
  const greeting = "Hello, World!";

  }
  sayHello();
  ```

