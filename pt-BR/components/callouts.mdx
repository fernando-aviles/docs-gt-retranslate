---
- title: Destaques
- description: Use destaques para adicionar contexto chamativo ao seu conteúdo
- icon: info
---

Os destaques podem ser estilizados como Nota, Aviso, Informação, Dica ou Verificação:

<Note>Isso adiciona uma nota no conteúdo</Note>

```jsx
<Note>This adds a note in the content</Note>
```

<Warning>Isso levanta um aviso para ficar atento</Warning>

```jsx
<Warning>This raises a warning to watch out for</Warning>
```

<Info>Isso chama a atenção para informações importantes</Info>

```jsx
<Info>This draws attention to important information</Info>
```

<Tip>Isso sugere uma dica útil</Tip>

```jsx
<Tip>This suggests a helpful tip</Tip>
```

<Check>Isso nos traz um status verificado</Check>

```jsx
<Check>This brings us a checked status</Check>
```

  ```jsx Callout Example
  <Note>This adds a note in the content</Note>
  ```
</RequestExample>
