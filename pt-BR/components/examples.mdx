Os componentes `<RequestExample>` e `<ResponseExample>` fixam blocos de código no canto superior direito de uma página mesmo enquanto você rola. Os componentes funcionam em todas as páginas, mesmo que você não use um playground de API.

`<RequestExample>` e `<ResponseExample>` aparecem como blocos de código regulares em dispositivos móveis.

## Exemplo de Requisição

O componente `<RequestExample>` funciona de maneira similar ao [CodeGroup](/pt-BR/components/code-groups), mas exibe o conteúdo da requisição na barra lateral direita. Assim, você pode colocar vários blocos de código dentro de `<RequestExample>`.

Por favor, defina um nome em cada bloco de código que você colocar dentro de RequestExample.

<RequestExample>
  ````md RequestExample Example
  <RequestExample>

  ```bash Request
    curl --request POST \
      --url https://dog-api.kinduff.com/api/facts
  ```

  </RequestExample>
  ````
</RequestExample>

## Exemplo de Resposta

O componente `<ResponseExample>` é o mesmo que `<RequestExample>` mas aparecerá abaixo dele.

<ResponseExample>
  ````md ResponseExample Example
  <ResponseExample>

  ```json Response
  { "status": "success" }
  ```

  </ResponseExample>
  ````
</ResponseExample>
