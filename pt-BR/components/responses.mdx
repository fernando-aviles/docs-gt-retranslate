---
- title: Campos de Resposta
- description: Exibir valores de resposta da API
---

O componente`<ResponseField>`é projetado para definir os valores de retorno de uma API. Muitos documentos também usam`<ResponseField>`em páginas quando você precisa listar os tipos de algo.

<ResponseField name="response" type="string" required>
  Um exemplo de campo de resposta
</ResponseField>

```jsx
<ResponseField name="response" type="string" required>
  A response field example
</ResponseField>
```

## Propriedades

<ResponseField name="name" type="string" required>
  O nome do valor de resposta.
</ResponseField>

<ResponseField name="type" type="string" required>
  Tipo esperado do valor de resposta - isso pode ser qualquer string arbitrária.
</ResponseField>

<ResponseField name="default" type="string">
  O valor padrão.
</ResponseField>

<ResponseField name="required" type="boolean">
  Mostra "obrigatório" ao lado do nome do campo.
</ResponseField>

<ResponseField name="deprecated" type="boolean">
  Se um campo está obsoleto ou não.
</ResponseField>

<ResponseField name="pre" type="string[]">
  Rótulos que são mostrados antes do nome do campo
</ResponseField>

<ResponseField name="post" type="string[]">
  Rótulos que são mostrados após o nome do campo
</ResponseField>
