Você pode adicionar qualquer número de abas e outros componentes dentro das abas.

<Tabs>
  <Tab title="Primeira Aba">
    ☝️ Bem-vindo ao conteúdo que você só pode ver dentro da primeira Aba.
    Você pode adicionar qualquer número de componentes dentro das abas.

    ```java HelloWorld.java
      class HelloWorld {
          public static void main(String[] args) {
              System.out.println("Hello, World!");
          }
      }
    ```
  </Tab>

  <Tab title="Segunda Aba">
    ✌️ Aqui está o conteúdo que está apenas dentro da segunda Aba.
  </Tab>

  <Tab title="Terceira Aba">
    💪 Aqui está o conteúdo que está apenas dentro da terceira Aba.
  </Tab>
</Tabs>

<RequestExample>
  ````jsx Tabs Example
  <Tabs>
    <Tab title="First Tab">
      ☝️ Welcome to the content that you can only see inside the first Tab.
      ```java HelloWorld.java
        class HelloWorld {
            public static void main(String[] args) {
                System.out.println("Hello, World!");
            }
        }
      ```
    </Tab>
    <Tab title="Second Tab">
      ✌️ Here's content that's only inside the second Tab.
    </Tab>
    <Tab title="Third Tab">
      💪 Here's content that's only inside the third Tab.
    </Tab>
  </Tabs>
  ````
</RequestExample>

## Propriedades da Aba

<ResponseField name="title" type="string" required>
  O título da aba. Títulos curtos são mais fáceis de navegar.
</ResponseField>
