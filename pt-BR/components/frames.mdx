---
- title: Frames
- description: Use o componente Frame para envolver imagens ou outros componentes
    em um contêiner.
- icon: frame
---

Frames são muito úteis se você quiser centralizar uma imagem.

<Frame>
  <img src="https://mintlify-assets.b-cdn.net/yellowstone.jpeg" />
</Frame>

## Legendas

Você pode adicionar contexto adicional a uma imagem usando a prop opcional `caption`.

<Frame caption="O Parque Nacional de Yosemite é visitado por mais de 3,5 milhões de pessoas todos os anos">
  <img src="https://mintlify-assets.b-cdn.net/yosemite.jpg" />
</Frame>

## Props

<ResponseField name="caption" type="string">
  Texto de legenda opcional para mostrar centralizado abaixo do seu componente.
</ResponseField>

<RequestExample>
  ```jsx Frame
  <Frame>
    <img src="/path/image.jpg" />
  </Frame>
  ```

  ```jsx Frame with Captions
  <Frame caption="Caption Text">
    <img src="/path/image.jpg" />
  </Frame>
  ```
</RequestExample>
