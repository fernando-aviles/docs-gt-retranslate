---
- title: Grupos de Código
- description: O componente CodeGroup permite combinar blocos de código em uma
    exibição separada por abas
- icon: group
---

Você precisará fazer [Blocos de Código](/pt-BR/code) e então adicionar o componente `<CodeGroup>` ao redor deles. Cada Bloco de Código deve ter um nome de arquivo porque usamos os nomes para os botões das abas.

Veja abaixo um exemplo do resultado final.

<CodeGroup>
  ```javascript helloWorld.js
  console.log("Hello World");
  ```

  ```python hello_world.py
  print('Hello World!')
  ```

  ```java HelloWorld.java
  class HelloWorld {
      public static void main(String[] args) {
          System.out.println("Hello, World!");
      }
  }
  ```
</CodeGroup>
