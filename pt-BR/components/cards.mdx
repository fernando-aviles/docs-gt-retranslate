## Cartão de link

Você pode personalizar o CTA e se deseja ou não exibir a seta no cartão. Por padrão, a seta só será exibida para links externos.

<Card title="Cartão de link" icon="link" href="/pt-BR/components/columns" arrow="true" cta="Clique aqui">
  É assim que você usa um cartão com um ícone e um link. Clicar neste cartão
  leva você à página Columns.
</Card>

<RequestExample>
  ```jsx Card Example
  <Card
    title="Link card"
    icon="link"
    href="/components/columns"
    arrow="true"
    cta="Click here"
  >
    This is how you use a card with an icon and a link. Clicking on this card
    brings you to the Columns page.
  </Card>
  ```
</RequestExample>

## Agrupando cartões

Você pode agrupar cartões em [columns](/pt-BR/components/columns).

<Columns cols={2}>
  <Card title="Primeiro Cartão" icon="panel-left-close">
    Este é o primeiro cartão.
  </Card>

  <Card title="Segundo Cartão" icon="panel-right-close">
    Este é o segundo cartão.
  </Card>
</Columns>

## Props

<ResponseField name="title" type="string" required>
  O título do cartão
</ResponseField>

<ResponseField name="icon" type="string or svg">
  Um ícone [Font Awesome icon](https://fontawesome.com/icons), [Lucide
  icon](https://lucide.dev/icons), ou código SVG em `icon={}`
</ResponseField>

<ResponseField name="iconType" type="string">
  Um dos seguintes: `regular`, `solid`, `light`, `thin`, `sharp-solid`, `duotone`, `brands`
</ResponseField>

<ResponseField name="color" type="string">
  A cor do ícone como um código hexadecimal
</ResponseField>

<ResponseField name="href" type="string">
  A URL para onde o clique no cartão navegará o usuário
</ResponseField>

<ResponseField name="horizontal" type="boolean">
  Torna o cartão mais compacto e horizontal
</ResponseField>

<ResponseField name="img" type="string">
  A URL ou caminho local para uma imagem a ser exibida no topo do cartão
</ResponseField>

<ResponseField name="cta" type="string">
  Rótulo para o botão de ação
</ResponseField>

<ResponseField name="arrow" type="boolean">
  Ativa ou desativa o ícone de seta do link
</ResponseField>
