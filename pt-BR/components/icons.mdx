## Ícones Inline

O ícone será colocado inline quando usado em um parágrafo.

```markdown Inline Icon Example
<Icon icon="flag" iconType="solid" /> The documentation you want, effortlessly 
```

<Icon icon="flag" iconType="solid" /> A documentação que você deseja, sem esforço

### Props

<ResponseField name="icon" type="string" required>
  Um ícone [Font Awesome](https://fontawesome.com/icons) ou [Lucide](https://lucide.dev/icons)
</ResponseField>

<ResponseField name="iconType" type="string">
  Um de `regular`, `solid`, `light`, `thin`, `sharp-solid`, `duotone`, `brands` (apenas para ícones [Font Awesome](https://fontawesome.com/icons)).
</ResponseField>

<ResponseField name="color" type="string">
  A cor do ícone como um código hexadecimal (por exemplo, `#FF5733`)
</ResponseField>

<ResponseField name="size" type="number">
  O tamanho do ícone em pixels
</ResponseField>
