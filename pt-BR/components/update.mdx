O componente`Update`é usado para acompanhar mudanças e atualizações.

<Update label="2024-10-12" description="v0.1.1">
  <Frame>
    <img className="block" src="https://mintlify-assets.b-cdn.net/hero-light.png" />
  </Frame>

  ## Changelog

  Você pode adicionar qualquer coisa aqui, como uma captura de tela, um trecho de código ou uma lista de mudanças.

  #### Recursos

  * Design responsivo
  * Seção fixa para cada changelog
</Update>

<Update label="2024-10-11" description="v0.1.0">
  ### Como usar

  ```md
  <Update label="2024-10-12" description="v0.1.1">
    This is how you use a changelog with a label 
    and a description.
  </Update>
  <Update label="2024-10-11" description="v0.1.0">
    This is how you use a changelog with a label 
    and a description.
  </Update>
  ```

  Você pode usar múltiplos componentes`Update`para criar changelogs.
</Update>

<Tip>
  Cada`label`cria uma âncora e também aparece no sumário à direita.
</Tip>

## Props

<ResponseField name="label" type="string" required>
  O rótulo dado à atualização, aparece como texto fixo à esquerda do changelog.
</ResponseField>

<ResponseField name="tags" type="string[]">
  Tags para o changelog, serão mostradas como filtros no painel do lado direito.
</ResponseField>

<ResponseField name="description" type="string">
  Descrição da atualização, aparece abaixo do rótulo e da tag.
</ResponseField>

</RequestExample>
