## 画像

### Markdownの使用

この[markdown構文](https://www.markdownguide.org/basic-syntax/#images)を使用すると、以下のコードで画像を追加できます

```md
![title](/path/image.jpg)
```

画像ファイルのサイズは5MB未満である必要があることに注意してください。それ以外の場合は、[Cloudinary](https://cloudinary.com/)や[S3](https://aws.amazon.com/s3/)のようなサービスでのホスティングをお勧めします。そのURLを使用して埋め込むことができます。

### 埋め込みの使用

画像をより柔軟にカスタマイズするために、埋め込みを使用して画像を追加することもできます。

```html
<img height="200" src="/path/image.jpg" />
```

### 画像ズームの無効化

画像のデフォルトのクリック時ズームを無効にするには、画像埋め込みにnoZoomプロパティを追加します。

```html
<img height="200" noZoom src="/path/image.jpg" />
```

### 画像のリンク

画像にリンクを設定するには（例えば、ドキュメントにボタンを作成する場合）、画像をリンクで囲み`noZoom`プロパティを設定します。`a`タグ内の画像は自動的にポインターカーソルが表示されます。

```html
<a href="https://mintlify.com" target="_blank">
  <img height="200" noZoom src="/path/image.jpg" />
</a>
```

### ダークモード

ライトモードとダークモードで別々の画像を使用するには、Tailwind CSSを使用して画像の表示/非表示を切り替えます。

```html
<img className="block h-32 dark:hidden" src="/path/image-light.jpg" />
<img className="hidden h-32 dark:block" src="/path/image-dark.jpg" />
```

## 動画

<iframe className="w-full aspect-video rounded-xl" src="https://www.youtube.com/embed/4KzFe50RQkQ" title="YouTube video player" frameBorder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowFullScreen />

<br />

<Tip>
  Mintlifyは[MarkdownでのHTMLタグ](https://www.markdownguide.org/basic-syntax/#html)をサポートしています。これは、HTMLタグをMarkdown構文の代わりに使用したい場合に便利で、無限の柔軟性でドキュメントを作成することができます。
</Tip>

YouTubeの動画には以下を使用します：

```html
<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/4KzFe50RQkQ"
  title="YouTube video player"
  frameborder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowfullscreen
></iframe>
```

その他の動画には以下を使用します：

```html
<video
  controls
  className="w-full aspect-video"
  src="link-to-your-video.com"
></video>
```

動画を自動再生するには以下を使用します：

```html
<video
  autoPlay
  muted
  loop
  playsInline
  className="w-full aspect-video"
  src="link-to-your-video.com"
></video>
```

<Warning>
  MintlifyはJSX構文に準拠する必要があるため、二重語の属性はキャメルケースで記述する必要があります：autoPlay、playsInline。
</Warning>

## iFrame

ドキュメント内に別のHTMLページを読み込みます。

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```

<Tip>
  必須ではありませんが、`alt`と`title`属性を画像に追加することをお勧めします。より良いSEOとアクセシビリティのために。詳細は[画像SEO](https://yoast.com/image-seo-alt-tag-and-title-tag-optimization/)をご覧ください。
</Tip>
