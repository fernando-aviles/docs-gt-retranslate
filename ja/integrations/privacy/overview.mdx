## データプライバシー統合の有効化

ドキュメントにデータプライバシープラットフォームを追加できます。`integrations`フィールドを`docs.json`ファイルに、それぞれのスクリプトと共に追加します。

```json
  "integrations": {
    "osano": "SOURCE"
  }
```

データプライバシープラットフォームの統合をリクエストしたい場合は、[コミュニティ](https://mintlify.com/community)でお知らせください。

## クッキー同意とテレメトリーの無効化

GDPR準拠のためにユーザーがすでにクッキーに同意しているかを確認する必要がある場合、`cookies`でローカルストレージのキーと値を指定できます：

```json
  "integrations": {
    "cookies": {
      "key": "LOCAL STORAGE KEY",
      "value": "LOCAL STORAGE VALUE"
    }
  }
```

これらの値が設定されている場合、ユーザーがクッキーに同意しているかをローカルストレージでチェックします。同意していない場合、テレメトリーは無効化されます。

すべてのユーザーに対してテレメトリーを無効化したい場合は、`docs.json`ファイルに以下を追加できます：

```json
  "integrations": {
    "telemetry": {
      "enabled": false
    }
  }
```
