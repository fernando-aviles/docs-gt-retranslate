---
title: Fathom
---

以下を`docs.json`ファイルに追加してFathomにアナリティクスを送信します。

スクリプト設定から`siteId`を取得できます。

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "fathom": {
          "siteId": "required"
      }
  }
  ```

  ```json Example
  "integrations": {
      "fathom": {
          "siteId": "YSVMSDAY"
      }
  }
  ```
</CodeGroup>
