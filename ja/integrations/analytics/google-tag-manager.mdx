---
title: Google Tag Manager
---

タグIDを`docs.json`ファイルに追加すると、すべてのページにGoogle Tag Managerスクリプトを挿入します。

必要な場合、Google Tag Managerでクッキー同意バナーを設定する責任はあなたにあります。

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "gtm": {
          "tagId": "required"
      }
  }
  ```

  ```json Example
  "integrations": {
      "gtm": {
          "tagId": "GTM-MGBL4PW"
      }
  }
  ```
</CodeGroup>
