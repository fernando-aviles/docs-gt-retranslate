新しい<Tooltip tip="Google Analytics 4">GA4</Tooltip>プロパティを生成する必要があります。収集されたデータは、他のGoogle Analyticsデータと同じプロジェクトに格納されます。

古いバージョンのGoogle Analytics（Universal Analytics）を使用している場合でも、<Tooltip tip="Google Analytics 4">GA4</Tooltip>プロパティを生成することができます。<Tooltip tip="Google Analytics 4">GA4</Tooltip>のデータはUAデータとは若干異なりますが、同じプロジェクトで収集されます。

## MintlifyにGA4を接続する方法

### Webストリームの作成

Mintlifyに入力するための測定IDを取得するには、Webストリームを作成する必要があります。

Google Analytics画面の左下にある歯車アイコンをクリックし、データストリームをクリックします。

<Frame>![](/images/ga4-web-streams.png)</Frame>

Webストリームを作成し、MintlifyドキュメントサイトのURLをストリームURLとして入力します。

測定IDは`G-XXXXXXX`のような形式で、Webストリームを作成した直後にストリームの詳細に表示されます。

### docs.jsonに測定IDを入力

測定IDを`docs.json`ファイルに以下のように追加します：

```json docs.json
"integrations": {
    "ga4": {
        "measurementId": "G-XXXXXXX"
    }
}
```

### 待機

Google Analyticsがデータを表示するまでに2〜3日かかります。

アナリティクスが正しく有効化されているかを確認するには、[Google Analytics Debugger](https://chrome.google.com/webstore/detail/google-analytics-debugger/jnkmfdileelhofjcijamephohjechhna?hl=en)を使用できます。この拡張機能は、GA4がリクエストを行うたびにブラウザのコンソールにログを記録します。

<Note>
  プレビューリンクではアナリティクスは無効になっています。
</Note>
