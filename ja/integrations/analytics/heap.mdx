title: Heap

以下を`docs.json`ファイルに追加してHeapにアナリティクスを送信します。

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "heap": {
          "appId": "required"
      }
  }
  ```

  ```json Example
  "integrations": {
      "heap": {
          "appId": "1234567890"
      }
  }
  ```
</CodeGroup>
