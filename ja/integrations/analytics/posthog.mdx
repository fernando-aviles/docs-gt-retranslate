title: PostHog
---

以下を`docs.json`ファイルに追加してPostHogにアナリティクスを送信します。

PostHogをセルフホストしている場合は`apiHost`のみを含める必要があります。デフォルトでは`https://app.posthog.com`にイベントを送信します。

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "posthog": {
          "apiKey": "YOUR_POSTHOG_PROJECT_API_KEY",
          "apiHost": "optional"
      }
  }
  ```

  ```json Example
  "integrations": {
      "posthog": {
          "apiKey": "phc_TXdpocbYTeZVm5VJmMzHTMrCofBQu3e0kN7HGMNGTVW"
      }
  }
  ```
</CodeGroup>

<br />

<Warning>
  PostHogアナリティクスを有効にすると、Mintlifyダッシュボードのアナリティクスは無効になります。
</Warning>

## セッション記録

セッション記録を受信するには、PostHogのプロジェクト設定にある「録画用の承認済みドメイン」にドキュメントウェブサイトのURLを追加する必要があります。URLを追加するオプションはPostHogのプロジェクト設定にあります。
