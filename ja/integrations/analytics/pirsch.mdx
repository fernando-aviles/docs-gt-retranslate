title: Pirsch
---

以下を`docs.json`ファイルに追加してPirschにアナリティクスを送信します。

サイトIDは設定 > 開発者 > 識別コードから取得できます。

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "pirsch": {
          "id": "required"
      }
  }
  ```

  ```json Example
  "integrations": {
      "pirsch": {
          "id": "8Kw7OKxBfswOjnKGZa7P9Day8JmVYwTp"
      }
  }
  ```
</CodeGroup>
