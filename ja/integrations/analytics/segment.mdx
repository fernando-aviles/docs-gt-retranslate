---
title: Segment
---

Segmentの書き込みキーを`docs.json`ファイルに追加してSegmentにアナリティクスを送信します。

<CodeGroup>
  ```json Analytics options in docs.json
  "integrations": {
      "segment": {
          "key": "required",
      }
  }
  ```

  ```json Example
  "integrations": {
      "segment": {
          "key": "nqJxiRG15Y6M594P8Sb8ESEciU3VC2"
      }
  }
  ```
</CodeGroup>
