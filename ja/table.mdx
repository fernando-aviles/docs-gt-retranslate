| プロパティ | 説明                   |
| ----- | -------------------- |
| 名前    | ユーザーのフルネーム           |
| 年齢    | 報告された年齢              |
| 参加    | ユーザーがコミュニティに参加したかどうか |

### テーブルの作成

<Tip>
  テーブルコンポーネントは公式の[マークダウン構文](https://www.markdownguide.org/extended-syntax/#tables)に従います。
</Tip>

<span className="hover:bg-cyan-500">
  テーブルを追加するには
</span>

、3つ以上のハイフン（`---`）を使用して各列のヘッダーを作成し、パイプ（`|`）を使用して各列を区切ります。互換性のために、行の両端にもパイプを追加する必要があります。

```md
| Property | Description                           |
| -------- | ------------------------------------- |
| Name     | Full name of user                     |
| Age      | Reported age                          |
| Joined   | Whether the user joined the community |
```
