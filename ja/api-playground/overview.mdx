APIプレイグラウンドは、APIエンドポイントのリクエストとプレビューを行うためのインタラクティブな環境です。

<Info>
  OpenAPIを使用したAPIページの自動生成により、APIプレイグラウンドが自動的に生成されます。OpenAPIの使用について詳しくは[こちら](/api-playground/openapi)をご覧ください。
</Info>
