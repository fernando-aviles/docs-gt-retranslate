---
title: ローカル開発
description: ドキュメントを更新するためにローカルで変更をプレビュー
---

<Info>
  **前提条件**：続行する前にNode.js（バージョン19以上）をインストールしてください。
</Info>

**ステップ1**：Mintlify CLIをインストールします：

<CodeGroup>
  ```bash npm
  npm i -g mintlify
  ```

  ```bash yarn
  yarn global add mintlify
  ```

  ```bash pnpm
  pnpm add -g mintlify
  ```
</CodeGroup>

**ステップ2**：ドキュメントディレクトリ（`docs.json`ファイルがある場所）に移動し、以下のコマンドを実行します：

```bash
mintlify dev
```

あるいは、CLIをグローバルにインストールしたくない場合は、利用可能な実行スクリプトを使用できます：

<CodeGroup>
  ```bash npm
  npx mintlify dev
  ```

  ```bash yarn
  yarn dlx mintlify dev
  ```

  ```bash pnpm
  pnpm dlx mintlify dev
  ```
</CodeGroup>

<Warning>
  Yarnの"dlx"実行スクリプトにはyarnバージョン>2が必要です。詳細は[こちら](https://yarnpkg.com/cli/dlx)をご覧ください。
</Warning>

ドキュメントのローカルプレビューは`http://localhost:3000`で利用可能になります。

### カスタムポート

デフォルトでは、Mintlifyはポート3000を使用します。`--port`フラグを使用してポートをカスタマイズできます。例えば、Mintlifyをポート3333で実行するには、このコマンドを使用します：

```bash
mintlify dev --port 3333
```

既に使用中のポートで実行しようとすると、次の利用可能なポートを使用します：

```md
Port 3000 is already in use. Trying 3001 instead.
```

## バージョン

各CLIリリースは特定のバージョンのMintlifyに関連付けられていることに注意してください。ローカルウェブサイトが本番バージョンと一致しない場合は、CLIを更新してください：

<CodeGroup>
  ```bash npm
  npm i -g mintlify@latest
  ```

  ```bash yarn
  yarn global upgrade mintlify
  ```

  ```bash pnpm
  pnpm up --global mintlify
  ```
</CodeGroup>

## リンクの検証

CLIはドキュメント内の参照リンクの検証を支援できます。壊れたリンクを特定するには、以下のコマンドを使用します：

```bash
mintlify broken-links
```

## コードフォーマット

MDXを認識してフォーマットするためにIDEの拡張機能を使用することをお勧めします。VSCodeユーザーの場合は、[MDX VSCode拡張機能](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx)でシンタックスハイライトを、[Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)でコードフォーマットを行うことを検討してください。

## トラブルシューティング

<AccordionGroup>
  <Accordion title="Error: Could not load the &#x22;sharp&#x22; module using the darwin-arm64 runtime">
    これはnodeの古いバージョンが原因である可能性があります。以下を試してください：

    1. 現在インストールされているバージョンのmintlifyを削除します：`npm remove -g mintlify`
    2. Node v19以上にアップグレードします。
    3. mintlifyを再インストールします：`npm install -g mintlify`
  </Accordion>

  <Accordion title="Issue: Encountering an unknown error">
    解決策：デバイスのルートに移動し、\~/.mintlifyフォルダーを削除します。その後、`mintlify dev`を再度実行します。
  </Accordion>
</AccordionGroup>
