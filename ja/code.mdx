## 基本

### インラインコード

単語や`word`または`phrase`をコードとして示すには、バッククォート(\`)で囲みます。

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

### コードブロック

使用[フェンスで囲まれたコードブロック](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)は、3つのバッククォートでコードを囲み、先頭のバッククォートの後にスニペットのプログラミング言語を記述することで、シンタックスハイライトを得ることができます。オプションで、プログラミング言語の後にコードの名前を記述することもできます。

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

````md
```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
````
