## vercel.jsonの設定

Vercelを使用してカスタムサブパスでドキュメントをホストするには、以下の設定を`vercel.json`ファイルに追加する必要があります。

```json
{
  "rewrites": [
    {
      "source": "/docs",
      "destination": "https://[subdomain].mintlify.dev/docs"
    },
    {
      "source": "/docs/:match*",
      "destination": "https://[subdomain].mintlify.dev/docs/:match*"
    }
  ]
}
```

<Note>
  詳細については、Vercelの公式リライトガイドを参照してください：[プロジェクト設定：
  リライト](https://vercel.com/docs/projects/project-configuration#rewrites)
</Note>
