この`Update`コンポーネントは、変更と更新を追跡するために使用されます。

<Update label="2024-10-12" description="v0.1.1" tags={["Filter 1"]}>
  <Frame>
    <img className="block" src="https://mintlify-assets.b-cdn.net/hero-light.png" />
  </Frame>

  ## 変更履歴

  スクリーンショット、コードスニペット、変更リストなど、何でも追加できます。

  #### 機能

  * レスポンシブデザイン
  * 各変更履歴のスティッキーセクション
</Update>

<Update label="2024-10-11" description="v0.1.0" tags={["Filter 2"]}>
  ### 使用方法

  ```md
  <Update label="2024-10-12" description="v0.1.1" tags={["Filter 1"]}>
    This is how you use a changelog with a label 
    and a description.
  </Update>
  <Update label="2024-10-11" description="v0.1.0" tags={["Filter 2"]}>
    This is how you use a changelog with a label 
    and a description.
  </Update>
  ```

  複数の`Update`コンポーネントを使用して変更履歴を作成できます。
</Update>

<Tip>
  各`label`はアンカーを作成し、右側の目次にも表示されます。
</Tip>

## プロパティ

<ResponseField name="label" type="string" required>
  更新に付けられるラベルで、変更履歴の左側にスティッキーテキストとして表示されます。
</ResponseField>

<ResponseField name="tags" type="string[]">
  変更履歴のタグで、右側パネルにフィルターとして表示されます。
</ResponseField>

<ResponseField name="description" type="string">
  更新の説明で、ラベルとタグの下に表示されます。
</ResponseField>
