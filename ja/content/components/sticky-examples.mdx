この`<RequestExample>`と`<ResponseExample>`はスクロールしてもページの右上にコードブロックを固定します。これらのコンポーネントは、APIプレイグラウンドを使用しない場合でもすべてのページで機能します。

`<RequestExample>`と`<ResponseExample>`はモバイルでは通常のコードブロックとして表示されます。

## リクエスト例

この`<RequestExample>`コンポーネントは[CodeGroup](/content/components/code-groups)と同様に機能しますが、リクエストコンテンツを右サイドバーに表示します。そのため、複数のコードブロックを`<RequestExample>`の中に配置できます。

RequestExampleの中に配置する各コードブロックには名前を設定してください。

<RequestExample>
  ````md RequestExample Example
  <RequestExample>

  ```bash Request
    curl --request POST \
      --url https://dog-api.kinduff.com/api/facts
  ```

  </RequestExample>
  ````
</RequestExample>

## レスポンス例

この`<ResponseExample>`コンポーネントは`<RequestExample>`と同じですが、その下に表示されます。

<ResponseExample>
  ````md ResponseExample Example
  <ResponseExample>

  ```json Response
  { "status": "success" }
  ```

  </ResponseExample>
  ````
</ResponseExample>
