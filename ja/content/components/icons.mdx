## インラインアイコン

段落内で使用すると、アイコンはインラインで配置されます。

```markdown Inline Icon Example
<Icon icon="check" iconType="solid" /> The documentation you want, effortlessly 
```

<Icon icon="check" iconType="solid" />あなたが望むドキュメントを、簡単に

## プロパティ

<ResponseField name="icon" type="string" required>
  A [Font Awesome](https://fontawesome.com/icons)アイコン
</ResponseField>

<ResponseField name="iconType" type="string">
  以下のいずれか`regular`、`solid`、`light`、`thin`、`sharp-solid`、`duotone`、`brands`
</ResponseField>

<ResponseField name="color" type="string">
  16進数コードでのアイコンの色（例：`#FF5733`）
</ResponseField>

<ResponseField name="size" type="number">
  ピクセル単位でのアイコンのサイズ
</ResponseField>
