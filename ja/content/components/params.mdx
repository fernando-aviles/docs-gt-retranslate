`ParamField`コンポーネントは、APIやSDKのパラメータを定義するのに役立ちます。ParamFieldを追加すると、自動的に[API Playground](/api-playground/overview)に追加されます。

<ParamField path="param" type="string" required>
  パラメータフィールドの例
</ParamField>

<RequestExample>
  ```jsx Path Example
  <ParamField path="param" type="string">
    An example of a parameter field
  </ParamField>
  ```

  ```jsx Query Example
  <ParamField query="filter" type="string" default="none" required>
    The filtering command used to sort through the users
  </ParamField>
  ```

  ```jsx Body Example
  <ParamField body="user_age" type="integer" default="0" required>
    The age of the user. Cannot be less than 0
  </ParamField>
  ```
</RequestExample>

## プロパティ

<ParamField body="query, path, body, or header" type="string">
  クエリ、パス、ボディ、またはヘッダーパラメータであるかと、その名前
</ParamField>

<ParamField body="type" type="string">
  パラメータの値の予想される型

  サポートされる型：`number`、`string`、`bool`、`object`。

  配列は`[]`サフィックスを使用して定義できます。例：`string[]`。
</ParamField>

<ParamField body="required" type="boolean">
  パラメータが必須かどうかを示す
</ParamField>

<ParamField body="deprecated" type="boolean">
  パラメータが非推奨かどうかを示す
</ParamField>

<ParamField body="default" type="string">
  リクエストが値を提供しない場合にサーバーが使用するデフォルト値
</ParamField>

<ParamField body="initialValue" type="any">
  プレイグラウンドの初期化に使用される値
</ParamField>

<ParamField body="placeholder" type="string">
  プレイグラウンドの入力欄のプレースホルダーテキスト
</ParamField>

<ParamField body="children" type="string">
  パラメータの説明（マークダウン対応）
</ParamField>
