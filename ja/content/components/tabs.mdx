---
title: Tabs
description: Tabsコンポーネントを使用してコンテンツを切り替える
icon: window-restore
---

必要な数のタブを追加できます。

<Tabs>
  <Tab title="First Tab">
    ☝️ 最初のタブの中でのみ表示されるコンテンツへようこそ。
  </Tab>

  <Tab title="Second Tab">
    ✌️ これは2番目のタブの中にのみあるコンテンツです。
  </Tab>

  <Tab title="Third Tab">
    💪 これは3番目のタブの中にのみあるコンテンツです。
  </Tab>
</Tabs>

<RequestExample>
  ```jsx Tabs Example
  <Tabs>
    <Tab title="First Tab">
      ☝️ Welcome to the content that you can only see inside the first Tab.
    </Tab>
    <Tab title="Second Tab">
      ✌️ Here's content that's only inside the second Tab.
    </Tab>
    <Tab title="Third Tab">
      💪 Here's content that's only inside the third Tab.
    </Tab>
  </Tabs>
  ```
</RequestExample>

## タブのプロパティ

<ResponseField name="title" type="string" required>
  タブのタイトル。短いタイトルの方がナビゲーションしやすくなります。
</ResponseField>
