---
title: レスポンスフィールド
description: APIレスポンス値を表示
icon: message
---

この`<ResponseField>`コンポーネントはAPIのリターン値を定義するように設計されています。多くのドキュメントでは、何かの型をリストアップする必要がある場合に`<ResponseField>`をページで使用しています。

<ResponseField name="response" type="string" required>
  レスポンスフィールドの例
</ResponseField>

<RequestExample>
  ```jsx ResponseField Example
  <ResponseField name="response" type="string" required>
    A response field example
  </ResponseField>
  ```
</RequestExample>

## プロパティ

<ResponseField name="name" type="string" required>
  レスポンス値の名前。
</ResponseField>

<ResponseField name="type" type="string" required>
  レスポンス値の予想される型
</ResponseField>

<ResponseField name="default" type="string">
  デフォルト値。
</ResponseField>

<ResponseField name="required" type="boolean">
  フィールド名の横に「必須」と表示します。
</ResponseField>
