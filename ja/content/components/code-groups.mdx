---
title: Code Groups
description: CodeGroupコンポーネントを使用すると、タブで区切られた表示でコードブロックを組み合わせることができます
icon: rectangle-code
---

まず[コードブロック](/content/components/code)を作成し、`<CodeGroup>`コンポーネントでそれらを囲む必要があります。タブボタンに名前を使用するため、すべてのコードブロックにファイル名が必要です。

最終結果の例を以下に示します。

<CodeGroup>
  ```javascript helloWorld.js
  console.log("Hello World");
  ```

  ```python hello_world.py
  print('Hello World!')
  ```

  ```java HelloWorld.java
  class HelloWorld {
      public static void main(String[] args) {
          System.out.println("Hello, World!");
      }
  }
  ```
</CodeGroup>
