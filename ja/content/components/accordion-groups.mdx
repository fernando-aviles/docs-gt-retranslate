---
title: Accordion Groups
description: 複数のアコーディオンを1つの表示にグループ化します。
icon: table-rows
---

既存の`<AccordionGroup>`の周りに`<Accordion>`を追加するだけです。

<AccordionGroup>
  <Accordion title="FAQ without Icon">
    アコーディオン内に他のコンポーネントを配置できます。

    ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
  </Accordion>

  <Accordion title="FAQ with Icon" icon="alien-8bit">
    サポートされているプロパティについては[Accordion](/content/components/accordions)のドキュメントをご確認ください。
  </Accordion>

  <Accordion title="FAQ without Icon">
    サポートされているプロパティについては[Accordion](/content/components/accordions)のドキュメントをご確認ください。
  </Accordion>
</AccordionGroup>

<RequestExample>
  ````jsx Accordion Group Example
  <AccordionGroup>

              System.out.println("Hello, World!");
          }

    </Accordion>

    <Accordion title="FAQ with Icon" icon="alien-8bit">

    </Accordion>

    <Accordion title="FAQ without Icon">

    </Accordion>
  </AccordionGroup>
  ````
</RequestExample>
