## 基本事項

各ページはMDXファイルで、以下のメタデータ構文で始める必要があります`---`を先頭と末尾に付けて：

<CodeGroup>
  ```md Schema
  ---
  title: "title of the page"
  ---
  ```

  ```md Example
  ---
  title: "Page"
  ---
  ```
</CodeGroup>

<br />

<Tip>
  タイトルを含めるのを忘れた場合、Mintlifyはファイル名を使用します。
</Tip>

その他のメタデータは同じ`---`行の間に記述します。例えば、このページでは
以下のメタデータを使用しています：

```md
---
title: "Page Titles and Metadata"
description: "Setting the title and metadata for navigation and SEO"
---
```

## 説明

タイトルの直下に表示される説明を
`description`メタデータで設定できます。

```md
description: "Your description goes here."
```

## サイドバータイトル

ナビゲーションに異なるタイトルを表示するには`sidebarTitle`
メタデータを使用します。これは、タイトルが長く、ナビゲーションリンクでより短いものを
表示したい場合に便利です。

```md
---
title: "Your very long page title you want to shorten"
sidebarTitle: "Short title"
---
```

## サイドバーアイコン

のようなサイドバー項目にアイコンを付けたいですか？
[components](/content/components/accordions)？メタデータで`icon`属性を設定できます！[Font Awesome](https://fontawesome.com/icons)と[Lucide](https://lucide.dev/icons)のすべてのアイコンが
利用可能です。Font Awesomeアイコンのアイコンタイプも設定できます（オプション）。設定しない場合、アイコン
タイプはregularになります。

```md
---
title: "Code Block"
icon: "rectangle-code"
iconType: "solid"
---
```

## APIページ

APIページでは、インタラクティブなAPIプレイグラウンドを構築できます。APIページを作成するには、
ページメタデータに`api`または`openapi`プロパティを設定する必要があります。

APIプレイグラウンドについての詳細は[デモ](/api-playground)。

<CodeGroup>
  ```md Schema
  api or openapi: method endpoint
  ```

  ```md API Example
  api: "GET /users"
  ```

  ```md OpenAPI Example
  openapi: "GET /users"
  ```
</CodeGroup>

## ページモード

ページモード設定では、ページの外観をカスタマイズすることができます。異なるモードから選択して、必要に応じてレイアウトを調整できます。モードが指定されていない場合、ページはデフォルト設定を使用します。

### デフォルト

特定のモードが指定されていない場合、ページは標準設定にデフォルト設定されます。これは、ページが（見出しがある場合）デフォルトの目次やその他の標準要素と共に表示され、特別な調整なしで一般的なレイアウトを提供することを意味します。

```md
---
title: "Page with no ToC"
---
```

### ワイドモード

ワイドモードでは、ページの右側にある目次（ToC）を非表示にすることができます。これは、ページに見出しがない場合や、他のコンテンツのために余分な水平スペースを活用したい場合に特に便利です。

```md
---
title: "Page with no ToC"
mode: "wide"
---
```

### カスタムモード

カスタムモードは、トップバーを除くすべての要素を削除することで、ミニマリストなレイアウトを提供します。このモードは「ランディングページ」や、クリーンで邪魔のない環境が必要なページを作成するのに理想的な白紙のキャンバスを提供します。

```md
---
mode: "custom"
---
```

### センターモード

センターモードは、サイドバーと目次を削除し、ページコンテンツを中央に配置します。このモードは、変更ログや、コンテンツに焦点を当てたいページに最適です。

```md
---
mode: "center"
---
```

## 外部リンク

サイドバーで外部URLを開きたい場合は、`url`メタデータを任意のページで設定できます。

```md
---
title: "Page that goes to external link"
url: "https://www.npmjs.com/package/mintlify"
---
```

## 検索エンジン最適化

ページのメタデータにソーシャルメディアで共有された時に表示される画像などのメタタグを設定できます。コロンを含むメタタグは引用符で囲む必要があります。

```md
"twitter:image": "/images/your-photo.jpg"
```

詳しくは[SEOページ](/settings/seo)をご覧ください。サポートされているすべてのメタタグについて説明しています。
