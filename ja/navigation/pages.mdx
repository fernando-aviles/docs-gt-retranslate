## ページ

階層構造を必要としない場合は、`navigation`フィールド内でページを定義できます。

配列の各エントリは`pages`リポジトリ内に存在するファイルへのパスである必要があります。なお、ファイルパスに`.mdx`を追加する必要はありません。

```json
{
  "navigation": {
    "pages": [
      "overview",
      "quickstart",
      "advanced/components",
      "advanced/integrations"
    ]
  }
}
```

## グループ

グループを使用してページをグループ化できます。グループは互いにネスト化することもできます。

```json
{
  "navigation": {
    "groups": [
      {
        "group": "Getting Started",
        "pages": [
          "quickstart",
          {
            "group": "Editing",
            "icon": "pen-paintbrush",
            "pages": ["development", "web-editor"]
          }
        ]
      },
      {
        "group": "Writing Content",
        "pages": ["writing-content/page", "writing-content/text"]
      }
    ]
  }
}
```
