---
title: ローカライゼーション
---

`"languages"`の`docs.json`では、ナビゲーションを異なる言語に分割することができます。

現在、以下の言語でのローカライゼーションをサポートしています：

* アラビア語（`ar`）
* 中国語（`cn`または`zh`、`zh-Hans`は簡体字用、`zh-Hant`は繁体字用）
* 英語（`en`）
* フランス語（`fr`）
* ドイツ語（`de`）
* インドネシア語（`id`）
* イタリア語（`it`）
* 日本語（`jp`または`ja`）
* 韓国語（`ko`）
* ポルトガル語（`pt`）
* ポルトガル語（ブラジル）（`pt-BR`）
* ロシア語（`ru`）
* スペイン語（`es`）
* トルコ語（`tr`）

```json docs.json
{
  "navigation": {
    "languages": [
      {
        "language": "en",
        "groups": [
          {
            "group": "Getting Started",
            "pages": ["en/overview", "en/quickstart", "en/development"]
          }
        ]
      },
      {
        "language": "es",
        "groups": [
          {
            "group": "Getting Started",
            "pages": ["es/overview", "es/quickstart", "es/development"]
          }
        ]
      }
    ]
  }
}
```

## AI翻訳 \[BETA]

ボタンをクリックするだけで、ドキュメントを1つの言語から別の言語に自動的に翻訳できます。

<Info>
  AI翻訳はベータ版です - アクセスを得るには[sales@mintlify.com](mailto:sales@mintlify.com)にお問い合わせください。
</Info>

AI翻訳では、ドキュメントを翻訳したい言語を選択できます。ダッシュボードから翻訳をトリガーすると、翻訳されたページと編集された`docs.json`を含むPRが生成され、翻訳を確認する機会が与えられます。
