## リスト

### 順序付きリスト

順序付きリストを作成するには、ピリオドの後に行項目を追加します

1. 最初の項目
2. 2番目の項目
3. 3番目の項目
4. 4番目の項目

```md
1. First item
2. Second item
3. Third item
4. Fourth item
```

### 順序なしリスト

順序なしリストを作成するには、ダッシュ（`-`）、アスタリスク（`*`）、またはプラス記号（`+`）を行項目の前に付けます

* 1番目の項目
* 2番目の項目
* 3番目の項目
* 4番目の項目

```md
- First item
- Second item
- Third item
- Fourth item
```

### ネストされたリスト

リスト項目をネストするにはインデントを追加します

* 1番目の項目
* 2番目の項目
  * 追加項目
  * 追加項目
* 3番目の項目

```md
- First item
- Second item
  - Additional item
  - Additional item
- Third item
```

<Tip>
  リストは公式の[markdownシンタックス](https://www.markdownguide.org/basic-syntax/#lists-1)に従います。
</Tip>

## テーブル

| プロパティ | 説明                   |
| ----- | -------------------- |
| 名前    | ユーザーのフルネーム           |
| 年齢    | 報告された年齢              |
| 参加    | ユーザーがコミュニティに参加したかどうか |

### テーブルの作成

<Tip>
  テーブルコンポーネントは公式の[markdownシンタックス](https://www.markdownguide.org/extended-syntax/#tables)に従います。
</Tip>

テーブルを追加するには、3つ以上のハイフン（`---`）を使用して各列のヘッダーを作成し、パイプ（`|`）を使用して各列を区切ります。互換性のために、行の両端にもパイプを追加する必要があります。

```md
| Property | Description                           |
| -------- | ------------------------------------- |
| Name     | Full name of user                     |
| Age      | Reported age                          |
| Joined   | Whether the user joined the community |
```
