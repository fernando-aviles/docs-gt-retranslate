あなたのブランドに基づいてドキュメントのルックアンドフィールを差別化できるように、事前に構築されたテーマを提供しています。

## 設定

テーマは以下のファイルで設定できます：`docs.json`ファイル。

例えば、`maple`テーマを使用するには、`docs.json`ファイルに以下を追加します：

```json
{
  "theme": "maple"
}
```
