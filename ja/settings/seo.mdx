## デフォルトのメタタグ

デフォルトのメタタグを設定するには、`metadata`フィールドを`docs.json`に追加します。

```json
"seo": {
    "metatags": {
        "og:image": "link to your default meta tag image"
    }
}
```

## 1ページのメタタグを変更する

以下のようにページのメタデータにメタタグを追加します：

```md
---
title: 'Your Example Page Title'
'og:image': 'link to your meta tag image'
---
```

<Warning>
  コロンを含むメタタグは、ページのメタデータに記述する際に引用符で囲む必要があります。
</Warning>

## サイトマップ

Mintlifyは自動的に`sitemap.xml`ファイルと`robots.txt`ファイルを生成します。`/sitemap.xml`をウェブサイトURLの末尾に追加するだけでサイトマップを確認できます。SEOを改善するために、検索エンジンにサイトマップを提出することをお勧めします。

デフォルトでは、`docs.json`のページのみが含まれます。非表示のリンクを含めるには、`seo.indexing`を`docs.json`に追加します：

```json
"seo": {
    "indexing": all
}
```

## インデックス作成の無効化

検索エンジンによるページのインデックス作成を停止したい場合は、ページの[frontmatter](/page)に以下を含めることができます：

```
---
noindex: true
---
```

また、`noindex`をすべてのドキュメントページに対して`metadata.robots`フィールドを`"noindex"`に設定して`docs.json`で指定することもできます：

```json
"seo": {
    "metatags": {
      "robots": "noindex"
    }
  }
```
