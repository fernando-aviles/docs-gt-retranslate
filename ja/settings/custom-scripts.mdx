カスタムCSS & JSをドキュメントに追加して、外観を完全にカスタマイズできます。

## カスタムCSS

任意の数のCSSファイルをリポジトリに追加すると、定義されたクラス名がすべてのMDXファイルで適用され利用可能になります。

### style.cssの追加

例えば、以下のような`style.css`ファイルを追加してナビゲーションバーとフッターのスタイルをカスタマイズできます。

```css
#navbar {
  background: "#fffff2";
  padding: 1rem;
}

footer {
  margin-top: 2rem;
}
```

### 識別子の使用

Mintlifyには、UIの重要な要素にタグを付けるための一般的な識別子のセットがあります。以下に一部（すべてではありません）を示します：

`#topbar-cta-button` `#navbar` `#sidebar` `#content-area` `#table-of-contents`

<Tip>
  カスタマイズしたい要素の参照を見つけるには要素の検査を使用してください。
</Tip>

<Warning>
  プラットフォームの進化に伴い、一般的な要素の参照とスタイリングは変更される可能性があります。カスタムスタイリングは慎重に使用してください。
</Warning>

## カスタムJS

カスタムJSを使用すると、カスタム実行可能コードをグローバルに追加できます。これは、JSコードを含む`<script>`タグを各ページに追加するのと同じです。

### カスタムJavaScriptの追加

任意の`.js`ファイルをドキュメントのコンテンツディレクトリ内に配置すると、すべてのドキュメントページに含まれます。例えば、以下のような`ga.js`ファイルを追加して[Google Analytics](https://marketingplatform.google.com/about/analytics)をドキュメント全体で有効にできます。

```js
window.dataLayer = window.dataLayer || [];
function gtag() {
  dataLayer.push(arguments);
}
gtag('js', new Date());

gtag('config', 'TAG_ID');
```

<Warning>
  セキュリティの脆弱性を引き起こさないよう、慎重に使用してください。
</Warning>
