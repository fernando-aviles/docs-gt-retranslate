ドキュメントをカスタムドメインで設定するには、Mintlifyの設定で希望のカスタムドメインを設定し、ドメインプロバイダーでDNS設定を構成する必要があります。

<Info>
  mintlify.com/docsのようなカスタムサブディレクトリを設定したい場合は、手順を[こちら](/advanced/subpath/cloudflare)で確認してください。
</Info>

## ダッシュボード設定

1. まずは[ダッシュボード](https://dashboard.mintlify.com)
2. 「Settings」をクリックします。
3. 「Custom Domain」をクリックします。
4. 希望のカスタムドメインを入力します。例えば、`docs.mintlify.com`。

## Vercelでの認証

Vercelがドメインプロバイダーの場合、認証`TXT`レコードを追加する必要があります。この情報は、カスタムドメインを送信した後にダッシュボードに表示され、メールでも送信されます。

## DNSの設定

1. ドメインプロバイダーのウェブサイトでドメインのDNS設定に進みます。
2. 以下の値を入力して新しいDNSエントリを作成します：

```
CNAME | docs | cname.vercel-dns.com.
```

<Frame>
  <img alt="DNS" src="https://mintlify-assets.b-cdn.net/mintlify-dns.png" />
</Frame>

CloudflareをDNSプロバイダーとして使用している場合、httpsの設定で「full strict」セキュリティオプションを有効にする必要があります。

上記の設定後もカスタムドメインが設定されていない場合は、[サポートにお問い合わせ](mailto:sales@mintlify.com)ください。
