## Usage

Install the package:

```bash
npm i @mintlify/scraping
```

Then run one of these commands:

<CodeGroup>
```bash Multiple Pages
npx mintlify-scrape section [url]
```

```bash Single Page 
npx mintlify-scrape page [url]
```
</CodeGroup>

The scraper will automatically detect the framework of your existing documentation.
