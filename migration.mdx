## Quick Start

Install the package:
```bash
npm i @mintlify/scraping
```

Use the following commands to migrate your docs:
- `mintlify-scrape section [url]` - Scrapes multiple pages
- `mintlify-scrape page [url]` - Scrapes a single page

For OpenAPI migrations:
```bash
mintlify-scrape openapi-file [openApiFilename]
```

Options:
```bash
-w, --writeFiles  Write frontmatter files [boolean] [default: true]
-o, --outDir      Output directory for frontmatter files [string]
```
